# jp-study 项目学习规则与设定

## 1. 核心学习规则
- **材料来源**：所有讲解、复习、测试必须**严格以仓库中 `日语语法N5-N1.md` 文件的内容为主**。
- 可以适度发散文档内已有的相关知识，但**不能自行外部搜索**。
- 如果文档有错误，我会指出并帮助修改文档。

## 2. 学习流程与测试机制
- **逐题测试**：一个问题一个问题地进行，用户回答后立即批改。
- **掌握判定**：必须完全符合文档中的定义、表格、例句、注意点，才能算「掌握」。
- **反复复习**：主动安排针对性复习和抽测，防止遗忘。
- **进度记录**：所有掌握情况记录在 `jp-study-progress.md` 中。

## 3. 交互方式
- 用户可以要求「开始复习不扎实的地方」、「暂停」、「进入下一章节」等。
- 复习题只会干净提问，不会提前泄露答案。
- 测试后会更新进度文件并汇报。
