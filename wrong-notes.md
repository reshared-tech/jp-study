# 错题本

> **AI 维护，每次造句出错自动录入。每日学习开头随机抽 3-5 条复发率最高的回顾。**
>
> **规则**：复发 3 次以上 → 红名单（每周强化）/ 复发 5 次以上 → 顽固级（每日必抽）

## 🔴 红名单（复发 3 次以上 — 顽固错误）

### #01 修正回旋（最顽固 ⭐⭐⭐⭐⭐）

- **复发**：3 次
- **类型**：综合
- **描述**：修复一个错误时，把对的其他字也改坏（如「は↔が」「どんな↔どう」）
- **历史**：三轮 ① / 2.2 ③ / 2.4 ⑨
- **口诀**：**"精确手术修改：只动错的字，其他字锁住不动"**
- **预防**：写完先停 2 秒，识别要改的位置，其他不动

### #02 主语后助词漏写（写多了就忘 ⭐⭐⭐⭐⭐）

- **复发**：5+ 次
- **类型**：助词
- **描述**：「わたし」「私」后面忘加「は」
- **历史**：5.2 / 5.4 / 5.5 / 2.4 ③ / 等
- **口诀**：**"主语后必须有助词，零助词不存在"**

### #03 体不统一（前简后礼）⭐⭐⭐⭐

- **复发**：3 次
- **类型**：语体
- **描述**：句末是礼貌「です／ます」，但前句中顿/接续用了简体
- **典型错**：「中国語は難しい**が**、英語は簡単**です**」(前简后礼)
- **历史**：2.1 ② / 2.2 ③ / 5.16 Q3
- **口诀**：**"看见前句形容词/名词要加接续 → 先反射加です再加が"**

### #04「は」读 wa 写法陷阱 ⭐⭐⭐⭐

- **复发**：5 次
- **类型**：拼写
- **描述**：读音 wa 的助词写成「わ」，但实际写法是「は」
- **历史**：「でわ→では」「散歩しで→散歩して」「こんに**わ**」预警
- **当前状态**：已克服（5.20「こんにちは」首次写对）
- **口诀**：**"读 wa 的助词都写は：主题は / では / じゃ / こんにちは"**

## 🟡 黄名单（复发 2 次 — 注意）

### #05 二类形容词陷阱"加多 い"

- **复发**：2 次（三轮 ① 好きい / 三轮 ⑤ 厳し漏い）
- **类型**：形容词分类
- **描述**：陷阱二类词「好き / 嫌い / 綺麗 / 有名 / 得意 / 元気」**不加い**
- **口诀**：**"陷阱二类词尾已含い音，不要再加假名い"**

### #06 まで vs まだ 截胡

- **复发**：2 次（5.6 Q1 / 5.22 ①）
- **类型**：助词/副词混淆
- **描述**：「まで」(终点助词) 写成「まだ」(副词 = 还、仍然)
- **口诀**：**"目的地 → で 音 / 未完成 → だ 音"**

### #07 これ vs この 连体词混淆

- **复发**：2 次（2.5 ④ + N3 题）
- **类型**：章节一基础
- **描述**：连体词「この」必接名词 / 事物代词「これ」独立做主语
- **历史**：「これ机」「これチーズ」

### #08 自他动词混淆（入る vs 入れる / 開く vs 開ける）

- **复发**：1-2 次（2.4 ④ / 5.7 / 4.13 节）
- **类型**：动词分类
- **描述**：他动词配「を」/ 自动词配「が」
- **口诀**：**"助词反推：写了 を 必接他动词"**

- **复发**：1 次（5.30 ①「寝って」错）
- **描述**：2 类直接去る加て（寝る→寝て）

### #13 半浊音 ぱ vs 浊音 ば 错位

### #14 清浊音错位 こ ↔ ご

### #15 片假名拼写错

- **复发**：1 次（4.4 カーヒー / 5.24「スマホン」）

### #16 名词化「のは」漏 の

- **复发**：1 次（5.24 ②「いじるは」漏 の）
- **描述**：动词原形作主题必须加「の」名词化

### #17 「あなた」语用陷阱

- **复发**：1 次（2.2 ②）
- **描述**：对认识的人不用「あなた」，用「○○さん」

### #18 跨节助词混淆「英語に翻訳」≠「英語を翻訳」

- **复发**：1 次（2.2 ⑤）
- **描述**：に = 转换目标 / を = 直接宾语

### #19 二类形容词当动词错加ます

- **复发**：1 次（2.1 ⑤「好きます」）
- **描述**：「好き」是形容词，礼貌化用です不用ます

### #20「飛ぶします」动词叠加

- **复发**：1 次（2.3 ②）
- **描述**：已是动词原形不再加「します」

### #21 「来ない」读音

- **状态**：已稳定（9 次主动标音 こ）
- **口诀**：来る三读：来ない=こ / 来ます=き / 来た=き
