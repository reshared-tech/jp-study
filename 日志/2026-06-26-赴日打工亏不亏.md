## 一、语法点（每点一句例句）

- **〜から**（因为，口语主观）：**お金を稼げなかったから、損だと思います。**
- **〜からといって**（不能因为…就…，N2 10.2）：**東京の家は高いからといって、買うのは損だとは限らない。** ← 用户首次主动造句成功。
- **〜とは限らない**（未必，N2 10.6）：**損だとは限らない。**（限らない＝かぎらない）
- **〜と思う**（我觉得，と＝引号助词）：**損だと思います。**（想法直接用 思います，不加 ています）
- **本当は**（其实＝本当＋は，话锋一转）：**彼は本当は国に帰りたい。**
- **勤める**（任职/归属，配 に）：**今は東京の会社に勤めています。**
- **敬语方向 お〜**：**お持ちですか**（尊敬，问对方）→ 答自己 **持っていません**。

## 二、核心收获

### 1. 一条因果逻辑链（用户自己的观点拼成日语）
> 稼げなかった（没挣到）→ **からといって**（不能就因为这个）→ **損だとは限らない**（就一定亏）

用户自发用 **買うの**（の 名词化，把"买房"打包成名词），地道度很高。

### 2. 「我觉得」= …と思う
- 「○○」**と** 思う：と 是把想法打包的"引号助词"（和 とは限らない 的 と 同一个）。
- 「觉得」直接用 **思います**，不要 ~~思うています~~。

### 3. 三个 は 是同一个味道（对比/转折开关）
**損だとは** / **本当は** —— 都是表"表面 vs 其实"的对比 は。这是日语"话里有话"的开关。

### 4. 働く vs 勤める（终于切清楚）
| 词 | 焦点 | 助词 | 画面 |
|---|---|---|---|
| 働く | 流汗干活（动作） | で（舞台） | 在这儿干活 |
| 勤める | 我属于这组织（归属/身份） | に（附着点） | 我是这儿的人 |
🔗 勤める 的 に 和 **住む** 的 に 同一家人——都不是动作，是"长期存在/归属"。

### 5. ★ 敬语方向 お〜（昨天的坑填平了）
用户真正的卡点不是"敬语给谁用"，而是"**认不出 お 是粘上去的还是词本身**"。给了一把尺子：
- **お + 动词连用形 + です/になる** → 尊敬（指对方）：去掉 お 剩动词零件（持ち/帰り/分かり）→ 自己别用，露光秃秃动词（持っていません）。
- **お + 名词** → 美化（不指谁）：去掉 お 剩独立名词（会計/茶/願い）→ 自己也能用（お会計をお願いします ✓）。
- 三连判断（お名前/お会計/お分かりました）用户全对并能讲依据。

## 三、发音/变形小坑
- **勤めっています ✗ → 勤めています ○**：勤める 是一段动词，变 て 形**去る加て、不变音**；那个小 **っ** 只在五段（買う→買って、帰る→帰って）里出现。

## 四、自然复习到的（旧词新场景）
- **稼ぐ**（稼げなかった）、**帰る**（本当は国に帰りたい）、**たい**（帰りたい）——均主动用对，帰る/たい 达"掌握"门槛。

## 六、下半场（同日续，用户要求每天≈2h 广度）

挣扎项清空后解除门槛，跑广度引擎：N3 语法断层 + 限时阅读。

### N3 语法
- **〜のに**（明明…却…，N3 9.3 逆接，定级断层）：自带"不满/惋惜"情绪，黏前句屁股；接普通形。
  - 用户主动用对 3 句 2 场景：**給料が上がったのに、貯金できない。** / **彼は国に帰りたいのに、帰れない。**
  - 与 でも 区别：のに 带情绪，でも 中性另起。
- **〜たり〜たり**（又…又…，N3 9.11 列举）：动词た形+り，最后收 する。
  - ★时态只压在句尾 する 上（します/しました），前面 たり 是冻住的零件不变。
  - 用户造：**お弁当を買ったり、お金を出したりします／しました。**

### 发音/活用坑
- **可能形的 ら 坑**：~~帰れらない~~ → **帰れない**。**帰る/入る/要る/走る/知る 是伪装成一段的五段，可能形不带 ら**（ら 是一段专属）。
- **可能形彻底通关**：规则（稼げる/食べられる）+ 两刺头（**する→できる、来る→来られる**）+ **〜ことができる**（=可能形）三件套全齐。でき＝できる 词干。

### で/に 再钉一次
同一个 コンビニ，看后面动词选助词：**コンビニに行く**（に＝行的落点）vs **コンビニで買う**（で＝动作舞台）。用户曾误说 コンビニに買う，已纠正。

### 阅读模块（本周开张）
读「セブン銀行のATMに『やさしい日本語』画面ができました」（やさにちウォッチ）。用户**独立读懂大意 + 推对因果**（読取る→読む 是为方便外国人读懂）。文中自然复现 **分かりやすい**（复习 やすい），捞出 **〜たり〜たり / 確認できる**。
- 脸熟词（未入账，控负荷）：銀行・預ける・出す・機械・在留カード・本人・手続き・方法。

### 本场新增（账本）
のに / たり / 確認（かくにん）/ と思う / とは限らない；可能形升满 mastery5。活词 37→42，N3 语法 learning 0→2（のに・たり）。

## 七、第三场（同日续，用户："聊天学语法很不错"，继续攻 N3 语法）

主攻 N3「9.12 立场与对象」——定级时 にとって 标"不会"，今天补上。

### 〜にとって（对…来说 / 看视角）
- に+とって（取って）＝"拿…来看"，名词+にとって，后接**判断/评价**。
- 用户 2 句 2 场景主动用对：
  - **外国人にとって、日本で家を買うのはとても難しいです。**（で 也选对了）
  - **私にとって、貯金は難しいです。**

### 〜について（关于… / 圈话题）
- 名词/こと+について，后接**言说·思考类**（話す/考える/書く）。
- 用户造：**家を買うことについて、よく考えます。**

对照钩子：**にとって 看视角（后接难/重要），について 圈话题（后接说/想）。**

### ★ の / こと 名词化（今日反复踩，单列成 concept）
今天三次易错，规律理清：
1. の + こと 叠用 → ~~買うのこと~~ ✗
2. 给名词加 の → ~~貯金のは~~ ✗（貯金已是名词，直接 は）
3. について 前用 の → ~~買うのについて~~ ✗ → 買う**こと**について
规律：**の 后跟光杆 は/が/だ；こと 后跟词组型助词（について…）；名词本身不打包。** 标记 watch，后续复现。

### 本场新增（账本）
にとって / について / 名词化(concept) / 考える（かんがえる）。活词 42→46，N3 语法 learning 2→4（+にとって・について）。
