# 长句背诵计划（每日一句）

> 制定日期：2026-07-23
> 总目标：2026-12 考 N2。用「每天背诵一个包含语法和单词的长句（短文）」定向覆盖 N2 考点层，长尾词交给阅读和日常对话自然积累。

## 方法（每日流程）

1. **选材生成**：Claude 从「到期该复习的语法 + 未覆盖的 N2 高频词」中选材，生成一段 3～5 句的短文（不随机撞词）。
2. **先产出再对答案**：先自己凭中文意思试着说/写出句子，再看原文（遵循"限时提取、逼产出"原则，不做被动跟读起步）。
3. **拆解文档**：每词每语法拆到最细，存 `学习记录/YYYY-MM-DD-长句背诵拆解.md`，末尾附提取练习清单（核心语法/搭配/变形）。
4. **慢速音频**：Edge TTS（ja-JP-NanamiNeural，-30%）生成 `学习记录/YYYY-MM-DD-长句背诵-慢速.mp3`，背完后用于校对发音，不先听。
5. **复习**：旧句子按间隔复习（次日→3天→7天→半月），复习方式是拿清单做限时提取，不重读讲解。

## 选材规则（硬约束）

- 每句新语法 **3～5 个**：优先级 = 到期复习的语法 > 未覆盖的 N2 语法 > N3 巩固。
- 每句新词 **10～20 个**：优先级 = N3–N2 核心词 > 生活口语类词（拟声拟态/惯用句/缩约形/生活・贸易词，2026-07-23 已按等级并入 N 级词表）> 其他。
- **生词密度上限**：全是生词的句子背不下来；已知词做骨架，新词做血肉。
- **句式必须是日常高频表达**（2026-07-23 新增）：只用母语者日常真的会说的句式，语法点以其**最常见的口语形态**入句。避免语法上正确但生硬少用的堆叠形（反例：第 1 句的「覚えられていなくて」，自然说法是「なかなか覚えられなくて」）。判断标准：这句话原样出现在日剧对话/日常聊天里不违和。
- 尽量不重复已覆盖的内容词；助词、基础句型的自然重复不算重复（那是复习）。
- 题材贴近生活/工作场景（贸易公司、日常生活），配合"生活方式打到 N2"的路线。
