# 日语语法手册（N5–N1）

> 仅收录语法内容：指示词体系、助词、形容词、动词、句型、时态体、敬语、接续词与副词、中级 N3 语法、高级 N2 语法、最高级 N1 语法。
> 文字 / 发音 / 数字量词 / 词汇等非语法内容已分流到其他文档。

# 一、指示词体系（こそあど）

## 1.1 四组体系总览

|  | 近称（**こ**） | 中称（**そ**） | 远称（**あ**） | 疑问（**ど**） |
|---|---|---|---|---|
| **連体詞**（+ 名词）| **この** | **その** | **あの** | **どの** |
| **事物代詞**（独立）| **これ** | **それ** | **あれ** | **どれ** |
| **方向 / 礼貌指人** | **こちら** | **そちら** | **あちら** | **どちら** |
| **场所** | **ここ** | **そこ** | **あそこ** | **どこ** |
| **样态形容**（+ 名词）| **こんな** | **そんな** | **あんな** | **どんな** |
| **副词性**（怎样）| **こう** | **そう** | **ああ** | **どう** |

⚠️ 远称里**「あそこ」是唯一不规则**（不是 "あこ"），多了一个"そ"音。

## 1.2 各组用法

- **この/その/あの/どの**：连体词，**必须后接名词**。この本は私のです。（这本书是我的。）
- **これ/それ/あれ/どれ**：事物代词，**直接做主语**，不能后接名词。これは本です。（这是书。）
- **ここ/そこ/あそこ/どこ**：地点代词。図書館はどこですか？──あそこです。（图书馆在哪？──在那里。）
- **こちら/そちら/あちら/どちら**：万能 / 礼貌型，人 / 方向 / 事物都能指。こちらは李さんです。（这位是小李。）
- **こんな/そんな/あんな/どんな**：问性质，答案常用形容词或形容词修饰的名词短语。呉先生はどんな先生ですか。（吴老师是怎样的老师？）
- **こう/そう/ああ/どう**：副词性 / 状态。どうですか？（怎么样？）

## 1.3 远近判别：對立性 vs 融合性

| 場景 | 物的位置 | 用 |
|---|---|---|
| 對立性（说听人分开）| 离说话人近 | こ |
| | 离听话人近 | そ |
| | 两人都远 | あ |
| 融合性（两人在一起）| 离两人近 | こ |
| | 离两人远 | そ |
| | 离两人很远 | あ |

## 1.4 文脈指示（话题指代）

| 词 | 关键 |
|---|---|
| **その** | 前文已提到，**一方知道、另一方不知道** |
| **あの** | 前文已提到，**双方都知道** |

例：
- A：高橋さんは綺麗ですよ。B（不认识）：その人は独身ですか？
- B（也认识）：あの人は独身じゃないですよ。

## 1.5 速记

> **コ近・ソ中・ア遠・ド疑問**

# 二、助词系统

## 2.1 は（主题 / 对比）

主题 / 主语提示助词，读音 **wa**，本身不翻译。

例：
- 私は学生です。（我是学生。）
- 鈴木さんはどんな人ですか。（铃木是怎样的人？）
- 日本語は難しいですが、漢字は簡単です。（日语难，但汉字简单。）

⚠️ **が前・は後 强调规则**：「が」强调が前面，「は」强调は后面。
- 私が学生です（强调"是我"，前面是焦点）
- 私は学生です（强调"是学生"，后面是焦点；中性语境下即默认陈述）

## 2.2 が（主语 / 对象 / 转折）

### 用法 1：主语 / 新信息

例：友達が来ました。（朋友来了。）

### 用法 2：能力 / 喜好 / 情感的对象（不用を）

| 词类 | 例 |
|---|---|
| 好き / 嫌い | 私は犬が好きです |
| 上手 / 下手 / 苦手 | 彼はテニスが上手です（他网球打得好。日语母语者不直接对自己用「上手」，自我表达请改用「得意」：私はテニスが**得意**です。）|
| 欲しい / わかる / できる / 怖い | 日本語ができる |

⚠️ **苦手（主观，带抵触感）** vs **下手（客观，技术差）**。

### 用法 3：疑问词主语 → 必用が

⚠️ **铁律**：疑问词（どれ/どの/どこ/何/誰）作主语 → 必用 が。
- どれが高橋の本ですか。（哪个是高桥的书？）
- 誰が来ましたか。（谁来了？）

### 用法 4：转折 / 缓和（接续助词，附前句末）

- 美味しいですが、高いです。（好吃，但贵。）
- すみませんが、トイレはどこですか。（不好意思，洗手间在哪？）

## 2.3 を（宾语 / 移动范围 / 离开点）

### 用法 1：他动词宾语
ご飯を食べる。（吃饭。）

### 用法 2：移动 / 通过的范围（自动词搭配）
判别：① 停留时间短；② 移动范围大。
- 鳥が空を飛ぶ。（鸟在天空飞。）
- 公園を散歩する。（在公园散步。）

### 用法 3：离开 / 出发的起点
- 大学を卒業する。（大学毕业。）
- バスを降りる。（下公交。）

### ⚠️ を vs から（同表起点）
- **から**：一般起点，常配「まで」
- **を**：意识地、主动地离开（卒業、出発、家を出る）

## 2.4 に（多用法）

### 用法 1：时间点（精确）
⚠️ 数字时间加 に，相对时间不加。
- 加に：7時、3月、16日、2025年
- 不加に：今日、明日、毎日、来週

### 用法 2：目的地 / 到达点
学校に行く。家に入る。

### 用法 3：动作对象（单向）
母に手紙を書く。先生に質問する。

### 用法 4：着落点 / 附着位置
- 椅子に座る。（坐到椅子上。）
- バスに乗る。（上公交。）
- 冷蔵庫に入れる。（放进冰箱。）

### 用法 5：存在场所
公園に犬がいる。

### 用法 6：频率
1週間に3回。

### 用法 7：移动目的（動詞ます形 + に + 行く/来る）
食べに行く。買い物に行く。

### 用法 8：用途 / 名目 / 资格（"作为 X / 用作 X"）
表示"把 X 当成某种用途、名目、场合"。常见搭配「〜に〇〇を 取る／買う／贈る／使う／あげる」等。
- 朝（あさ）ごはん**に**パンを食べる。（**作为**早餐吃面包。）
- お土産（みやげ）**に**お菓子（かし）を買う。（**作为**伴手礼买点心。）
- お祝（いわ）い**に**花を贈（おく）る。（**作为**贺礼送花。）
- お礼（れい）**に**コーヒーをおごる。（**作为**答谢请喝咖啡。）

⚠️ 判别诀窍：中文能翻译成"**作为〜**"的「に」，就是这个用法。和用法 1（时间）/2（目的地）/5（场所）等的"具体物理点"不同，这里是**抽象的名目/场合归类**。

### ⚠️ 对比辨析

| 对比 | 区分 |
|---|---|
| に（存在）vs で（动作）| 学校にいる / 学校で勉強する |
| に（单向）vs と（双向）| 偶然彼に会う / 友達と会う（约定）|
| に（点）vs へ（方向）| 「に」具体某点 / 「へ」朝那个方向 |

## 2.5 で（场所 / 工具 / 限定 / 范围 / 材料）

### 用法 1：动作发生的场所
食堂でご飯を食べる。

### 用法 2：工具 / 手段 / 方法
- 箸で食べる。（用筷子吃。）
- 日本語で話す。（用日语说。）
- バスで行く。（坐公交去。）

### 用法 3：限定（时间 / 数量 / 范围）
- 5分で駅に着く。（5分钟到车站。）
- 日本で一番高い山。（日本最高山。）

### 用法 4：材料

| 用法 | 判别 | 例 |
|---|---|---|
| で | 肉眼可见 / 常识可知 | 木で箱を作る |
| から | 看不见 / 化学反应 | 牛乳からチーズ |

### 用法 5：事件发生地点（場所 + で + 事件 + が ある）
教室で試験があります。来週、東京でサッカーの試合があります。

### 用法 6：中顿 / 并列（名词、二类形容词）
私は中国人で、学生です。

## 2.6 と（伴随 / 引用 / 并列）

### 用法 1：共同动作的对象
友達とご飯を食べる。

### 用法 2：完全列举（A と B = 就这两个）
パンと飴を買った。

### 用法 3：引用 ── と言う / と思う / と申す
- これは中国語で何と言いますか。（这个用中文叫什么？）
- 李さんは来ないと思う。（我觉得小李不会来。）

⚠️ **「と思う」前必须接简体句**！「来ません**と**思う」❌。

## 2.7 から（起点 / 原因）

### 用法 1：地点 / 时间起点
- 7時から学校に行く。（7点开始去学校。）
- 海外から観光客が来る。

### 用法 2：原因（主观）
- から：主观（自己想做、觉得）
- ので：客观（事实陈述、新闻）

接续：
- 一类形容词 / 动词简体 + から：難しいから / 食べるから
- 二类形容词 / 名词 + だ + から：好きだから / 学生だから
- 丁寧体：難しいですから

## 2.8 まで / までに（终点对比）

| 助词 | 用法 | 接 |
|---|---|---|
| **まで** | 持续到某时 | 持续性动作。10時まで勉強した |
| **までに** | 某时点之前完成 | 瞬间动作。10時までに帰る |

> "做到 ○○ 时" → まで（持续）
> "在 ○○ 之前做完" → までに（瞬间）

## 2.9 より（比较基准）

> A は B より 形容词 = A 比 B 更~

例：
- 飛行機は車より速い。（飞机比车快。）
- 美穂さんの身長は私より高い。（美穗比我高。）

## 2.10 の（修饰 / 所属 / 同位 / 形式名词）

### 用法 1：所属 / 所有 ── 私の本（我的书）
### 用法 2：性质 / 内容 / 修饰 ── 日本語の本（关于日语的书）
### 用法 3：同位（前后名词指同一对象）

⚠️ 中文里没有这种用法 ── 必须单独记。
- 私は校長の張光輝です。（校长 = 张光辉 = 我）
- こちらは友達の王さん。（朋友 = 王同学）

### 用法 4：形式名词（代替前文事物 / 人）
- このカバンは高橋さんの（カバン）です。
- 大きいのが欲しい。（要大的。）

⚠️ 「の」自身不能省！

### 用法 5：动词名词化
日本語を学習するのは面白いです。

### 用法 6：助词复合 への / での / との / からの / までの

| 形式 | 例 |
|---|---|
| **での** | アメリカでの滞在 |
| **への** | 母への手紙 |
| **との** | 友達との約束 |
| **からの** | 海外からの観光客 |
| **までの** | 学校までの案内 |

## 2.11 も（也 / 同类 / 强调多）

### 用法 1：类同（"也"）
私は中国人です。小王も中国人です。

⚠️ 替换原句的「は/が」── 不能同时用 は + も。

### 用法 2：疑问词 + も + 否定 = 全面否定
- 何も食べない。（什么都不吃。）
- どこにも行かない。（哪儿都不去。）
- 誰も来なかった。（谁都没来。）

⚠️ 「も」+ が/を → が/を **省略**（写作「にも」「でも」「からも」「とも」「へも」则保留）。

### 用法 3：数量词 + も = 主观觉得多
- 1時間も待ちました。（足足等了 1 小时！）
- 45,000 円もするんですか。（竟然 4 万 5 千日元！）

## 2.12 や（不完全列举）

> A や B など = A 啊 B 啊等等

| や（不完全） | と（完全） |
|---|---|
| パンや飴などを買った | パンと飴を買った（只两样）|

## 2.13 か（疑问 / 选择 / 不定指代）

### 用法 1：句末疑问 ── あなたは学生ですか。
⚠️ 传统书面体规范是用句号「。」；但现代日语在口语、漫画、SNS、广告以及省略「か」的疑问句中也大量使用「？」。初学正式书面写作时优先用「。」。

### 用法 2：选择并列（A か B）── 月曜日か火曜日に行く
### 用法 3：疑问词 + か = 不定指代
- 何か食べましたか。（吃了什么吗？）
- 誰か来た。（有人来了。）

## 2.14 へ（方向）

> 北京へ行く。⚠️ 读作 **e**（不是 he）！

## 2.15 とか（口语列举）

| 表达 | 接续 | 语体 |
|---|---|---|
| A とか B とか | 名词 / 动词原形都可以 | 口语 |
| A や B など | 只能名词 | 偏书面 |

例：ゲームとかネットショッピングとかします。

## 2.16 でも（举例提议）

⚠️ 不是接续词"但是"，而是"举例性建议"。
- 北京ダックでも食べませんか。（要不吃个北京烤鸭？）
- お茶でも飲みましょう。（喝杯茶之类的吧。）

接续：を/が 被吸收；に/へ 保留（NにでもN）。

## 2.17 ずつ（平均分配）

> 数字 / 数量词 + ずつ
- 1人に1個ずつ。（每人1个。）
- 少しずつ上手になった。（一点点变好。）

⚠️ ずつ（等量分配）≠ だんだん（时间渐变）。

## 2.18 だけ vs しか〜ない（限定）

| 形式 | 后接 | 客 / 主观 | 含义 |
|---|---|---|---|
| **だけ** | 可肯可否 | 客观陈述 | 只…… |
| **しか〜ない** | 必须否定 | 主观（少 / 遗憾）| 才…… |

| 例 | 用 |
|---|---|
| 10日だけです | 中性"只有10天" |
| 10日しかありません | 主观"才10天，太少" |

助词搭配：「だけ」+ が/を → が/を **可省略**；「だけ」+ に/で/と/へ → **保留**。

## 2.19 终助词 ね / よ / な / の

| 助词 | 我的状态 | 想要的反应 |
|---|---|---|
| **ね** | 我已知 / 觉得是 | 求对方认同 |
| **よ** | 我知 / 对方不知 | 让对方知道 |
| **な（咏叹）** | 自我感叹 / 独白 / 轻语调 | 綺麗だな / よく頑張ったな |
| **な（禁止）** | **动词原形 + な = 禁止"不要做"** | **行くな！**（别去！）/ **触るな！**（别碰！）|
| **の** | 女性 / 软口气疑问 | 何してるの？ |

⚠️ 终助词与对话对象：「よ」一般需要对话对象（告知/强调给对方）；「ね」既可对人（求认同）也可自言自语（自我确认，如「困ったね…」「綺麗だね…」）。

## 2.20 は vs が 专节

### 核心规则

| 助词 | 用途 | 强调点 |
|---|---|---|
| は | 提示已知话题 | は后面 |
| が | 引出新信息 / 强调主语 | が前面 |

### 双主语句型（は + が）

日语特有的"大小主语"结构：
```
大主语 + は ＋ 小主语 + が ＋ 谓语
```

| 例 | 大主语 | 小主语 | 谓语 |
|---|---|---|---|
| 象は鼻が長い | 象 | 鼻 | 長い |
| 北京は冬が寒い | 北京 | 冬 | 寒い |
| 私は寿司が好き | 私 | 寿司 | 好き |

### 疑问词作主语必用が
- 何が難しいですか。（什么难？）
- 誰が来ましたか。（谁来了？）

回答时也用 が：「日本語の助詞が難しいです」。

# 三、形容词系统

## 3.1 分类与判别

| 类别 | 词尾 | 修饰名词 | 例 |
|---|---|---|---|
| **一类形容词**（い形容词）| 词尾「い」 | **直接接** | 高い・新しい・大きい |
| **二类形容词**（な形容词）| 词典中以"だ"结尾 / 词干独立 | + **な** | 静か・きれい・有名 |

## 3.2 长得像一类的二类陷阱词

⚠️ 写法以「い」结尾但实际是二类：きれい / きらい / 有名（ゆうめい）/ 得意（とくい）。

⚠️ きれい（漂亮）vs きらい（讨厌）── 只差一个元音，意思完全相反！

## 3.3 真正一类（看似不像）

**かっこいい**（帅 / 酷）── 一类形容词。因为后半的「いい」不规则，所以活用时要先变成「かっこよく〜」：かっこ**よく**ない / かっこ**よかった**。

## 3.4 4 时态全表

### 一类形容词（以「大きい」为例）

| 时态 | 丁寧体 | 简体 |
|---|---|---|
| 现在肯定 | 大きいです | 大きい |
| 现在否定 | 大きくないです / 大きくありません | 大きくない |
| 过去肯定 | 大きかったです | 大きかった |
| 过去否定 | 大きくなかったです / 大きくありませんでした | 大きくなかった |

⚠️ **过去否定 = 先否定再过去**：大きい → 大きくない → 大きくなかった。

### 二类形容词 / 名词（共用规则）

⚠️ **二类形容词和名词的时态变化完全一样**。

| 时态 | 丁寧体（学生 / 静か）| 简体 |
|---|---|---|
| 现在肯定 | 学生です / 静かです | 学生だ / 静かだ |
| 现在否定 | 学生ではありません / じゃありません | 学生ではない / じゃない |
| 过去肯定 | 学生でした | 学生だった |
| 过去否定 | 学生ではありませんでした | 学生ではなかった |

### 修饰名词时的接续

| 类别 | 修饰名词 | 例 |
|---|---|---|
| **一类** | 直接接 | 大きい学校 |
| **二类** | + な | 立派な図書館 |
| **名词** | + の | 日本語の本 |

⚠️ **「同じ」是な形容词的例外**：直接修饰普通名词时**不加な**（同じクラス）；但谓语位置仍用「同じです / 同じだ」（これと同じです），后接「の/のに」时则写「同じなのに」。
- ✅ 同じクラス　❌ 同じなクラス
- ✅ これと同じです / 同じなのに

## 3.5 特殊词：いい / よい

⚠️ 「いい」是不规则词 ── 凡是要变形，必须先变回「よい」。

| 形态 | 写法 |
|---|---|
| 现在肯定 | いい / よい |
| 现在否定 | よくない |
| 过去肯定 | よかった |
| 过去否定 | よくなかった |
| 中顿 | よくて |
| 变化 | よくなる |

⚠️ 「かっこいい」继承这套规则：かっこよくない / かっこよかった。

## 3.6 第一连用形（副词化，修饰动词）

| 类型 | 规则 | 例 |
|---|---|---|
| 一类 | 去 い + く | 早く起きる |
| 二类 | + に | 静かに座る |

例：
- 早く起きる。（早起。）
- 静かに座る。（安静地坐。）
- 久しぶりに会った。（久违地见面。）

⚠️ 副词化后必须紧贴动词。

⚠️ 修饰动词 vs 修饰名词：
- 多く集まった（修饰动词）
- 多くのメンバー（修饰名词）

## 3.7 第二连用形（并列 / 中顿）

| 类型 | 规则 | 例 |
|---|---|---|
| 一类 | 去 い + くて | 美味しくて安い |
| 二类 | + で | 便利で人気がある |
| 名词 | + で | 学生で会社員 |

例：
- 高橋さんは明るくて、優しい人です。（高桥又开朗又温柔。）
- ここは静かで、便利だ。（这里又安静又方便。）
- 学校は大きくて、学生も多いです。（学校大，学生也多。）

⚠️ **时态只挂最后一个词**：明るくて、親切でした。

| 形容词 | 名词 | 含义 |
|---|---|---|
| 長い | 長さ | 长度 |
| 高い | 高さ | 高度 |
| 大きい | 大きさ | 大小 |
| 重い | 重さ | 重量 |
| 厚い | 厚さ | 厚度 |
| 深い | 深さ | 深度 |

# 四、动词系统

## 4.1 动词分类（核心！）

⚠️ 所有日语动词的原形（辞书形）都以「う段假名」结尾。

| 类别 | 别名 | 包含 | 判别 |
|---|---|---|---|
| **3 类** | サ变・カ变 | する・来る + 名词+する | 死记 |
| **2 类** | 一段 | 以「る」结尾 **且** る 前一假名为 イ段或エ段 | 看る前假名 |
| **1 类** | 五段 | 其他所有 | 排除法 |

### 判别流程

```
原形 → 是否 来る/する/名+する？
        ├── 是 → 3 类
        └── 否 → 是否"る"结尾？
                ├── 否 → 1 类
                └── 是 → 前假名是イ段或エ段？
                        ├── 是 → 2 类（注意伪一段！）
                        └── 否 → 1 类
```

## 4.2 "伪二类"（必背！）

⚠️ 长得像 2 类，实际是 1 类：

| 动词 | 假名 | 含义 |
|---|---|---|
| **入る** | はいる | 进入 |
| **走る** | はしる | 跑 |
| **帰る** | かえる | 回家 |
| **知る** | しる | 知道 |
| **切る** | きる | 剪 |
| **要る** | いる | 需要 |

### 同假名陷阱：きる

| 写法 | 假名 | 类别 | 含义 |
|---|---|---|---|
| 着る | きる | 2 类 | 穿衣服 |
| 切る | きる | 1 类（伪）| 剪 |

### 来る不规则三读

| 形态 | 读音 | 例 |
|---|---|---|
| ない形 | **こ**ない | 来ない（こない）|
| ます形 | **き**ます | 来ます（きます）|
| 辞書形 | **く**る | 来る（くる）|
| た形 | **き**た | 来た（きた）|

## 4.3 ます形

### 3 类
| 原形 | マス形 |
|---|---|
| する | します |
| 来る | **来ます（きます）** ⚠️ 特殊 |
| 勉強する | 勉強します |

### 2 类（去る + ます）
食べる → 食べます / 見る → 見ます

### 1 类（う段 → イ段 + ます）

| う段 | → イ段 |
|---|---|
| う / く / ぐ / す / つ / ぬ / ぶ / む / る | い / き / ぎ / し / ち / に / び / み / り |

| 原形 | マス形 |
|---|---|
| 買う | 買います |
| 聞く | 聞きます |
| 話す | 話します |
| 待つ | 待ちます |
| 飛ぶ | 飛びます |
| 読む | 読みます |
| 帰る | 帰**ります**（伪一段）|

## 4.4 マス形 4 时态

| 形式 | 后缀 |
|---|---|
| 现在肯定 | ます |
| 现在否定 | ません |
| 过去肯定 | ました |
| 过去否定 | ませんでした |

⚠️ **「来ない」 vs 「来ませんでした」 ── 时间完全不同**！
- 来ない = 现在 / 未来不来
- 来ませんでした = 过去那个时点没来

## 4.5 ない形

### 3 类
- する → しない
- 来る → **こない（来ない）** ⚠️ 不是 くらない

### 2 类（去る + ない）
食べる → 食べない / 見る → 見ない

### 1 类（う段 → ア段 + ない）

| う段 | → あ段 |
|---|---|
| く / ぐ / す / つ / ぬ / ぶ / む / る | か / が / さ / た / な / ば / ま / ら |
| **う** | **わ**（特殊！）|

| 原形 | ない形 |
|---|---|
| 行く | 行かない |
| 読む | 読まない |
| 待つ | 待たない |
| **買う** | **買わない** ⚠️ う→わ |

### ⚠️ 特殊：ある → ない（不是 あらない）

| 形态 | 写法 |
|---|---|
| 简体肯定 | ある |
| 简体否定 | **ない** |
| 丁寧体否定 | ありません |

## 4.6 た形

### 3 类
- する → した
- 来る → 来た（きた）

### 2 类（去る + た）
食べる → 食べた

### 1 类（5 种音便）

| 词尾 | 音便 | 例 |
|---|---|---|
| **く** | い音便 | 書く → 書いた |
| **ぐ** | い音便（带浊）| 泳ぐ → 泳いだ |
| **う / つ / る** | 促音便 | 買う → 買った、待つ → 待った、作る → 作った |
| **ぬ / ぶ / む** | 撥音便 | 死ぬ → 死んだ、飛ぶ → 飛んだ、読む → 読んだ |
| **す** | 无音便 | 話す → 話した |

⚠️ **行く是 1 类里唯一的例外**：行**った**（不是行いた）。

## 4.7 なかった形（简体过去否定）

> ない形 → 把ない当一类形容词 → なかった

| 类别 | 例 |
|---|---|
| 1 类 | 買わない → 買わなかった |
| 2 类 | 食べない → 食べなかった |
| 3 类 | しない → しなかった / こない → こなかった |

## 4.8 4 时态简体 vs 丁寧体 对照

| 时态 | 简体 | 丁寧体 |
|---|---|---|
| 非过去 肯 | 食べる | 食べます |
| 非过去 否 | 食べない | 食べません |
| 过去 肯 | 食べた | 食べました |
| 过去 否 | 食べなかった | 食べませんでした |

## 4.9 て形

### 变化规则 = た形（た→て / だ→で）

| 词尾 | て形 |
|---|---|
| く | いて |
| ぐ | いで（注意浊音）|
| う / つ / る | って |
| ぬ / ぶ / む | んで |
| す | して |

特例：行く → 行**って**。

### て形的用法

#### ① 连续动作 / 先后顺序
朝起きて、運動して、朝食を食べて、会社へ行きます。（早上起床，运动，吃早饭，去公司。）

⚠️ 时态由最后一个动词决定。

#### ② 中顿 / 并列
学校は大きくて、学生も多いです。

#### ③ 原因 / 理由
時間がなくて、行けません。（因为没时间，去不了。）

#### ④ 接补助动词
〜ている / 〜てください / 〜てから / 〜てもいい / 〜てはいけない / 〜てしまう / 〜てみる / 〜てある。

## 4.10 〜ている / 〜ていた

### 三种用法

| # | 用法 | 例 |
|---|---|---|
| 1 | 动作进行 | 今、食べています（正在吃）|
| 2 | 状态结果存续 | もう来ています（来了，还在）|
| 3 | 习惯 / 反复 | 毎朝、新聞を読んでいます |

### ⚠️ 「知る」的特殊用法

| 问 | 肯定 | 否定 |
|---|---|---|
| 知っていますか？ | はい、**知っています** | いいえ、**知りません** ⚠️ |

⚠️ 「知る」的否定不能用「知っていません」！固定形式：**「知りません / 知らない」**。

### ⚠️ まだ〜ていない

| 形 | 中文 |
|---|---|
| まだ食べない | 还不吃（不打算吃）|
| まだ食べ**ていない** | **还没吃**（暂时未吃）|

⚠️ 中文"还没吃"对应日语「まだ〜ていない」。

### ⚠️ ある / いる 选择

| 主语 | 用 |
|---|---|
| 人 / 动物（有意识）| いる / います |
| 物 / 植物 / 抽象（无意识）| ある / あります |

## 4.11 可能形

### 两种意义

| 类型 | 含义 | 例 |
|---|---|---|
| 能力 | 个人能力 | 私は日本語が話せる |
| 可能性 | 客观允许 | ここで食事ができる |

### 变形规则

| 类别 | 规则 | 例 |
|---|---|---|
| 3 类 | する → **できる** / 来る → こられる | 勉強する → 勉強できる |
| 2 类 | 去る + **られる** | 食べる → 食べられる |
| 1 类 | う段 → え段 + **る** | 行く → 行ける / 読む → 読める |

例：
- 私は泳げます。（我会游泳。）
- 日本語が話せる。（会说日语。）
- 日本語を話すことができる。（能说日语。）── 等价句型

### ら抜き言葉（口语省ら）
食べられる → 食べれる / 見られる → 見れる / **来られる（こられる）→ これる** ── 书面 / 正式语用完整 られる。

### 助词变化
能动态时，**を 一般变 が**：魚を食べる → 魚**が**食べられる。

### ⚠️ 非生物主语不用能动态
- ❌ 荷物が届けられる　✅ 荷物が届く（自动词）
- ❌ 会議は始められる　✅ 会議は始まる

### ⚠️ 自带可能义的动词
**見える**（自然能看见）/ **聞こえる**（自然能听见）/ **わかる**（懂）── 不再变能动。

### 見える / 聞こえる vs 見られる / 聞ける

| 表达 | 语感 |
|---|---|
| 見える / 聞こえる | 自然映入眼帘 / 传入耳朵 |
| 見られる / 聞ける | 人为前往 / 努力才能 |

## 4.12 命令形 / 意志形 / 禁止形

| 类型 | 形式 | 例 |
|---|---|---|
| 命令形（直接 / 强硬）| ── | 頑張れ |
| 意志形（自我）| 行こう / 食べよう | ─ |
| 意志形（邀请）| 〜ましょう | 行きましょう |
| 禁止 | 〜てはいけない / 〜てはダメ | ─ |
| 必须 | 〜なければならない / 〜なくてはいけない | ─ |

### 命令形变形规则

| 词类 | 规则 | 例 |
|---|---|---|
| 一类（五段）| う段 → え段 | 行く→**行け** / 飲む→**飲め** / 頑張る→**頑張れ** |
| 二类（一段）| 去る + ろ / よ | 食べる→**食べろ** / 見る→**見ろ** |
| 三类（不规则）| する→**しろ**（或せよ） / 来る→**来い（こい）** | する→**しろ** / 来る→**来い** |

⚠️ 命令形本身**带粗鲁/强硬语感**，多见于父对子、上司对下属（极少数场合）、紧急/警告（「逃げろ！」「止まれ！」）、男性口语。日常请求请用「〜てください」。

## 4.13 自动词 vs 他动词

### 定义

| 概念 | 定义 | 助词 |
|---|---|---|
| **自动词** | 主语自身发出动作，强调状态变化 | 主语 + **が** + 動詞 |
| **他动词** | 对某物施加动作，强调改变状态的原因 | 主语 + 宾语 + **を** + 動詞 |

### 经典对

| 自动词 | 他动词 |
|---|---|
| ドアが**開く**（あく）| ドアを**開ける**（あける）|
| 電気が**つく** | 電気を**つける** |
| ドアが**閉まる** | ドアを**閉める** |
| 〜が**落ちる** | 〜を**落とす** |
| 車が**止まる** | 車を**止める** |
| 〜が**消える** | 〜を**消す** |
| 〜が**集まる** | 〜を**集める** |
| 〜が**始まる** | 〜を**始める** |
| 〜が**届く** | 〜を**届ける** |
| 〜が**上がる** | 〜を**上げる** |
| 〜が**入る** | 〜を**入れる** |
| 〜が**出る** | 〜を**出す** |
| 〜が**動く** | 〜を**動かす** |

### 自他对应辨别规则

| 优先级 | 规则 | 判定 |
|---|---|---|
| ① | 「す」结尾 → 他动词 | 同组另一个 → 自动词 |
| ② | 「ア段 + る」结尾 → 自动词 | 同组另一个 → 他动词 |

### ⚠️ 反向特例：割る / 割れる

| 单词 | 看似 | 实际 |
|---|---|---|
| 割る（わる）| 自 | **他動詞** |
| 割れる（われる）| 他 | **自動詞** |

### 自他兼用：終わる
- 授業が終わります（自）
- 授業を終わります（他）

### 自然现象 / 情绪类 = 全部自动词
- 雨が降る、風が吹く、花が咲く
- 怒る、泣く、笑う

### ⚠️ 入る vs 入れる vs 入れる（可能形）── 读法陷阱

| 写法 | 读法 | 类型 | 助词 |
|---|---|---|---|
| 入る | **はいる** | 自动词 | に |
| 入れる | **いれる** | 他动词 | を |
| 入れる（可能形）| **はいれる** | 可能形 | 主语 が / 目的地 に |

三个读法完全不同！例：
- 家に入る（家に **はいる**：进家）
- 牛乳を入れる（**いれる**：加牛奶）
- 11時以降は入れません（**はいれません**：进不去）

### 易混对组

| 区分 | 说明 |
|---|---|
| **借りる**（かりる）借入 vs **貸す**（かす）借出 | 方向相反！|
| **帰る** 回家 / **戻る** 回出发点 / **返す** 还东西 | 三义 |
| **見る** 主动看 / **見える** 自然能见 / **見せる** 出示 | 三义 |
| **着る**（上半身、套头、外套）vs **履く**（下半身：裤、鞋、袜、裙）| 部位 |
| **入れる**（封闭空间）vs **置く**（开放平面）| 空间形态 |
| **探す**（找新的）vs **捜す**（找丢的）| 同读 |
| **言う**（引用、单句）vs **話す**（互动交流）| 双向性 |
| **教える** 我教别人 vs **習う** 跟别人学 | 方向 |
| **行く** 一次性 vs **通う** 长期反复（常用通っている）| 持续 |
| **住む** 长期居住 vs **泊まる** 短期留宿 | 时长 |

### 開く 两种读音

| 写法 | 读音 | 类型 | 意思 |
|---|---|---|---|
| 開く | ひらく | **自他両用** | 开（书、账户、店、会）|
| 開く | あく | 自动词 | 开了（门、窗、缝隙）|

## 4.14 名词 + する → 3 类动词

⚠️ 动作性名词 + する：

| 名词 | + する |
|---|---|
| 勉強 / 学習 | 勉強する / 学習する |
| 散歩 / 旅行 | 散歩する / 旅行する |
| 連絡 / 注文 | 連絡する / 注文する |
| 結婚 / 案内 / 紹介 | ─ |
| 提出 / 出席 / 参加 | ─ |
| 翻訳 / 操作 / 運転 | ─ |
| 削除 / 復習 / 洗濯 / 掃除 | ─ |

⚠️ 限制：必须是**动作性名词**。

⚠️ 「N**を**する」 ≈ 「N**の**N をする」 ── 两种写法等价：
- 部屋を掃除する = 部屋の掃除をする
- 日本語を勉強する = 日本語の勉強をする

## 4.15 受身形（被动）

### 变形规则

| 类别 | 规则 | 例 |
|---|---|---|
| 1 类（五段）| う段 → あ段 + **れる** | 書く → 書か**れる** / 飲む → 飲ま**れる** |
| 2 类（一段）| 去る + **られる** | 食べる → 食べ**られる** / 見る → 見**られる** |
| 3 类 | する → **される** / 来る → **こられる** | 紹介する → 紹介**される** |

⚠️ 「う」结尾的 1 类动词：あ段为「わ」── 言う → 言**わ**れる、買う → 買**わ**れる。

### 三种用法

| # | 用法 | 句型 | 例 |
|---|---|---|---|
| ① | **直接被动** | A は B に〜（ら）れる | 母に叱られた。（被妈妈骂了。）|
| ② | **间接被动（迷惑受身）** | 主语遭受其他人/事的动作而受困扰 | 雨に降られた。（被雨淋了 / 偏偏下雨。）|
| ③ | **物为主语**（事物/客观陈述）| 物 は〜（ら）れる | この本は1980年に書かれた。（这本书是1980年写的。）|

例：
- 母に叱られた。（被妈妈骂了。）
- 雨に降られて、風邪を引いた。（被雨淋了，感冒了。）
- この本は1980年に書かれた。（这本书是 1980 年写的。）

⚠️ 间接被动多带"困扰 / 不愉快"语感（迷惑受身）：友達に来られて勉強できなかった。（朋友来了害我没法学习。）

⚠️ 行为发出者用「に」；物体来源 / 制作时常用「**によって**」：『源氏物語』は紫式部**によって**書かれた。

## 4.16 使役形（让 / 使）

### 变形规则

| 类别 | 规则 | 例 |
|---|---|---|
| 1 类（五段）| う段 → あ段 + **せる** | 書く → 書か**せる** / 飲む → 飲ま**せる** |
| 2 类（一段）| 去る + **させる** | 食べる → 食べ**させる** |
| 3 类 | する → **させる** / 来る → **こさせる** | 勉強する → 勉強**させる** |

### 助词规则

| 原动词 | 句型 | 助词 |
|---|---|---|
| 自动词 | A は B **に** 〜させる | に |
| 他动词 | A は B **に** 〜を 〜させる | に + を |
| 强制 / 放任色彩 | A は B **を** 〜させる（自动词时）| を（强迫 / 容许）|

例：
- 先生は学生に本を読ませた。（老师让学生读书。── 他动词）
- 母は子供に野菜を食べさせた。（妈妈让小孩吃蔬菜。）
- 父は弟を留学に行かせた。（父亲让弟弟去留学。── 自动词 + を 强制色彩）

### 〜（さ）せてください（请允许我〜）

> 使役て形 + ください = 请让我做〜

例：
- 少し休ませてください。（请让我休息一下。）
- 私にやらせてください。（请让我来做。）

## 4.17 使役受身（被迫 / 不得不）

### 变形规则

> 使役形 + 受身 = 〜させられる

| 类别 | 形式 | 一类常用缩约 |
|---|---|---|
| 1 类 | う段 → あ段 + **せられる** | → 〜**される**（飲ませられる → **飲まされる**）|
| 2 类 | 〜**させられる** | 食べさせられる |
| 3 类 | する → **させられる** / 来る → **こさせられる** | 勉強させられる |

⚠️ 一类缩约规则：〜される 在口语常见。但**「す」结尾的 1 类**（例：話す）**不缩**，保持「話させられる」。

### 含义

被某人强迫做某事（带不情愿 / 抱怨语感）。

例：
- 母に野菜を食べさせられた。（被妈妈逼着吃蔬菜。）
- 上司に毎日残業させられる。（被上司每天逼着加班。）
- 子供のころ、ピアノを習わされた。（小时候被逼着学钢琴。── 一类缩约形）

⚠️ 与单纯使役不同 ── 使役受身**强调"我"的不情愿**。
- 使役：母は私に野菜を食べさせた。（妈妈让我吃蔬菜。── 视点在妈妈）
- 使役受身：私は母に野菜を食べさせられた。（我被妈妈逼着吃。── 视点在我，含抱怨）

# 五、句型与表达

## 5.1 基本句型（名词谓语句）

| 类型 | 句型 | 例 |
|---|---|---|
| 肯定 | 名1 は 名2 です。| わたしは学生です。|
| 疑问 | 名1 は 名2 ですか。| わたしは学生ですか。|
| 否定 | 名1 は 名2 ではありません。| わたしは学生ではありません。|

⚠️ 「では / じゃ」中的「は」也读 **wa**！

否定的三种说法（递降）：ではありません / じゃありません / じゃない。

### NもN / NのN / 中顿で
- 私も中国人です。（我也是中国人。）
- 私の本。（我的书。）
- 私は中国人で、先生です。（我是中国人，也是老师。）── 主语必须相同。

## 5.2 存在句

### あります / います

| 词 | 用于 |
|---|---|
| あります | 无意识（物・植物）|
| います | 有意识（人・动物）|

### 两种句型

| 型 | 句型 | 强调 |
|---|---|---|
| A | 場所 + に + 物/人 + が + あります/います | 某地有什么 |
| B | 物/人 + は + 場所 + に + あります/います | 某物/人在哪里 |

例：
- 教室に学生がいます。（教室里有学生。）
- 学生は教室にいます。（学生在教室。）
- 庭に桜があります。（院子里有樱花。）

### 場所詞 vs 普通名词

⚠️ 非場所詞必须加方位名词：
- ❌ お皿に新聞があります　✅ お皿の上に新聞があります
- ❌ 箱に眼鏡があります　✅ 箱の中に眼鏡があります

### 事件的存在：場所 + で + 事件 + が + あります

| 句型 | 助词 | 例 |
|---|---|---|
| 静态存在 | に | 教室に学生がいる |
| 事件 | **で** | 教室で試験がある |

例：
- 来週、東京でサッカーの試合があります。（下周东京有足球比赛。）
- 昨日北海道で地震がありました。（昨天北海道发生地震。）

## 5.3 愿望表达

### ほしい（想要某物）

> N + が + ほしい（一类形容词活用）

⚠️ 限制：第 1 人称陈述 / 第 2 人称疑问 ✓；第 3 人称用 **ほしがる**。

例：
- 私は車がほしい。（我想要车。）
- 何がほしいですか。（你想要什么？）
- 子供は新しいおもちゃをほしがる。（小孩想要新玩具。）

### 〜たい（想做某事）

> 動詞ます形（去ます）+ たい（一类形容词活用）

| 时态 | 形式 |
|---|---|
| 现在肯定 / 否定 | 食べたい / 食べたくない |
| 过去肯定 / 否定 | 食べたかった / 食べたくなかった |

⚠️ 人称限制：第 1 / 2 人称用 たい；第 3 人称用 **〜たがる**。

### 助词 を / が 可互换（仅他动词）
- 水を飲みたい（强调动作）
- 水が飲みたい（强调对象）

⚠️ 非「を」类动词不能换：
- 大学に入りたい（不能改 を / が）
- 友達に会いたい（不能改 を / が）

### 〜たがる（第三人称想做）

例：父は新しいパソコンを買いたがっている。（父亲想买新电脑。）

## 5.4 经验（〜たことがある）

> 動詞た形 + ことがある = 曾经~过

例：
- 私は上海に3度行ったことがあります。（我去过 3 次上海。）
- 日本料理を食べたことがある。（吃过日料。）
- 富士山に登ったことがありません。（没爬过富士山。）

## 5.5 能力（〜ことができる）

> 動詞原形 + こと + ができる

例：
- 日本語を話すことができます。（能说日语。）── 等价：日本語が話せる。
- ここで写真を撮ることができます。（这里可以拍照。）
- 泳ぐことができません。（不会游泳。）

## 5.6 义务（必须）

| 形式 | 语感 |
|---|---|
| ~なくてはいけない（イケナイ组）| 个人主观 |
| ~なくてはならない（ナラナイ组）| 社会规则、客观 |
| ~なければいけない / ならない | 同上 |

⚠️ "**个人的事**" 用 イケナイ 组 / "**社会的事**" 用 ナラナイ 组。

⚠️ 实际口语中两组常通用，差别较细微，初学不必过分纠结；写作或考试遇到"个人主观 vs 社会规则"对照时再严格区分。

例：
- 11時までに帰らなくてはなりません。（必须在 11 点前回去。）── 宿舍规定
- 家を出る時、電気を消さなければなりません。（出门必须关灯。）
- 明日、日本語で自己紹介しなきゃなりません。（缩约：明天得用日语自我介绍。）

口语缩约：
- なくては → **なくちゃ**
- なければ → **なきゃ**

## 5.7 不必（〜なくてもいい）

> 动词ない形（末尾的「い」变为「くてもいい」）→ 〜なくてもいい
> 例：食べない → 食べなくてもいい / 行かない → 行かなくてもいい

⚠️ 即"否定假设 + 许可" → "也可以不做……"。

例：
- 靴を脱がなくてもいいですよ。（不脱鞋也可以。）
- もう薬を飲まなくてもいい。（已经不用吃药了。）
- 宿題をやらなくてもいい。（不做作业也可以。）

## 5.8 许可与禁止

### ~てもいい（许可）

> 動詞て形 + もいい = 可以~

例：
- ここに座ってもいいですか。（可以坐这吗？）
- 作文は日本語で書いてもいいですよ。
- 音をもう少し大きくしてもいいですか。（音量调大点行吗？）

⚠️ 对长辈不能用陈述句（疑问句征求许可可以）。

### ~てはいけない / ~てはダメ（禁止）

例：
- 男性は女子寮に入ってはいけません。（男生不能进女生宿舍。）
- 教室ではタバコを吸ってはいけません。
- ここで写真を撮ってはいけません。

### ~ないでください（请不要）

> ない形 + で + ください

例：
- 窓を開けないでください。（请不要开窗。）
- 気にしないでください。（请别在意。）
- 大きい声で話さないでください。（请不要大声说话。）

⚠️ 接续是**動詞ない形 + で + ください**（中间是「で」不是「て」）。例：行か**ないで**ください。

### ~てください（请做某事）

> 動詞て形 + ください

例：
- 見せてください。（请给我看。）
- ゆっくり話してください。（请慢点说。）
- 静かにしてください。（请安静。）

⚠️ 「ください」带命令性 → 对长辈不用。紧急 / 口语可省略「ください」：「ちょっと待って」「助けて」。

### ~をください（请给我物品）

> 物 + を + ください

例：
- これをください。（请给我这个。）
- ハンバーガーを3個ください。（请给我 3 个汉堡。）

⚠️ 数量词紧贴 ください 前。

### でございます（です 的尊敬版）

- はい、ABC会社でございます。（您好，ABC 公司。）
- お手洗いは2階でございます。

## 5.9 推量

### 〜でしょう（推測 / 確認）

「でしょう」── 两种意义：

| 意义 | 用法 | 声调 |
|---|---|---|
| **推測** | 我猜想……吧 | 末尾**升调** ↗ |
| **確認** | 你也是这么想的吧？ | 末尾**降调** ↘ |

### 接续

| 词类 | 接续 |
|---|---|
| 动词 / 一类形容词 | 简体 + でしょう |
| 名词 / 二类形容词 | **直接** + でしょう（**词干**即可，**不加だ**）|

⚠️ **名词 / 二类形容词后不加だ**！
- ❌ 学生だでしょう / ですでしょう
- ✅ 学生でしょう

### 例

確認用法：
- この問題は難しいでしょう？（这题挺难吧？）
- 王さんはまだ2年生でしょう。（小王还是 2 年级吧？）
- 見て、このコップ、かわいいでしょう。（你看，这杯子很可爱吧？）

推測用法（天气预报最常用）：
- 明日は雨が降るでしょう。（明天大概会下雨。）
- あの店がいいでしょう。（那家店应该不错吧。）
- エントさんも国へ帰らなかったでしょう。

⚠️ **天気予報**不能 100% 断言 → 必须用「でしょう」。

「だろう」是「でしょう」的简体形式。

### 〜でしょうか（更礼貌）
今、いいでしょうか？── 比「ですか」更柔和。

## 5.10 比较

### より（比较基准）
飛行機は車より速い。（飞机比车快。）

### のなかで / で（范围）

| 类型 | 用法 |
|---|---|
| 范围词（日本、クラス、公園）| **で** |
| 非范围词（果物、人群）| **〜の中で** |

例：
- 日本で富士山が一番高い山。（日本最高山是富士山。）
- 果物の中でいちごが一番好き。（水果里最喜欢草莓。）

### 一番（最~）
> N1 は N2 で 一番 ~
- 富士山は日本で一番高い山です。（富士山是日本最高的山。）

## 5.11 条件 / 让步

### 〜ても（让步）
> 即使~也~

接续：
- 动词て形 → て + も（食べても）
- 一类形容词 → 〜くても（高くても）
- 二类形容词 / 名词 → で + も（静かでも / 学生でも）

例：明日、雨が降っても、旅行に行きたい。（即使明天下雨，也想去旅行。）

### 〜ば（一般条件 / 假设）

**接续**：
| 词类 | 形式 |
|---|---|
| 1 类动词 | う段 → え段 + ば（行く → 行け**ば**）|
| 2 类动词 | 去る + **れば**（食べる → 食べ**れば**）|
| 3 类动词 | する → **すれば** / 来る → **くれば** |
| 一类形容词 | 去 い + **ければ**（高い → 高**ければ**）|
| 二类形容词 / 名词 | + **なら(ば)**（静かなら / 学生なら）|
| 否定 | 〜ない → 〜なければ |

**含义**：① 自然规律 / 必然结果；② 一般假设；③ 谚语 / 普遍真理。

例：
- 春になれば、桜が咲く。（一到春天樱花就开。）
- 安ければ買います。（便宜的话就买。）
- このボタンを押せば、水が出ます。（按这个按钮，水就会出来。）

⚠️ 〜ば 句**后项一般不接命令、请求、意志**（前项是状态时除外，如「暇なら手伝って」可改用 なら）。

### 〜たら（最通用 / 口语）

**接续**：动词 / 形容词 / 名词的 **た形** + ら
- 食べた → 食べ**たら**
- 高かった → 高かっ**たら**
- 静かだった → 静かだっ**たら**
- 学生だった → 学生だっ**たら**

**含义**：① 假设；② "做完 A 后……"（前后接续紧密）；③ 偶然发现。后项**最自由** ── 可接命令、请求、意志、过去。

例：
- 雨が降ったら、行きません。（如果下雨就不去。）
- 駅に着いたら、電話してください。（到了车站请打电话。）
- 朝起きたら、雪が積もっていた。（早上起来，发现雪堆了一地。── 偶然发现）

### 〜なら（基于对方话题的假设 / 建议）

**接续**：
- 动词 / 形容词 → **简体** + なら（行くなら / 高いなら）
- 二类形容词 / 名词 → **直接** + なら（静かなら / 学生なら）

⚠️ **"简体"包含 4 种时态**（与丁寧体相对的所有形态）—— 都可接 なら：
| 时态 | 例 | 含义 |
|---|---|---|
| 现在肯定 | 食べる**なら** | 如果（要 / 将要）吃的话 |
| 现在否定 | 食べない**なら** | 如果不吃的话 |
| 过去肯定 | 食べた**なら** | 如果已经吃了的话 |
| 过去否定 | 食べなかった**なら** | 如果（之前）没吃的话 |

因为 〜なら 是"基于对方话题的回应"，所以时态自然跟随对方的语境（对方说"我吃了" → 你回 食べた**なら**；对方说"我打算吃" → 你回 食べる**なら**）。

**含义**：① 接对方话题"如果是〜的话"；② 给建议；③ 主题对比假设。

例：
- 日本へ行くなら、京都がいいですよ。（要去日本的话，京都不错。）
- 寿司なら、あの店が一番だ。（说到寿司，那家店最棒。）
- 安いなら、買います。（便宜的话我就买。）

⚠️ **〜なら 的时间顺序与其他三者相反** ── "行くなら 〜" 通常指**去之前**的准备，"行ったら 〜" 指**到了之后**。

### 〜と（必然 / 自然结果）

**接续**：动词 / 形容词 **辞书形** + と / 名词 + だ + と
- 行く + と / 高い + と / 静かだ + と / 学生だ + と

**含义**：① 自然规律 / 机械结果 / 操作说明；② 习惯反复；③ 偶然发现（小说常见）。

例：
- 春になると、桜が咲く。（一到春天樱花就开。）
- このボタンを押すと、ドアが開きます。（一按这个按钮，门就开。）
- 家に帰ると、母はもう寝ていた。（一回家，发现妈妈已经睡了。── 偶然发现）

⚠️ 〜と 后项**绝对不能**接命令、请求、意志、许可 ──「❌ 春になると、桜を見に行こう」要改 たら / ば。

### 四个条件对比（核心总结）

| 形式 | 适用场景 | 后项可接命令/意志? | 时间顺序 |
|---|---|---|---|
| **〜ば** | 自然规律 / 普遍假设 / 谚语 | △（前为状态时可）| 假设 |
| **〜たら** | **最通用**：假设 / 完成 / 偶然 | ✅ | 假设 / 先后 |
| **〜なら** | 基于对方话题 / 建议 / 主题 | ✅ | **可反向**（先〜なら 后准备）|
| **〜と** | **必然 / 机械 / 自然 / 操作说明** | ❌ | 必然 |

经典区分例：
- 日本へ**行くなら**、辞書を買ったほうがいい。（要去日本的话，去**之前**买字典。）
- 日本へ**行ったら**、辞書を買ったほうがいい。（到了日本**之后**再买字典。）
- 日本へ**行けば**、日本語が上達する。（去日本的话，日语会进步。── 一般假设）
- 日本へ**行くと**、毎回お土産を買う。（每次去日本都买伴手礼。── 习惯反复）

⚠️ 选不出来时**默认用 たら** ── 适用面最广。

## 5.12 目的（移动）

### 動詞ます形 + に + 行く / 来る

| 形式 | 例 |
|---|---|
| 动作性名词 + に + 行く | 散歩に行く / 買い物に行く |
| 動詞ます形 + に + 行く | 食べに行く / 見に行く |

例：
- 学校が終わった後、本を買いに行きます。（放学后去买书。）
- 学生たちはもみじを見に来ました。（学生们来看红叶了。）
- コーヒーを飲みに公園に行く。（去公园喝咖啡。）

## 5.13 原因 / 理由

### から（主观理由）

接续：
- 一类形容词 / 动词简体 + から
- 二类形容词 / 名词 + だ + から
- 丁寧体：~ですから

例：
- ルールですから、仕方がありません。（这是规定，没办法。）
- 暑いから、窓を開けてください。（热，请开窗。）
- もう大丈夫だから、心配しないでください。（已经没事了，别担心。）

特点：
- 主观主张（"我认为是因为…"）
- 后项可接命令 / 意志 / 邀请 / 推荐（〜てください / 〜ましょう / 〜（よ）う）
- 语气可强可弱，日常会话最常用

### ので（客观理由，更正式 / 商务首选）

接续（⚠️ 与 から 关键差异）：
- 一类形容词 / 动词简体 + ので
- 二类形容词 / 名词 + **な** + ので（だ → な）
- 丁寧体可直接 ~ですので

例：
- 雨(あめ)が降(ふ)っているので、出(で)かけません。（在下雨，所以不出门。）
- 静(しず)かなので、よく勉強(べんきょう)できます。（很安静，所以能好好学习。）
- 学生(がくせい)なので、お金(かね)がありません。（因为是学生，所以没钱。）
- すみませんが、頭(あたま)が痛(いた)いので、早(はや)く帰(かえ)らせていただけませんか。（不好意思，头疼，能让我早点回家吗？）

特点：
- 客观陈述（"事实上是因为…"）
- 更柔和、更礼貌——**商务邮件 / 客户应对首选**
- 商务习惯避免在「ので」后直接接命令形

### から vs ので 辨析

| 维度 | から | ので |
|---|---|---|
| 语感 | 主观主张 | 客观陈述 |
| 礼貌度 | 中性 / 偏低 | 高 / 商务推荐 |
| 接续点（名词·二类）| + **だ** + から | + **な** + ので |
| 后接命令 / 意志 | 自由 | 避免（商务尤其） |
| 适用 | 日常会话 / 强主张 | 商务 / 委婉解释 |

### 形容词 + くて / で（原因）

| 类型 | 形 | 例 |
|---|---|---|
| 一类 | 〜くて | 時間がなくて、行けません |
| 二类 | 〜で | 暇で、テレビを見ました |

⚠️ 与 3.7 中顿（くて / で）外形相同，靠后项判别：
- **3.7 中顿**：单纯并列、罗列两个属性（彼(かれ)は背(せ)が高(たか)くて、ハンサムです）
- **5.13 原因**：后项必须是「被动结果 / 不可控反应 / 不可能」——驚(おどろ)く / 困(こま)る / 行(い)けない / 疲(つか)れる / 大変(たいへん)だ など
- 后项**不能**接 命令 / 意志 / 邀请 / 推荐（要接这些必须改用 から / ので）

## 5.14 〜んです / 〜のだ（解释 / 强调）

> 表"解释 / 说明 / 引入话题"

「んです」 = 「のです」的口语缩约 ── 名词化 + です。

### 4 种形式对照（语体 × 口语化）

| 形式 | 语体 | 口语化 | 场景 |
|---|---|---|---|
| **のです** | 礼貌 | 书面 | 正式书面 / 论文 |
| **んです** | 礼貌 | 口语 | **日常 / 商务最常用** ⭐ |
| **のだ** | 简体 | 书面 | 男性 / 文学 / 独白 |
| **んだ** | 简体 | 口语 | 朋友 / 家人对话 |

→ 本节实操以「んです」为主，「のだ / んだ」为衍生形（同接续规则）。

### 接续

| 词类 | 接续 |
|---|---|
| 动词 | 简体 + んです（行くんです / 行ったんです）|
| 一类形容词 | 简体 + んです（高いんです）|
| **二类形容词** | + な + んです（便利**な**んです）|
| **名词** | + な + んです（学生**な**んです）|

⚠️ 不要写"学生のんですか" ── 名词 / な形容词后要加 **な**。

例：
- 顔色が悪いですね。何かあったんですか。（脸色不好，发生什么了？）
- このオンライン決済の使い方がわからないんです。（不会用线上支付。）
- 風邪を引いたんです。（感冒了。）

### どうして〜んですか（为什么）

> どうして + 简体 + んですか

| 形式 | 语感 |
|---|---|
| **どうして** | 较正式 |
| **なんで** | 口语、随意 |

例：
- どうして話さなかったんですか。（为什么没说话？）
- どうしてこんなに値段が高いんですか。（为什么这么贵？）
- 高橋さんは、どうして行かないんですか。（高桥同学为什么不去？）

### 回答用「〜んです」

A：昨日、なんで休んだの？　B：風邪を引いたんです。

### ⚠️ 关键易错点

⚠️ **回答时不要再加「から」**！「んです」本身已表理由 → 不能叠加「から」。
- ❌ 風邪を引いたんですから。
- ✅ 風邪を引いたんです。

### ⚠️ 时态选择

| 场景 | 时态 |
|---|---|
| 活动还没开始 → "为什么不来？" | 来ないんですか |
| 活动已经结束 → "为什么没来？" | 来なかったんですか |

### 〜んですが（柔和铺垫）

> ~んです + 接续助词 が，更柔和 / 未完待续。

例：
- すみません、お土産を買いたいんですが…（不好意思，我想买点伴手礼……）
- このパソコンの使い方がわからないんですが、教えてください。
- ちょっとうるさいんですが、少し静かにしてください。（有点吵，请小声一点。）

## 5.15 〜にする / 〜になる（变化）

⚠️ **核心**：「する」和「なる」是一对动词，接续完全相同，但主客观不同。

### なる vs する 对比

| 形 | 主客观 | 翻译 | 例 |
|---|---|---|---|
| **なる** | **客观** | 变得…… | 春になる / 髪が長くなる |
| **する** | **主观** | 把……变得…… | 髪を短くする / 音を大きくする |

### 接续公式（两者相同）

| 词类 | 接续 |
|---|---|
| **一类形容词** | 去 い + **く** + する / なる |
| **二类形容词 / 名词** | + **に** + する / なる |

| 词 | する形 | なる形 |
|---|---|---|
| 大きい（一类）| 大き**く**する | 大き**く**なる |
| 短い（一类）| 短**く**する | 短**く**なる |
| 静か（二类）| 静か**に**する | 静か**に**なる |
| きれい（二类）| きれい**に**する | きれい**に**なる |
| 春 / 夏（名词）| ── | 春**に**なる |
| ジュース（名词）| ジュース**に**する | ── |

### 句子结构（する変化）

> **[主语] を [変化内容（く / に）] する**

例：
- 音をもう少し大きくしてください。（请把音量调大一点。）
- 部屋をきれいにしてください。（请把房间打扫干净。）
- 髪を短くする。（把头发剪短。）

### なる 例
- 北京は寒くなりました。（北京变冷了。）
- 肌が綺麗になった。（皮肤变好了。）
- そろそろ 11 時になります。（差不多 11 点了。）
- 夏になりました。（夏天来了。）

### する変化的两种语境

| 语境 | 例 |
|---|---|
| ① 主观调整 / 改变事物 | 音を大きくする / 髪を短くする |
| ② 决定 / 选择（购物 / 点单）| これにします（我要这个）|

例：
- A：お決まりですか？　B：これにします。（决定好了吗？──我要这个。）
- 私はコーヒーにします。（我要咖啡。）
- パーティーは日曜日にしました。（聚会定在了周日。）

⚠️ なる（客观）vs する（主观）对照：
- パーティーは日曜日になります（"自然"是周日）
- パーティーは日曜日にしました（"我决定"周日）

⚠️ 与 5.30「〜ようにする / 〜ようになる」区分 ── 本节用于**名词 / 形容词**的状态变化；动词的"努力 / 变得能〜"用「〜ようにする / 〜ようになる」。

## 5.16 〜とき（~的时候）

⚠️ **铁律 0**：とき前**永远用简体形**——即使整句礼貌体（句末 ます/です），とき前的动词 / 形容词也必须用简体。
- ❌ 食べ**ます**とき / 高い**です**とき / 行き**ました**とき / 静か**でした**とき
- ✓ 食べ**る**とき / 高いとき / 行**った**とき / 静か**だった**とき

### 接续

| 形态 | 含义 |
|---|---|
| 動詞辭書形 + とき | 一次性时点 / **前句动作后做**（"…之前"）|
| 動詞テイル形 + とき | 持续 / 反复状态（例：勉強しているとき）|
| 動詞た形 + とき | 已完成的时刻 / **前句动作先做**（"…之后"）|
| い形容词（简体）+ とき | ~的时候（高いとき / 高かったとき）|
| **な形容词 + な + とき** | ~的时候（暇**な**とき）|
| **名词 + の + とき** | ~的时候（学生**の**とき）|

⚠️ 形容词不能接「~ているとき」── テイル形只属于动词。

### ⚠️ 原型 vs た形 决定先后

| 时间形 | "时" 这件事 | 前句动作时点 |
|---|---|---|
| **動詞原形 + 时** | 还没发生 | **后做** |
| **動詞た形 + 时** | 已经做完 | **先做** |

经典对比：
- 日本へ**行く**時、パソコンを買った。（去之前买 → 在**中国**买）
- 日本へ**行った**時、パソコンを買った。（到了后买 → 在**日本**买）
- 家に**帰る**時、果物を買ってください。（回家路上 / 还没到家时买。）

记忆口诀：**"原形 = 后做，た形 = 先做"**。

## 5.17 〜後で / 〜てから（之后）

### 〜後で（あとで／时间顺序）

| 接续 | 形式 | 例 |
|---|---|---|
| 動詞 | た形 + 後で | 食べ**た**後で / 帰っ**た**後で |
| 動作性名词 | N + **の** + 後で | 食事(しょくじ)**の**後で / 授業(じゅぎょう)**の**後で |

⚠️ 前面动词必须**た形**（即使整句讲未来）。

例：
- 食べた後で、サッカーをします。（吃完后踢足球。）
- 授業の後で、宿題をします。（下课后写作业。）
- 仕事(しごと)が終(お)わった後で、お酒(さけ)を飲(の)みに行きます。（下班后去喝酒。）

### 〜てから（〜之后再 / 必要前提）

| 接续 | 形式 | 例 |
|---|---|---|
| 動詞 | **て形** + から | 食べ**て**から / 帰っ**て**から |

⚠️ 名词 / 形容词**不能**接 〜てから——只能接动词。

例：
- お仕事が終わってから、一緒(いっしょ)に飲みましょう。（工作做完后一起喝酒吧。）
- 家(うち)に帰ってから、手(て)を洗(あら)ってください。（回家以后请洗手。）

### 〜後で vs 〜てから 辨析

| 维度 | 〜後で | 〜てから |
|---|---|---|
| 接续 | た形 / N + の | **て形** |
| 名词接续 | ✓（N + の）| ✗ 只接动词 |
| 语感 | **单纯**时间顺序 | 强调 A 是 B 的**必要前提** |
| 翻译 | "之后" | "之后再" / "才〜" |

经典对比：
- 食べ**た後で**、テレビを見ます。（吃完看电视——时间顺序）
- 食べ**てから**、テレビを見ます。（吃完才看电视——强调先吃）

### 〜てから 第二用法：起点（"自从〜以来"）

> 〜てから + 経過時間(けいかじかん) + 動詞 = 自从 A 以来过了多久

例：
- 日本に来てから、3年(さんねん)経(た)ちました。（来日本以来过了 3 年。）
- 大学(だいがく)を卒業(そつぎょう)してから、もう 10年(じゅうねん)になります。

### 5.16 / 5.17 三兄弟系统

| 句型 | 接续 | 含义 | 焦点 |
|---|---|---|---|
| **た形 + 時**（5.16）| た形 | 已完成的时刻 | 时点 |
| **〜後で**（5.17）| た形 / N + の | 之后 | 时间顺序 |
| **〜てから**（5.17）| て形 | 之后再 / 必要前提 | A → B 因果链 |

## 5.18 〜と思う

> ~と思う = 觉得 / 认为~

⚠️ **铁律 1**：前面必须**简体句**（与 5.16 铁律 0 同源）。

⚠️ **铁律 2**：「と思う」**第 1 人称专用**——描述他人内心想法必须用「**〜と思っている**」（与 5.3 ほしがる / たがる + ている 同源逻辑）。

### 接续（4 词类）

| 词类 | 接续 | 例 |
|---|---|---|
| 動詞 | 简体（4 时态）+ と思う | 行く / 行かない / 行った / 行かなかった + と思う |
| 一类形容词 | 简体 + と思う | 高(たか)い / 高くない / 高かった + と思う |
| **二类形容词** | + **だ** + と思う | 静か**だ**と思う / 静かじゃない**と思う** |
| **名词** | + **だ** + と思う | 学生**だ**と思う / 学生じゃない**と思う** |

⚠️ **关键差异**：5.18 二类·名词后加「**だ**」——和 5.13 ので / 5.14 んです / 5.16 とき 的「**な**」**完全不同**！

| 节 | 二类 / 名词后 |
|---|---|
| 5.13 ので | + **な** + ので |
| 5.14 んです | + **な** + んです |
| 5.16 とき | + **な** / **の** + とき |
| **5.18 と思う** | + **だ** + と思う |

### 否定形（"我不觉得"／"我觉得不"）

- 小李**会**来（我觉得）= 李さんは来ると思います。
- 小李**不会**来（我觉得）= 李さんは**来ない**と思います。 ⭐⭐ 推荐
- 我**不觉得**小李会来 = 李さんが来るとは思いません。（较婉转）

⭐ **日语习惯**：把否定贴前句（**前否**），不贴 と思う 自身（**后否**）。
- 中文"我**不**觉得 X" → 日语 "我觉得**不** X"

### ⚡ 〜と思う vs 〜と思っている

| 形式 | 适用人称 | 含义 |
|---|---|---|
| **〜と思う** | **第 1 人称**（我）| 此刻我的看法 |
| **〜と思っている** | **第 3 人称**（他 / 她）| 持续的"想法状态"，外部观察 |

例：
- わたしは留学(りゅうがく)に行きたい**と思う**。（我想去留学。）
- 田中さんは留学に行きたい**と思っている**。（田中（说他）想去留学。）

→ **日语文化层**：不能"擅自断言他人内心" → 第 3 人称必用「〜と思っている」（与 5.3〜たがる + ている 同源逻辑）。

### 〜たいと思う / 〜たいと思っている（柔化主张）

3 个礼貌度递增：

| 说法 | 语感 |
|---|---|
| 行きたいです | 直接 / 主张强 |
| 行きたい**と思います** | 弱化主张 / 较婉转 |
| 行きたい**と思っています** | 持续想法 / 最委婉 / **商务首选** ⭐ |

例：
- 来年(らいねん)、留学に行きたい**と思っています**。（明年想去留学。）

### 跨节回拾

- 2.6 用法 3：「と思う 前限简体」首次接触 ✓ 本节升级为正式句型
- 5.3 〜たがる + ている：内心 vs 外部观察分流（同源逻辑）
- 4.10 〜ている 用法 ①：状态持续
- 5.16 铁律 0：从句简体 + 主句礼貌

## 5.19 反问 / 确认（じゃありませんか）

### 核心句型

| 语体 | 形式 |
|---|---|
| 礼貌 | **〜じゃありませんか** ／ **〜ではありませんか**（更书面） |
| 简体 | **〜じゃない？** ／ **〜じゃないか？** |
| 口语 | **〜じゃないですか** |

### 三大用法

| 用法 | 含义 | 例 |
|---|---|---|
| **① 反问·确认**（说话人**已有答案**，向对方求同感）| "（这）难道不是 ~ 吗？" | 今は呉先生の授業じゃありませんか。（现在不是吴老师的课吗？） |
| **② 温和提议·提醒** | "（这）不就是 ~ 吗？" | あれ、◯◯さんの財布じゃありませんか。（咦，那不是您的钱包吗？） |
| **③ 意外发现·惊讶** | "（这）不就是 ~ 嘛！" | これ、田中さんじゃないですか。（这不就是田中嘛！） |

### 结构本质

- **じゃありません**（不是 = です 的礼貌否定）+ **か**（疑问）
- **否定 + 疑问 = 反问** → 实际表"确认/惊讶/同意诱导"
- 类比中文"难道不是~吗"

### 接续规则（4 词类）

| 词类 | 接续 | 例 |
|---|---|---|
| **名词** | N + じゃありませんか | 学生**じゃありませんか** |
| **二类形容词** | 词干 + じゃありませんか | 静か**じゃありませんか** |
| **一类形容词** | くない + ですか（**不用じゃ**）| 高**くないですか** |
| **動詞** | ない形 + ですか／んですか | 来ない**んですか** ／ 行か**ないですか** |

⚠️ **重点**：一类形容词和动词**不能用「じゃありませんか」**，要用「**くないですか**」「**ないですか / んですか**」对应形式。

### 与 5.9 でしょう 的区别

| 句型 | 说话人立场 | 期待回应 |
|---|---|---|
| **でしょう**（推测）| 不太确定 | 对方告知 |
| **でしょう**（確認 ↘）| 双方共有信息 | 对方附和 |
| **じゃありませんか** | **已经有答案** | 对方同意/承认 |

例对照：
- 今日は寒い**でしょう**。（推测：今天大概冷吧）
- 今日は寒い**じゃないですか**！（反问：今天不冷吗！="今天明明就冷"）

### 与真否定问的区别（语调）

| 句子 | 语调 | 意思 |
|---|---|---|
| 学生じゃありません。 | 降 ↘ | 陈述："不是学生" |
| 学生じゃありませんか？ | **升 ↑** | 反问："难道不是学生吗？" |

### 升级形（N4 / 商务）

- **〜のではないでしょうか**：更委婉，常用于建议/委婉表达观点
  - 例：もう少し考えた方がいい**のではないでしょうか**。（不再考虑一下会不会更好？）
- **〜じゃないかと思います**：含蓄表达意见（"我觉得是不是~"）

### tosho-trading 商务场景

| 场景 | 句 |
|---|---|
| 提醒客户错误 | この金額、間違いじゃありませんか。 |
| 委婉确认 | 田中部長、ご指示の件はこちらの資料じゃありませんか。 |
| 软提议 | この方法のほうがよいのではないでしょうか。 |

## 5.20 ましょう / ましょうか / ませんか

| 句型 | 语气 | 含义 |
|---|---|---|
| 〜ましょう | 直接 | 一起做吧 |
| 〜ましょうか | 中等 | 一起做好吗？/ 我帮你做~吧？ |
| 〜ませんか | 最委婉 | 要不一起~？ |

⚠️ ませんか **只能**邀请别人一起；ましょう / ましょうか 还可以表"我自己来做"。

例：
- 私が押しましょう。（我来推。）
- お手伝いしましょうか。（我帮你吧？）
- 一緒に行きませんか。（要不要一起去？）

## 5.21 どうですか / いかがですか

| 用法 | 礼貌 |
|---|---|
| どうですか | 一般 |
| いかがですか | 更礼貌（敬语）|

两种含义：① 询问感想 / 评价；② 提议（不能用过去时）。

例：
- 先週の試験はどうでしたか。（上周的考试怎么样？）── 询问感想
- コーヒーはどうですか。（喝杯咖啡怎么样？）── 提议

## 5.22 〜どのくらい（多少 / 多久）

### 本质与拼写

- 词源：**どの**（疑问连体词，章一）+ **くらい / ぐらい**（程度·概数）
- 三种写法**等价**：**どのくらい** / **どのぐらい** / **どれくらい**（ぐ 更口语 / どれ 也通用）

### 含义：万能"程度疑问代词"

询问**时间 / 数量 / 距离 / 程度**——比"何時間 / 何個 / いくら"更宽泛、更口语。

### 典型搭配

| 问什么 | 例 |
|---|---|
| 多久（时间） | 旅行（りょこう）に**どのくらい**かかりましたか。 |
| 多少（钱） | **どのくらい**お金（かね）がかかりましたか。 |
| 多远（距离） | 駅（えき）まで**どのくらい**ですか。 |
| 多大程度 | この本（ほん）は**どのくらい**難（むずか）しいですか。 |
| 多少（数量） | お酒（さけ）は**どのくらい**飲（の）めますか。 |

### 回答形式对应（くらい / ぐらい = 大约）

| 问 | 答 |
|---|---|
| どのくらいかかりますか。 | 3 時間（じかん）**くらい**かかります。 |
| 駅までどのくらいですか。 | 10 分（ぷん）**くらい**です。 |
| お金はどのくらい？ | 1 万円（まんえん）**くらい**です。 |

📌 回答里的「くらい/ぐらい」=「大约/差不多」(N5 高频概数后缀)

### ⚠️ 与「どの」严格区分

| 词 | 词性 | 必接 | 例 |
|---|---|---|---|
| **どの** | 连体词（章一） | **必接名词** | **どの**本ですか。（哪本书？）|
| **どのくらい** | 程度疑问代词 | 独立 / 修饰程度 | 本は**どのくらい**ありますか。（书有多少？）|

### 与其他疑问词的分工

| 疑问词 | 问什么 | 精确度 |
|---|---|---|
| 何時（なんじ）| 几点 | 精确 |
| 何時間（なんじかん）| 几小时 | 精确 |
| いつ | 什么时候 | 时间点 |
| いくら | 多少钱 | 钱专用 |
| 何人（なんにん）/ 何個（なんこ）/ 何冊（なんさつ）| 数量 + 量词 | 精确 |
| **どのくらい** | 概数 / 程度 / 多久 / 多少 | **宽泛** ⭐ |

### tosho-trading 商务场景

| 场景 | 句 |
|---|---|
| 询问交货期 | 納品（のうひん）まで**どのくらい**かかりますか。 |
| 询问数量 | 在庫（ざいこ）は**どのくらい**ありますか。 |
| 询问预算 | ご予算（よさん）は**どのくらい**でしょうか。 |
| 询问经验 | 業界（ぎょうかい）の経験（けいけん）は**どのくらい**ですか。 |

## 5.23 授受动词与〜て授受

### 基础授受三动词

| 动词 | 视点 | 方向 | 例 |
|---|---|---|---|
| **あげる** | 给方视点（我→人 / 内→外）| 给出去 | 私は彼にプレゼントをあげた。|
| **くれる** | 收方视点（人→我 / 外→内）| 给我（方）| 友達が私にプレゼントをくれた。|
| **もらう** | 收方视点（我从人那里得）| 收到 | 私は友達にプレゼントをもらった。|

⚠️ **内外关系核心**：
- 自己 / 自己家人 / 自己公司 = **内**
- 别人 / 别人家人 / 别的公司 = **外**
- 由外向内 → 必须用 **くれる**（不能用「あげる」从他人角度说）
- 由内向外 → **あげる**

### 〜て授受（动作的施与受）

| 句型 | 含义 | 例 |
|---|---|---|
| **〜てあげる** | 我 / 内 帮 别人 / 外 做（带恩惠语感）| 私は友達に日本語を教えてあげた。|
| **〜てくれる** | 别人 / 外 帮 我 / 内 做（感激语感）| 友達が日本語を教えてくれた。|
| **〜てもらう** | 我 / 内 请 / 让 别人 / 外 做（自己受益）| 私は友達に日本語を教えてもらった。|

例：
- 母は弟に本を読んであげた。（妈妈给弟弟读书。）
- 田中さんが手伝ってくれた。（田中帮了我。）
- 先生に推薦状を書いてもらった。（请老师写了推荐信。）

⚠️ 「〜てあげる」对**上位 / 长辈不宜直接用**（有施恩感），改用「お〜する」谦让形。

### 敬体（敬语版授受）

| 平体 | 尊敬 / 谦让 | 用法 |
|---|---|---|
| あげる | **差し上げる**（谦让）| 我给上位者 |
| くれる | **くださる**（尊敬）| 上位者给我 |
| もらう | **いただく**（谦让）| 我从上位者收到 |
| 〜てあげる | お〜する / お〜いたす | 我为上位者做 |
| 〜てくれる | **〜てくださる** | 上位者为我做 |
| 〜てもらう | **〜ていただく** | 请上位者为我做 |

例：
- 先生が説明してくださった。（老师为我讲解了。）
- 先生に説明していただきました。（请老师为我讲解了。）── 同义、视点不同
- 社長にお土産を差し上げました。（送了社长伴手礼。）

⚠️ 「**〜ていただけませんか**」是请求**最礼貌**的句型 ──「教えていただけませんか。」（能请您教我吗？）

## 5.24 〜ながら（同时进行）

**接续**：動詞ます形（去ます）+ ながら

**含义**：① 一边〜一边〜（同一主语两个动作同时进行）；② 〜然而（书面让步，N3）。

例：
- 音楽を聞きながら勉強する。（一边听音乐一边学习。）
- 歩きながら携帯を見るのは危ない。（边走边看手机很危险。）
- お茶を飲みながら話しましょう。（一边喝茶一边聊吧。）

⚠️ 主要 / 重点动作放**后面**：「テレビを見ながらご飯を食べる」= 重点是"吃饭"。

⚠️ 两个动作必须**同一主语**才能用「ながら」。

## 5.25 〜たり〜たりする（列举动作）

**接续**：動詞た形 + り、〜た形 + り + する

**含义**：① 列举若干代表性动作（不分先后 / 不完全列举）；② 反复交替；③ 列举形容词。

例：
- 休みの日は本を読んだり、映画を見たりします。（休息日读书或看电影什么的。）
- 雨が降ったり止んだりしている。（雨下下停停。── 反复交替）
- 部屋が広かったり狭かったりする。（房间有的宽有的窄。── 形容词列举）

⚠️ **句末「する」时态决定全句时态** ── 過去用「〜たり〜たりした」。

⚠️ 与「〜て、〜て」区别：て表先后顺序，たり表无序举例。

## 5.26 〜たほうがいい / 〜ないほうがいい（建议）

**接续**：
- 動詞た形 + ほうがいい（推荐做）
- 動詞ない形 + ほうがいい（不推荐做 ── **ない形不变 た形**）

**含义**：建议、劝告（语感比单纯「〜たらいい」更强、更具体）。

例：
- 早く寝たほうがいいですよ。（早点睡比较好哦。）
- タバコは吸わないほうがいいです。（最好别抽烟。）
- 病院に行ったほうがいいです。（最好去医院。）

⚠️ 肯定建议用**た形**（不是辞书形）── ❌ 行くほうがいい / ✅ 行**った**ほうがいい。

⚠️ 用于"明显比较好"的具体劝告；对长辈直接用偏失礼，宜加「〜と思います」缓和。

## 5.27 〜つもり / 〜予定 / 〜ようと思う（打算与计划）

### 〜つもり（个人意志 / 主观打算）

**接续**：動詞辞書形 / ない形 + つもり

例：
- 来年、日本に留学するつもりです。（明年打算去日本留学。）
- 今日は飲まないつもりです。（今天不打算喝酒。）
- そんなつもりはありません。（没那个意思。）

### 〜予定（已确定的客观安排）

**接续**：動詞辞書形 / N + の + 予定

例：
- 来週、出張する予定です。（下周预定出差。）
- 会議は3時の予定です。（会议预定 3 点。）

⚠️ つもり = **主观意志**、予定 = **客观日程**。

### 〜（よ）うと思う（现在产生的打算 / 较具体）

**接续**：意志形 + と思う

| 形 | 例 |
|---|---|
| 〜（よ）うと思う | 现在 / 刚刚的打算 |
| 〜（よ）うと思っている | 一段时间以来的打算（更持续）|

例：
- 来年、車を買おうと思います。（明年想买车。）
- 日本語を勉強しようと思っている。（一直想学日语。）

⚠️ 三者强度：**予定 ＞ つもり ＞ ようと思う**（确定度递减）。

## 5.28 〜そうだ（様態 + 伝聞）

### 様態用法（看起来 / 快要〜）

**接续**：
| 词类 | 接续 |
|---|---|
| 动词 | ます形（去ます）+ そう |
| 一类形容词 | 词干（去 い）+ そう |
| 二类形容词 | 词干 + そう |

⚠️ **例外**：
- いい → **よさ**そう（不能"いそう"）
- ない → **なさ**そう（形容词の「ない」）
- 名词不能用样态 そうだ

**含义**：① 视觉判断"看起来〜"；② 即将发生"快要〜"。

例：
- このケーキ、美味しそう。（这蛋糕看起来好吃。）
- 雨が降りそうです。（看样子要下雨。）
- 元気がなさそうですね。（你看起来没精神。）

### 伝聞用法（据说 / 听说）

**接续**：**普通形** + そうだ（包括 〜だそうだ）
- 動詞普通形 + そうだ：行くそうだ / 行ったそうだ
- 一类形容词 + そうだ：高いそうだ
- 二类形容词 / 名词 + **だ** + そうだ：きれい**だ**そうだ / 学生**だ**そうだ

例：
- 天気予報によると、明日は雨だそうです。（据天气预报，明天下雨。）
- 田中さんは結婚したそうです。（听说田中结婚了。）
- このレストランは美味しいそうだ。（听说这家餐厅好吃。）

⚠️ **两者最重要区别**：
- 様態：動詞ます形 / 形容词词干 + そう（"看起来要〜"）
- 伝聞：普通形 + そう（"据说〜"）

例：雨が**降り**そうだ（看样子要下雨）vs 雨が**降る**そうだ（据说会下雨）。

## 5.29 〜らしい（推断 / 据闻 / 典型性）

### 用法 1：根据信息推断（〜らしい ≈ 据可靠信息看似〜）

**接续**：
- 动词 / 一类形容词 → 普通形 + らしい
- 二类形容词 / 名词 → 词干（**不加だ**）+ らしい

例：
- 田中さんは病気らしい。（田中似乎病了 ── 有客观信息支持。）
- もう着いたらしい。（看样子到了。）
- あの店は美味しいらしい。（听说那家店很好吃。）

### 用法 2：典型性（"像〜的样子 / 有〜风范"）

**接续**：N + らしい（作形容词使用）

例：
- 男らしい人。（有男人味的人。）
- 春らしい天気。（很有春天感觉的天气。）
- 子供らしくない。（不像小孩。）

⚠️ 〜らしい vs 〜そうだ（伝聞）vs 〜ようだ：
- **〜そうだ**（伝聞）：单纯转述、信息源明确
- **〜らしい**：基于客观信息推断、信息源含糊
- **〜ようだ / 〜みたい**：基于自己感觉 / 直接观察

## 5.30 〜ようにする / 〜ようになる（习惯化 / 能力变化）

### 〜ようにする（努力做 / 设法）

**接续**：動詞辞書形 / ない形 + ようにする

**含义**：①（一次性决心）尽量做〜；②（长期习惯）养成〜的习惯。

例：
- 毎日運動するようにしています。（坚持每天运动。）
- 甘いものを食べないようにする。（尽量不吃甜的。）
- 約束を忘れないようにしてください。（请尽量别忘了约定。）

### 〜ようになる（变得能〜 / 变成〜）

**接续**：動詞辞書形 / 可能形 / ない形 + ようになる

**含义**：能力 / 习惯 / 状态的变化（之前不能 / 不做 → 现在能 / 做）。

例：
- 日本語が話せるようになった。（变得会说日语了。）
- 最近、本を読むようになった。（最近开始读书了。）
- 朝食を食べないようになった。（变得不吃早餐了。── 也常说「食べなくなった」）

⚠️ 与 5.15 〜にする / 〜になる 区别：
- **〜にする / 〜になる**：状态变化（名词 / 形容词）── 静かになる / 髪を短くする
- **〜ようにする / 〜ようになる**：**动作 / 能力**变化（动词）── 食べるようになる

## 5.31 〜のに（逆接 / 不满）

**接续**：
| 词类 | 接续 |
|---|---|
| 动词 / 一类形容词 | 简体 + のに |
| 二类形容词 / 名词 | + **な** + のに（静か**な**のに / 学生**な**のに）|

**含义**：① 逆接"明明〜却〜"；② 含意外、不满、遗憾语气。

例：
- 雨が降っているのに、彼は出かけた。（明明在下雨他却出门了。）
- もう4月なのに、まだ寒い。（都 4 月了还冷。）
- 一生懸命勉強したのに、試験に落ちた。（拼命学习了却没考上。）

⚠️ 〜のに vs 〜が / 〜けど：
- **〜が / 〜けど**：单纯转折，中性
- **〜のに**：带说话人的**意外 / 不满 / 遗憾**情绪

⚠️ 二类形容词 / 名词后**必须加な**：❌ 学生のに / ✅ 学生**な**のに。

⚠️ 〜のに 后**不能接命令、请求、意志** ── 因为是对已发生事实的吐槽。

# 六、时态与体

## 6.1 时态总览

⚠️ 日语只有 **4 类**（远少于英语 12 类）：

| # | 时态 | 中文对照 |
|---|---|---|
| ① | 非过去 + 肯定 | 是 / 现在 / 将会 |
| ② | 非过去 + 否定 | 不是 / 现在不 / 将来不 |
| ③ | 过去 + 肯定 | 过去是 |
| ④ | 过去 + 否定 | 过去不是 |

⚠️ **日语没有"将来时"** ── "现在"和"将来"用同一时态。

## 6.2 时态 + 时间词必须一致

⚠️ 如果句子里有过去时间词，整个句子必须用过去时！

| 中文 | 日语错例 | 日语正例 |
|---|---|---|
| 昨天天气不好 | 昨日の天気は良くないです ❌ | 昨日の天気は良くなかったです ✅ |
| 上周的考试很难 | 先週の試験は難しいです ❌ | 先週の試験は難しかったです ✅ |

⚠️ 中文形容词不变，靠时间词判断；日语**时间词 + 形容词时态必须同时变**。

## 6.3 现在状态是否成立决定时态

| 句子 | 含义 |
|---|---|
| 王さんと李さんは高校時の友達です | 现在还是朋友 |
| 王さんと李さんは高校時の友達でした | 以前是朋友，现在不是 |

## 6.4 进行体 vs 完整体

| 体 | 形式 | 用法 |
|---|---|---|
| 完整体 | 普通形（食べる / 食べた）| 动作整体 / 完成 |
| 持续体 | 〜ている / 〜ていた | 动作进行 / 状态持续 |

### 〜ている 3 种用法
1. 动作进行：今、食べている
2. 状态结果存续：来ている / 結婚している / 北京に住んでいる
3. 习惯 / 反复：毎日、新聞を読んでいる

### 「見た」vs「見ていた」

| 形 | 意思 |
|---|---|
| 試合を見た | 看过 / 看完了 |
| 試合を見ていた | 当时正在看 |

## 6.5 完成 / 状态相关补助形

- **〜てある** = 已~好（状态）
- **〜てしまう** = 完成 / 遗憾（"~了"）
- **〜てみる** = 试试~

### 〜てある（状态结果存续，由人为动作产生）

| 项 | 〜てある | 〜ている |
|---|---|---|
| 接续 | **他动词て形 + ある** | 自动词 / 他动词 te 形 + いる |
| 强调 | 有人**做完**留下的状态 | 自然状态 / 持续动作 |
| 助词 | **が**（被动作物为主语）| が / を 视动词性而定 |

例：
- 黒板に名前**が書いてある**。（黑板上写着名字 — 有人写完留下的状态。）
- ドア**が開けてある**。（门是开着的 — 有人开过没关。）vs ドア**が開いている**。（门开着 — 不强调谁开的。）

# 七、敬语体系

## 7.1 总览

| 类别 | 用途 |
|---|---|
| **丁寧体（也叫敬体，です/ます）** | 普通敬语，正式 |
| **尊敬語** | 抬高对方 / 第三者 |
| **謙讓語** | 降低自己 |
| **美化語** | お~ / ご~ 接头 |

## 7.2 丁寧体 vs 簡体

| 形态 | 使用场合 |
|---|---|
| 丁寧体（です/ます）| 正式 / 对外人 / 对**别人家**的长辈 |
| 簡体（普通形）| 家人之间 / 亲密朋友 / 日记 |

⚠️ "对长辈用丁寧体"指别人家或外人的长辈。涉及自家父母时分两种情况：
① **在家中对父母讲话**可用簡体（亲密关系）
② **对外人提到自家父母**时用谦称「父」「母」（不用「お父さん」「お母さん」），动词部分仍按对方关系决定敬/简

## 7.3 美化语：お / ご 接头

| 前缀 | 用法 | 例 |
|---|---|---|
| ご | 接**汉语词** | ご家族 / ご主人 / ご指導 / ご無沙汰 |
| お | 接**和语词 / 部分汉语词** | お父さん / お元気 / お土産 / お疲れ様 |

⚠️ 称呼**自己家人**一般**不加**这些前缀；称呼**对方家人**则**必加**。

## 7.4 自己 vs 别人 ── 内外区别

| 关系 | 自己的（谦称） | 别人的（尊称）|
|---|---|---|
| 爸爸 | 父（ちち）| お父さん |
| 妈妈 | 母（はは）| お母さん |
| 爷爷 / 外公 | 祖父（そふ）| おじいさん |
| 奶奶 / 外婆 | 祖母（そぼ）| おばあさん |
| 哥哥 | 兄（あに）| お兄さん |
| 姐姐 | 姉（あね）| お姉さん |
| 弟弟 | 弟（おとうと）| 弟さん |
| 妹妹 | 妹（いもうと）| 妹さん |
| 丈夫 | 夫（おっと）/ 主人 | ご主人 |
| 妻子 | 妻（つま）/ 家内 | 奥さん |
| 儿子 | 息子（むすこ）| 息子さん |
| 女儿 | 娘（むすめ）| 娘さん |

⚠️ **长音差一个就完全不同**：
- おじ**い**さん（爷爷）vs おじさん（叔叔）
- おば**あ**さん（奶奶）vs おばさん（阿姨）

## 7.5 称呼后缀

| 后缀 | 用法 |
|---|---|
| さん | 通用敬称 |
| 様（さま）| 更礼貌（书信、客户）|
| 君（くん）| 男性 / 同辈 / 下属 |
| ちゃん | 亲昵 |
| 先生 | 老师 / 医生 / 律师 / 议员等专业人士 |

⚠️ 称呼别人加「さん」，自我介绍**不加**「さん」。

## 7.6 第一 / 第二人称

| 自称 | 性别 | 礼貌度 |
|---|---|---|
| 私（わたし）| 男女通用 | 高 / 正式 |
| 僕（ぼく）| 男性 | 中（柔和）|
| 俺（おれ）| 男性 | 低（粗野）|
| わたくし | 男女通用 | 最高 |
| あたし | 女性 | 口语 |

⚠️ 知道对方名字 → 用 **"姓 + さん"**，**不用 あなた**！

## 7.7 言う / 食べる / 行く 系列敬语

| 一般 | 尊敬语（对方）| 谦让语（自己）|
|---|---|---|
| 言う | おっしゃる | 申す / 申し上げる |
| 食べる / 飲む | 召し上がる（めしあがる）| いただく |
| 行く / 来る | いらっしゃる / おいでになる | 伺う（うかがう）/ 参る（まいる）|
| いる | いらっしゃる | おる |
| する | なさる | いたす |
| 見る | ご覧になる | 拝見する（はいけんする）|
| 知っている | ご存知だ（ごぞんじだ）| 存じている（ぞんじている）|
| 聞く / 問う | ── | 伺う（うかがう）/ 承る（うけたまわる）|
| あげる / くれる / もらう | ── | 差し上げる / くださる / いただく |

例：
- 私は王と申します。（我叫小王。）── 自谦
- 先生がおっしゃった。（老师说……）── 尊敬

## 7.8 行く / 来る 的视点

⚠️ 中文里"来 / 去"靠**听者位置**判断；**但日语靠"说话人的视点"**！

| 动词 | 朝向 |
|---|---|
| 来る | 朝**说话人所在地方**移动 |
| 行く | 朝**说话人之外的地方**移动 |

## 7.9 婉拒文化：ちょっと

⚠️ 「ちょっと…」是日本人婉拒的关键词。听到「ちょっと」就放弃追问理由。

例：
- 小王：明日はどうですか？
- 高桥：明日はちょっと…（→ 不追问）

## 7.10 でございます

服务行业、客服、电话接待用 **でございます**（です 的尊敬版）。

# 八、接续词与副词（仅语法规则）

## 8.1 接续词

### 转折 / 让步

| 词 | 风格 |
|---|---|
| **でも** | 口语，新句开头 |
| **しかし** | 书面 |
| **が**（接续助词）| 中性，附前句末 |
| **けど / けれども** | 口语缓冲 |

### 因果

| 词 | 用法 |
|---|---|
| **だから / ですから** | 所以（接续词，新句开头）|
| **から / ので** | 因为（接续助词，前句末）|

⚠️ 「だから」不能直接连前句末，必须放新句开头。
- ✅ パンが美味しい。だから好き。
- ❌ パンが美味しいだから好き。

### 顺接 / 并列 / 累加

| 词 | 用法 |
|---|---|
| **そして** | 然后、而且；可表先后 |
| **それから** | 然后、接着；可表先后 / 累加 |
| **それに** | 而且、再加上（累加）；**不能表先后** |
| **その上** | 而且、在~之上 |
| **また** | 还、又 |

例：9時から11時まで勉強する。そして、11時半に寝る。（先后，不能用それに）

### 话题转换

| 词 | 用法 |
|---|---|
| **ところで** | 对了 / 顺便问一下 |
| **じゃ / じゃあ** | 那么 |

### 例示 / 假设

| 词 | 用法 |
|---|---|
| **例えば（たとえば）** | 比如 |
| **もし** | 假如 |

## 8.2 副词搭配规则

### 必接否定的副词

- **あまり** + 否定 = 不太~
- **全然（ぜんぜん）** + 否定 = 完全不~
- **そんなに** + 否定 = 没那么~（带感情）

例：
- 私は全然休みません。（我完全不休息。）
- 私はあまり休みません。（我不怎么休息。）
- そんなに難しくないです。（没那么难。）

### 必接肯定的副词

⚠️ 初级阶段，**とても**主要接肯定（非常〜）。中级会遇到「とても〜ない」的特殊用法 = "怎么也〜不"（如「とてもできない」"怎么也做不到"、「とても信じられない」"实在难以相信"）。但写作和初级表达里仍以"とても+肯定"为主。
- 初级：❌ とても難しくない → ✅ あまり / そんなに 難しくない
- 中级特殊：✅ とてもできない（怎么也做不到）/ とても信じられない（实在难以相信）

### 频度副词（频率高低排列）

| 频率 | 副词 | 必接 |
|---|---|---|
| 100% | いつも | 灵活 |
| ~90% | 毎日 | 灵活 |
| ~70% | よく | 灵活 |
| ~40% | 時々（ときどき）| 灵活 |
| ~30% | たまに | 灵活 |
| ~15% | ほとんど | **通常接否定**（少数接肯定表"几乎全部"）|
| ~5% | あまり | **必否定** |
| 0% | 全然 | **必否定** |

⚠️ 「よく」有两个意思：① 频率"经常"；② 程度"好 / 出色"（よくできました）。

### もう vs まだ vs また

| 副词 | 接 | 含义 |
|---|---|---|
| **もう** | 肯定 / 否定 | 已经~ / 已经不~ |
| **まだ** | 肯定 / 否定 | 还在~ / 还没~ |
| **また** | 独立 | 再、又 |

⚠️ 「まだ」+ 否定 ≠ 「もう」+ 否定：
- もう食べた = 已经吃了
- もう食べない = 已经不吃了
- まだ食べていない = **还没吃**

⚠️ 「まだ」和「また」读音相近，意思完全不同。

### 数量副词（搭配规则）

| 副词 | 用法 |
|---|---|
| **たくさん** | 万能（人 / 物 / 动物）|
| **大勢（おおぜい）** | **仅限人** |
| **少し / ちょっと** | 一点（书 / 口）|
| **全部** | 全部、所有（数量副词）|
| **約（やく）** | 大约、大概（估算副词）|

⚠️ 「大勢の人」✓ / 「大勢の果物」❌（"大勢"只用于人）。

### 数量字 + も = 主观觉得多
> 数词 + も + 谓语 = "竟然~"

例：
- 1時間も待ちました。（足足等了 1 小时。）
- 4 つも食べました。（竟然吃了 4 个。）

### 接续辅助副词（顺序连接）

| 副词 | 用法 |
|---|---|
| **まず** | 首先 |
| **次に** | 接下来 |
| **それから** | 然后 |
| **最後に** | 最后 |

### 推量 / 决心副词（常与でしょう / ます形配合）

| 副词 | 用法 |
|---|---|
| **きっと** | 一定 / 想必（推量）|
| **必ず（かならず）** | 一定（决心）|
| **ぜひ** | 务必（请求 / 邀请）|
| **絶対に** | 绝对 |
| **やはり / やっぱり** | 果然 |
| **もちろん** | 当然 |

# 九、中级语法（N3）

> 本章收录约 50 个 N3 核心句型。多数为 N4 基础上扩展的细微功能差异、书面 / 半正式表达、推断与让步、复合表达。每条按"接续公式 + 含义 + 例句"组织。

## 9.1 表达可能与变化

### 〜ようにする（努力做 / 设法）

**接续**：動詞辞書形 / ない形 + ようにする
**含义**：努力使〜成为习惯 / 尽量〜

例：
- 毎日野菜を食べるようにしている。（一直坚持每天吃蔬菜。）
- 遅刻しないようにしてください。（请尽量别迟到。）
- 早寝早起きするようにしましょう。（养成早睡早起的习惯吧。）

⚠️ 与 5.30 同 ── N3 强调"长期努力"语感。

### 〜ようになる（变得能〜 / 形成习惯）

**接续**：動詞辞書形 / 可能形 / ない形 + ようになる
**含义**：从不能 / 不做 → 能 / 做（变化）

例：
- 子供が一人で歩けるようになった。（小孩变得能自己走了。）
- 最近、自炊するようになった。（最近开始自己做饭了。）
- 漢字が読めるようになりたい。（想能读汉字。）

### 〜なくなる / 〜くなる（消失 / 变成）

**接续**：
- 動詞ない形 → ない去い + **くなる**（食べなくなる）
- 一类形容词 → 去 い + くなる
- 名词 / 二类形容词 → + になる

**含义**：状态消失或转变。

例：
- 最近、彼から連絡が来なくなった。（最近他不再联系了。）
- 部屋が暗くなった。（房间变暗了。）
- 子供が静かになった。（小孩变安静了。）

### 〜にくい / 〜やすい（难 / 易于〜）

**接续**：動詞ます形 + にくい / やすい（一类形容词活用）
**含义**：动作 / 性质上的难易。

例：
- この字は読みにくい。（这字很难读。）
- このペンは書きやすい。（这笔好写。）
- 風邪を引きやすい季節。（容易感冒的季节。）

### 〜すぎる（过度）

**接续**：
- 動詞ます形 + すぎる
- 一类形容词去 い + すぎる
- 二类形容词词干 + すぎる

**含义**：太过〜（多含贬义）。

例：
- 食べすぎてお腹が痛い。（吃太多肚子疼。）
- この問題は難しすぎる。（这题太难了。）
- 静かすぎてかえって怖い。（太安静反而吓人。）

⚠️ 例外：いい → **よ**すぎる、ない → な**さ**すぎる。

### 〜だす / 〜はじめる / 〜つづける / 〜終わる（复合动词）

**接续**：動詞ます形 + だす / はじめる / つづける / 終わる
**含义**：起始 / 持续 / 终了的体助动词。

| 形 | 含义 | 例 |
|---|---|---|
| 〜だす | **突然**开始（多用于不可控）| 急に泣きだした。|
| 〜はじめる | 有意识地开始 | 日本語を習いはじめた。|
| 〜つづける | 持续做 | 3時間も走り続けた。|
| 〜終わる | 做完 | 本を読み終わった。|

例：
- 突然、雨が降りだした。（突然下起雨来。）
- 来月からヨガをはじめます。（下个月开始练瑜伽。）
- 彼は話し続けた。（他一直说不停。）

⚠️ 〜だす ≠ 〜はじめる：だす 强调"出乎意料 / 不可控"，はじめる 强调"意志开始"。

## 9.2 时间关系

### 〜うちに / 〜ないうちに

**接续**：
- 動詞辞書形 / ている形 / 一类形容词 + うちに
- 名词 + の + うちに
- 動詞ない形 + うちに

**含义**：① 在〜的期间内（趁着〜）；② 在还没〜之前（〜ないうちに）。

例：
- 若いうちに、いろんな国へ行きたい。（趁年轻想去各种国家。）
- 熱いうちに食べてください。（请趁热吃。）
- 暗くならないうちに帰ろう。（趁天还没黑回家吧。）

### 〜あいだ / 〜あいだに（期间 vs 期间中某点）

**接续**：動詞ている形 / 形容词 / 名词 + の + あいだ / あいだに

**含义**：
- **〜あいだ**：整个期间持续做某事
- **〜あいだに**：在那段期间内的某个时间点做完某事

例：
- 母が寝ているあいだ、私はテレビを見ていた。（妈妈睡觉的整段时间我都在看电视。）
- 母が寝ているあいだに、買い物に行った。（趁妈妈睡觉去买了东西。）
- 夏休みのあいだに本を3冊読んだ。（暑假期间读了 3 本书。）

⚠️ 〜あいだ → 后项**持续**动作；〜あいだに → 后项**一次性**动作。

### 〜たびに（每当〜）

**接续**：動詞辞書形 / N + の + たびに
**含义**：每次〜都〜（含情感反应）。

例：
- 彼に会うたびに昔を思い出す。（每次见到他都想起从前。）
- 旅行のたびに写真をたくさん撮る。（每次旅行都拍很多照片。）
- このCDを聞くたびに泣いてしまう。（每次听这张 CD 都会哭。）

⚠️ 与「〜ごとに」区别：たびに 强调情感反应；ごとに 强调机械重复。

### 〜たとたん / 〜たとたんに（〜的瞬间）

**接续**：動詞た形 + とたん（に）
**含义**：刚〜就〜（意外、瞬间发生）。

例：
- ドアを開けたとたん、猫が飛び出した。（一开门猫就冲出来了。）
- 立ち上がったとたんに、めまいがした。（一站起来就头晕。）
- 彼の顔を見たとたん、泣き出した。（一看见他的脸就哭了。）

⚠️ 后项**必须是意外 / 不可控**的事情，不接命令、意志。

### 〜ところ / 〜ているところ / 〜たところ（动作的三阶段）

**接续**：
| 形式 | 阶段 | 含义 |
|---|---|---|
| 動詞辞書形 + ところ | 即将开始 | 正要〜 |
| 動詞ている形 + ところ | 进行中 | 正在〜 |
| 動詞た形 + ところ | 刚结束 | 刚〜 |

例：
- 今から出かけるところです。（正要出门。）
- 今、食べているところです。（正在吃。）
- 今、駅に着いたところです。（刚到车站。）

### 〜まま（保持原状）

**接续**：
- 動詞た形 / ない形 + まま
- 一类形容词 + まま
- 二类形容词 + な + まま
- 名词 + の + まま

**含义**：保持〜的状态不变（多含本应改变却没改变的语感）。

例：
- メガネをかけたまま寝てしまった。（戴着眼镜就睡着了。）
- 靴を履いたまま部屋に入った。（穿着鞋就进了房间。）
- 昔のままだ。（还是从前的样子。）

### 〜て以来（自从〜以来）

**接续**：動詞て形 + 以来
**含义**：从某动作发生后一直〜（多用于较长期、较重要的事）。

例：
- 日本へ来て以来、毎日忙しい。（自从来日本以后每天都忙。）
- 彼に会って以来、連絡していない。（自从见过他以后没联系过。）
- 卒業して以来、彼とは会っていない。（毕业以后就没见过他。）

⚠️ 〜以来 后项常用「ずっと〜ている / 〜ていない」── 表持续。

## 9.3 推断与判断

### 〜はずだ / 〜はずがない

**接续**：
- 动词 / 一类形容词 → 简体 + はずだ
- 二类形容词 → + な + はずだ
- 名词 → + の + はずだ

**含义**：
- **〜はずだ**：理应〜、按理说应该〜（基于根据的推断）
- **〜はずがない**：不可能〜（强否定推断）

例：
- 田中さんはもう着いたはずだ。（田中应该已经到了。）
- 彼が嘘をつくはずがない。（他不可能撒谎。）
- 子供にこんな問題が解けるはずがない。（小孩不可能解出这种题。）

⚠️ 〜はずだ ≠ 〜べきだ。はずだ = 推断；べきだ = 应该（义务建议）。

### 〜わけだ（原来如此 / 自然得出结论）

**接续**：
- 动词 / 一类形容词简体 + わけだ
- 二类形容词 + な + わけだ
- 名词 + の / という + わけだ

**含义**：① 推断出"难怪〜 / 也就是说〜"；② 自然得出的结论。

例：
- 雪が降っているの？道理で寒いわけだ。（在下雪？难怪冷。）
- 10年も日本にいたのか。日本語が上手なわけだ。（在日本 10 年了，难怪日语好。）
- つまり、もう間に合わないというわけだ。（也就是说，已经来不及了。）

⚠️ 〜わけがない = 不可能（同 〜はずがない）；〜わけではない = 并不是（局部否定）。

### 〜にちがいない（一定〜 / 肯定〜）

**接续**：
- 动词 / 一类形容词简体 + にちがいない
- 二类形容词 / 名词 + にちがいない（**不加 だ**）

**含义**：根据强烈推断"绝对〜"（书面 / 较正式）。

例：
- 彼は犯人にちがいない。（他一定是犯人。）
- この絵は本物にちがいない。（这幅画肯定是真品。）
- 彼女は知っているにちがいない。（她肯定知道。）

### 〜らしい（N3 复习强化）

**接续**：
- 动词 / 一类形容词简体 + らしい
- 二类形容词 / 名词 + らしい（**不加 だ**）

**含义**：① 根据可靠信息推断；② 接 N 表"有〜风范"。

例：
- 山田さんは結婚したらしい。（听说山田结婚了。）
- あの店は美味しいらしい。（听说那家店好吃。）
- 男らしい性格。（很有男人味的性格。）

⚠️ 三者强度排序：**にちがいない ＞ はずだ ＞ らしい ＞ ようだ / みたい**。

### 〜のではないか / 〜じゃない？（委婉推测）

**接续**：
- 简体 + のではないか / んじゃない？
- 二类形容词 / 名词 + な + のではないか

**含义**：① "是不是〜？"（委婉推断、提议）；② 口语「〜じゃない？」更随意。

例：
- 雨が降るのではないかと思う。（我觉得是不是会下雨。）
- もう遅すぎるんじゃない？（是不是已经太晚了？）
- 彼、忘れたんじゃないかな。（他是不是忘了？）

## 9.4 引用与传达

### 〜と言われている / 〜と聞いた（被人说 / 我听说）

**接续**：簡体 + と言われている / と聞いた

例：
- 富士山は日本一の山だと言われている。（人们都说富士山是日本第一山。）
- 彼は来週帰国すると聞いた。（听说他下周回国。）
- 健康にいいと言われている食品。（被认为有益健康的食品。）

### 〜ということだ / 〜とのことだ（据说 / 听说）

**接续**：簡体 + ということだ / とのことだ

**含义**：传达自他人的信息（〜とのことだ 略正式、书信常用）。

例：
- 田中さんは欠席するということです。（据说田中要缺席。）
- 会議は中止になったとのことです。（据悉会议取消了。）
- 今日は雨が降るということだ。（据说今天会下雨。）

### 〜ように言う（命令 / 请求的间接引用）

**接续**：動詞辞書形 / ない形 + ように言う / 頼む / 伝える

**含义**：转达命令、请求"叫〜做 / 别做"。

例：
- 母に早く帰るように言われた。（被妈妈叫早点回家。）
- 静かにするように注意した。（提醒（他）安静。）
- 来週来るように伝えてください。（请转告（他）下周来。）

### 〜なんて / 〜なんか / 〜とか（轻视 / 列举 / 不定）

**接续**：N / 簡体 + なんて / なんか / とか

**含义**：
- **〜なんて**：① 表轻蔑、惊讶；② 强调
- **〜なんか**：① 表轻蔑；② 表"〜之类的"（口语）
- **〜とか**：① 不完全列举（〜啊〜啊）；② 表传闻不确定

例：
- 試験に落ちるなんて、信じられない。（居然没考上，真不敢相信。）
- お酒なんか飲まない。（酒之类的我才不喝。）
- 田中さんは引っ越したとか聞いた。（好像听说田中搬家了。）
- 来週、東京とか大阪とかに行く予定だ。（下周打算去东京啦大阪啦。）

⚠️ 〜なんか + 否定 = "〜什么的（不）"；〜なんて 表强烈情感反应。

## 9.5 程度与限定

### 〜ほど / 〜くらい（程度）

**接续**：N / 簡体 + ほど / くらい（≈ ぐらい）

**含义**：
- **〜ほど**：① 表程度"〜的程度"；② 〜ほど〜ない 比较；③ 〜ば〜ほど 越〜越〜
- **〜くらい**：① 表轻微程度"差不多〜 / 至少〜"；② 表轻视

例：
- 死ぬほど疲れた。（累得要死。）
- 北京は東京ほど大きくない。（北京没有东京那么大。）
- 自分の名前くらい書けるよ。（自己的名字总能写吧。）
- 30分くらいかかる。（大约要 30 分钟。）

⚠️ 〜ほど 多用于较大程度 / 比较；〜くらい 多用于"约 / 起码"。

### 〜だけ / 〜しか〜ない / 〜ばかり（限定）

**接续**：N / 動詞辞書形 + だけ / しか + 否定 / ばかり

**含义**：
- **〜だけ**：仅、只（中性限定）
- **〜しか〜ない**：除了〜以外没有（必否定，强调少）
- **〜ばかり**：①光〜 / 净〜（多含贬义）；② 大约（数量）

例：
- これだけで十分です。（这些就够了。）
- 千円しかない。（只有 1000 日元。）
- 弟は遊んでばかりいる。（弟弟整天就知道玩。）
- お菓子ばかり食べないで。（别光吃零食。）

### 〜さえ / 〜だけでなく（甚至 / 不仅）

**接续**：
- N + さえ / N + だけでなく〜も
- 動詞ます形 + さえすれば（只要〜就〜）

**含义**：
- **〜さえ**：连〜都〜（表极端）
- **〜さえ〜ば**：只要〜就〜
- **〜だけでなく〜も**：不仅〜还〜

例：
- ひらがなさえ読めない。（连平假名都读不了。）
- 薬を飲みさえすれば治る。（只要吃药就能好。）
- 彼は英語だけでなく中国語も話せる。（他不仅会英语还会汉语。）

### 〜は〜が（话题对比 / 一部分肯定）

**句型**：N1 は〜が、N2 は〜

**含义**：对比两个主题（同一句框架下并列对照）。

例：
- 寿司は好きですが、刺身は苦手です。（喜欢寿司，但不爱生鱼片。）
- 漢字は読めますが、書けません。（汉字能读，但写不出来。）
- 値段は安いが、品質はよくない。（价格便宜但质量不好。）

## 9.6 目的与原因

### 〜ように / 〜ために（目的的两种）

**接续**：
- 動詞辞書形 / ない形 / 可能形 + ように（非意志 / 自然结果 / 希望）
- 動詞辞書形 + ために（意志 / 主动目的）
- N + の + ために（为了〜）

**含义**：
- **〜ように**：前后**主语可不同** / 动词**多为不可控**（可能形、自动词、ない形）
- **〜ために**：前后**主语必须相同** / 动词**必须可控**（意志动词）

例：
- 日本語が話せるように毎日練習している。（为了能说日语每天练习。── 可能形 + ように）
- 日本に留学するために、お金をためている。（为了去日本留学在存钱。── 意志 + ために）
- 子供のために働く。（为了孩子工作。）
- 風邪を引かないように気をつけてください。（请小心别感冒。── ない + ように）

⚠️ 选择关键：动词**有无意志**。
- 「合格する」（自然结果）→ ように
- 「勉強する」（意志动作）→ ために

### 〜ため / 〜ためだ（原因）

**接续**：
- 动词 / 形容词简体 + ため
- 名词 + の + ため

**含义**：原因（书面，正式）。

例：
- 雪のため、電車が止まった。（因下雪电车停了。）
- 体が弱いため、運動できない。（因身体弱不能运动。）
- 道が混んでいるため、遅刻しました。（因路堵迟到了。）

### 〜せいで / 〜せいか / 〜おかげで（原因 + 评价色彩）

**接续**：
- 动词 / 一类形容词简体 + せいで / おかげで
- 二类形容词 + な + せいで / おかげで
- 名词 + の + せいで / おかげで

**含义**：
- **〜せいで**：因为〜（**坏结果**，归咎）
- **〜せいか**：可能因为〜吧（**推测**）
- **〜おかげで**：多亏〜（**好结果**，感激）

例：
- 雨のせいで、試合が中止になった。（因为下雨比赛取消了。）
- 年のせいか、最近疲れやすい。（可能因为上年纪了，最近容易累。）
- 先生のおかげで合格できた。（多亏老师我考上了。）

⚠️ 区分原则：好結果 → おかげで／悪結果 → せいで／推測 → せいか。
偶尔反讽用「〜おかげで」配坏結果（语感讽刺）。

### 〜ばかりに（仅仅因为〜 / 偏偏因为〜导致坏结果）

**接续**：动词た形 / 简体 + ばかりに（二类形容词 / 名词 + である + ばかりに）

**含义**：就因为〜（导致负面结果，含懊悔语感）。

例：
- 一言多く言ったばかりに、喧嘩になった。（就因为多说了一句，吵起来了。）
- 名前を間違えたばかりに、合格できなかった。（就因为写错了名字没考上。）
- 貧しいばかりに、進学を諦めた。（仅因贫困放弃了升学。）

## 9.7 让步与假设

### 〜ても / 〜でも（让步强化）

**接续**：
- 動詞て形 + も
- 一类形容词 → 〜くても
- 二类形容词 / 名词 → で + も

**含义**：即使〜也〜（N3 常配「いくら / どんなに / たとえ」加强）。

例：
- いくら頑張っても、できない。（无论怎么努力都做不到。）
- たとえ雨が降っても、行きます。（即使下雨也去。）
- どんなに高くても買う。（不管多贵都买。）

### 〜くらいなら（与其〜不如〜）

**接续**：動詞辞書形 + くらいなら
**含义**：与其〜还不如〜（带消极、宁愿）。

例：
- 嫌な仕事をするくらいなら、辞めたほうがいい。（与其做讨厌的工作，还不如辞职。）
- 謝るくらいなら、最初からするな。（要道歉的话当初就别做。）
- そんなことを言われるくらいなら、行かないほうがいい。（被说成那样还不如不去。）

### 〜のに（逆接 N3 复习强化）

**接续**：
- 动词 / 一类形容词简体 + のに
- 二类形容词 / 名词 + な + のに

**含义**：明明〜却〜（含不满 / 意外）。

例：
- 約束したのに、来なかった。（明明约好了却没来。）
- 若いのに、よく頑張っている。（年纪轻轻就很努力。）
- 日曜日なのに、会社に行かなければならない。（明明是周日还得去公司。）

### 〜ば〜ほど（越〜越〜）

**接续**：
- 動詞ば形 + 辞書形 + ほど（読めば読むほど）
- 一类形容词 → 〜ければ〜ほど（高ければ高いほど）
- 二类形容词 → 〜なら〜なほど（静かなら静かなほど）

**含义**：越〜越〜（程度递增）。

例：
- 読めば読むほど面白い。（越读越有意思。）
- 練習すればするほど上手になる。（越练越熟练。）
- 高ければ高いほど美味しいわけではない。（不是越贵就越好吃。）

### 〜たって / 〜だって（口语让步）

**接续**：
- 動詞た形 + って（食べたって）
- 一类形容词 → 〜くたって（高くたって）
- 二类形容词 / 名词 → だって（静かだって / 学生だって）

**含义**：即使〜也〜（口语，=〜ても）。

例：
- 雨が降ったって行く。（就算下雨也去。）
- 子供だって分かる。（连小孩都懂。）
- いくら言ったって聞かない。（再怎么说也不听。）

## 9.8 提议与命令体

### 〜べきだ（应该 / 应当）

**接续**：動詞辞書形 + べきだ（する → す**る**べきだ / す**べき**だ 均可）
**含义**：道义、社会规范上应该。

例：
- 約束は守るべきだ。（应当守约。）
- もっと早く行くべきだった。（应该早点去的。── 过去未做的后悔）
- 学生はよく勉強するべきだ。（学生应该好好学习。）

⚠️ 否定 = 〜べきではない（不该）。〜べきだ 比 〜たほうがいい **语气更强 / 正式**。

### 〜ことだ（建议 / 忠告）

**接续**：動詞辞書形 / ない形 + ことだ
**含义**：建议、忠告（重大事项，类似格言）。

例：
- 健康のためには、よく寝ることだ。（为了健康，要好好睡觉。）
- あまり無理しないことだ。（别太勉强自己。）
- 合格したいなら、毎日勉強することだ。（想合格的话就每天学习。）

⚠️ 〜ことだ 比 〜たほうがいい 更具**老者 / 长辈 / 权威感**。

### 〜ように（命令 / 指示的引用形）

**接续**：動詞辞書形 / ない形 + ように
**含义**：① 指示、命令的间接引用（多用书面通知、揭示板）；② 期望、祈愿。

例：
- 9時までに集合するように。（请在 9 点前集合。── 揭示板）
- 遅れないように。（请别迟到。）
- 合格しますように。（祈祷合格。── 期望）

## 9.9 状态与外观

### 〜らしい（再次强化 N3）

详见 9.3、5.29，N3 重点：「N + らしい」表"有〜的样子"。

例：
- 子供らしい絵。（很有小孩样子的画。）
- それは彼らしくない。（那不像他会做的。）

### 〜みたいだ（口语推断 / 比喻）

**接续**：
- 动词 / 一类形容词简体 + みたい
- 二类形容词 / 名词 + みたい（**不加 だ**）

**含义**：
- ① 推断（=〜ようだ，更口语）
- ② 比喻（像〜一样）
- ③ 举例（〜这种 / 像〜的）

例：
- 雨が降っているみたいだ。（好像在下雨。）
- 子供みたいに泣いた。（像小孩一样哭了。）
- 田中さんみたいな人になりたい。（想成为像田中那样的人。）

⚠️ 〜みたいだ = 〜ようだ（口语）；〜ようだ 较正式。

### 〜気がする（感觉〜 / 好像〜）

**接续**：簡体 + 気がする（二类形容词 / 名词 + な / である + 気がする）
**含义**：主观感觉、模糊判断（比〜と思う更弱）。

例：
- 何かいいことがある気がする。（感觉会有好事发生。）
- どこかで会った気がする。（觉得在哪里见过。）
- 失敗する気がする。（感觉会失败。）

### 〜ふりをする（假装〜）

**接续**：
- 动词 / 一类形容词简体 + ふりをする
- 二类形容词 + な + ふりをする
- 名词 + の + ふりをする

**含义**：假装、装作〜。

例：
- 寝たふりをした。（装睡。）
- 知らないふりをしないで。（别装作不知道。）
- 学生のふりをして大学に入った。（假装是学生进了大学。）

## 9.10 复合表达

### 〜つもりだった（曾经打算但未实现）

**接续**：動詞辞書形 / ない形 + つもりだった

**含义**：本打算〜（实际没做）／还以为是〜（其实不是）。

例：
- 行くつもりだったが、急用ができて行けなかった。（本打算去，但有急事去不了了。）
- 説明したつもりだったが、相手は分かっていなかった。（自以为说明了，但对方没懂。）
- まだ若いつもりだったのに、もう40だ。（自以为还年轻，已经 40 了。）

### 〜とは限らない（不一定〜 / 未必〜）

**接续**：簡体 + とは限らない（名词 / 二类形容词 + だ + とは限らない 亦可省略 だ）

**含义**：不一定都〜（部分否定）。

例：
- 高いものがいいとは限らない。（贵的不一定就好。）
- 努力すれば成功するとは限らない。（努力不一定就成功。）
- 日本人がみんな寿司が好きだとは限らない。（不是所有日本人都喜欢寿司。）

### 〜こと / 〜の（动词名词化）

**接续**：動詞辞書形 + こと / の

**含义**：将动词短语名词化。

| 用法 | 用 こと | 用 の |
|---|---|---|
| 接 だ / である / にする / になる 等 | ✅ | ❌ |
| 接 見る / 聞く / 待つ / 手伝う 等**感官 / 直接动作** | ❌ | ✅ |
| 抽象内容 / 普遍真理 | ✅ | △ |

例：
- 私の趣味は本を読むことです。（我的爱好是读书。── 接 だ 用 こと）
- 友達が来るのを待っている。（在等朋友来。── 直接动作 用 の）
- 彼が泣いているのを見た。（看见他在哭。── 感官 用 の）

### 〜とおりに / 〜どおりに（按照〜）

**接续**：
- 動詞辞書形 / た形 + とおりに
- N + の + とおりに / N + どおりに（**直接接 N 时用 どおり**）

**含义**：按照〜（精确执行）。

例：
- 先生が言ったとおりにやってください。（请按老师说的做。）
- 説明書のとおりに組み立てる。（按说明书组装。）
- 予定どおりに進む。（按计划推进。）

### 〜ばかりだ（一味〜 / 只〜）

**接续**：
- 動詞辞書形 + ばかりだ（状况持续单方向变化"只会〜下去"）
- 動詞た形 + ばかりだ（刚刚〜）
- N + ばかりだ（净〜、光〜）

**含义**：
- ① 单方向变化"一直在〜"
- ② 刚〜不久
- ③ 净〜 / 光〜

例：
- 病気は悪くなるばかりだ。（病情只会越来越糟。）
- 日本に来たばかりです。（刚来日本。）
- 文句ばかり言う。（净发牢骚。）

### 〜ことになる / 〜ことにする（决定）

**接续**：動詞辞書形 / ない形 + ことになる / ことにする

**含义**：
- **〜ことになる**：（外因 / 组织）决定〜（客观、被动）
- **〜ことにする**：（自己）决定〜（主观、主动）

例：
- 来月、東京へ転勤することになった。（决定下个月调到东京。── 公司决定）
- 来月、東京へ引っ越すことにした。（我决定下个月搬到东京。── 自己决定）
- タバコをやめることにしました。（决定戒烟。）
- 結婚することになりました。（决定结婚了。── 双方 / 客观告知）

⚠️ 〜ことになっている / 〜ことにしている：① 表规定、惯例（外）；② 表个人长期习惯（内）。
- 学校では制服を着ることになっている。（学校规定要穿校服。）
- 毎朝、走ることにしている。（坚持每天早上跑步。）

# 十、高级语法（N2）

> 本章收录约 85 个 N2 核心句型。多数为 N3 基础上扩展的书面化表达、推断细分、让步结构、立场视角、评价功能等。每条按"接续公式 + 含义 + 例句"组织；近义形态在 ⚠️ 简短对比。

## 10.1 时间与时序

### 〜あげく（に）

**接续**：動詞た形 / N + の + あげく（に）
**含义**：（经过长时间反复折腾）最后……（结果多为负面 / 令人遗憾）。

例：
- さんざん悩んだあげく、会社を辞めた。（烦恼了很久，最后辞了职。）
- 議論のあげく、何も決まらなかった。（争论了半天，什么也没决定。）
- 道に迷ったあげく、約束に遅れてしまった。（迷了半天路，最后约会迟到了。）

⚠️ 前项必须是**经过一段过程**；瞬间动作不可。结果通常负面。

### 〜末（に）

**接续**：動詞た形 / N + の + 末（に）
**含义**：经过……之后，最终……（结果中性 / 偏正面亦可，书面）。

例：
- 長年の努力の末、彼は夢を実現した。（经过多年努力，他实现了梦想。）
- 悩んだ末に、留学を決めた。（思考再三，最终决定留学。）
- 激しい議論の末、結論が出た。（经过激烈讨论，得出了结论。）

⚠️ 〜あげく vs 〜末：あげく 多负面、口语；末 中性偏正面、书面。

### 〜うちに（N2 微妙用法）

**接续**：動詞辞書形 / ている形 / N + の + うちに
**含义**：① 趁着……（与 N3 同）；② **不知不觉间**发生变化（非意志结果）。

例：
- 話しているうちに、彼の良さが分かってきた。（说着说着，渐渐明白了他的好。）
- 本を読んでいるうちに眠ってしまった。（看着书不知不觉就睡着了。）
- 何度も練習するうちに、上手になった。（练着练着就熟练了。）

⚠️ N2 重点：后项**自然变化**（〜てくる / 〜ようになる / 〜てしまう）。

### 〜次第（一……就……）

**接续**：動詞ます形 / N + 次第
**含义**：一……立刻就……（多用于商务、正式场合，后项不可过去式）。

例：
- 詳細が分かり次第、ご連絡いたします。（详情一旦清楚就立刻联系您。）
- 雨が止み次第、出発します。（雨一停就出发。）
- 到着次第、お電話ください。（一到就请来电。）

⚠️ 后项不能用过去时、不能表过去事实；只用于"将来"。

### 〜たとたん（に）

**接续**：動詞た形 + とたん（に）
**含义**：刚……的瞬间，（意外地）……。

例：
- ドアを開けたとたん、猫が飛び出した。（一开门猫就冲出来了。）
- 立ち上がったとたんに、めまいがした。（一站起来就头晕。）
- 彼の顔を見たとたん、涙が出た。（一看见他的脸就掉了眼泪。）

⚠️ 后项**必须意外 / 不可控**；不接命令、意志、计划。

### 〜て以来

**接续**：動詞て形 + 以来
**含义**：自从……以来（长期持续状态）。

例：
- 日本へ来て以来、毎日忙しい。（自从来日本以后每天都忙。）
- 彼に会って以来、ずっと連絡していない。（自从见过他以后一直没联系。）
- 大学を卒業して以来、英語に触れていない。（大学毕业以来没接触过英语。）

⚠️ 后项常配「ずっと〜ている / 〜ていない」表持续。

### 〜とともに

**接续**：動詞辞書形 / N + とともに
**含义**：① 随着……（变化）；② 与……一起；③ 既……又……。

例：
- 年をとるとともに、体力が落ちてきた。（随着年龄增长体力下降。）
- 家族とともに新年を迎えた。（与家人一起迎来新年。）
- 経済の発展とともに、環境問題も深刻になった。（随着经济发展，环境问题也严重了。）

### 〜つつ / 〜つつある

**接续**：動詞ます形 + つつ / つつある
**含义**：
- **〜つつ**：一边……一边……（书面，等于〜ながら）；也可表"虽然……却……"。
- **〜つつある**：正在……（朝某方向变化中，书面）。

例：
- 音楽を聴きつつ、勉強した。（一边听音乐一边学习。）
- 悪いと知りつつ、嘘をついた。（明知不好却撒了谎。）
- 状況は改善しつつある。（情况正在好转。）

⚠️ つつある 多用于趋势性变化（増えつつある / 減りつつある）。

### 〜ところを

**接续**：動詞辞書形 / ている形 / た形 / イ形 / N + の + ところを
**含义**：① 正在……的时候（被打扰 / 被发现）；② 寒暄套语「お忙しいところを」。

例：
- 寝ているところを起こされた。（睡着的时候被叫醒。）
- お忙しいところをすみません。（百忙之中打扰您不好意思。）
- 万引きしているところを店員に見つかった。（偷东西时被店员发现了。）

⚠️ 后项常为**消极 / 被动 / 打扰**类动作。

### 〜たて

**接续**：動詞ます形 + たて
**含义**：刚刚……完（强调新鲜 / 不久）。多接食物、动作完成。

例：
- 焼きたてのパンはおいしい。（刚烤好的面包真好吃。）
- できたての料理を食べたい。（想吃刚做好的菜。）
- 入社したてで、まだ分からないことが多い。（刚入职，还有很多不懂的。）

⚠️ 接续动词有限（焼く / 作る / できる / 入社する 等"新鲜感"动词）。

## 10.2 因果与逻辑

### 〜以上（は）

**接续**：動詞辞書形 / た形 / イ形 / N + である + 以上（は）
**含义**：既然……就（应当 / 必须）……。后项多接义务、决心、判断。

例：
- 約束した以上、守らなければならない。（既然约定了就必须遵守。）
- 学生である以上、勉強が本分だ。（既然是学生，学习就是本分。）
- やると決めた以上は、最後までやる。（既然决定要做，就做到底。）

### 〜上は

**接续**：動詞辞書形 / た形 + 上は
**含义**：既然（已经）……就……（与〜以上 同义，更书面 / 决意感强）。

例：
- 引き受けた上は、最後まで責任を持つ。（既然接下了，就负责到底。）
- こうなった上は、やるしかない。（事已至此，只能做了。）

⚠️ 〜以上 / 〜上は / 〜からには 三者近义：以上 通用；上は 偏书面决意；からには 偏口语决心。

### 〜からこそ

**接续**：動詞 / イ形 / ナ形 + だ / N + だ + からこそ
**含义**：正因为……（强调原因）。

例：
- あなたのためを思うからこそ、厳しく言うのだ。（正因为为你着想才严厉地说。）
- 努力したからこそ、合格できた。（正因为努力了才能合格。）
- 君だからこそ、頼める。（正因为是你才能拜托。）

⚠️ 句末常配「のだ / んだ」加强解释语气。

### 〜からといって

**接续**：簡体 + からといって
**含义**：虽说……但也不能……（否定后项推断）。后项多搭配〜とは限らない / 〜わけではない / 否定。

例：
- 日本に住んでいるからといって、日本語が上手とは限らない。（虽说住在日本，日语也未必好。）
- 安いからといって、買いすぎるのはよくない。（虽说便宜，买太多也不好。）
- 一度失敗したからといって、諦めるな。（不能因为失败过一次就放弃。）

### 〜から見て / 〜からすると / 〜からいうと

**接续**：N + から見て / からすると / からいうと
**含义**：从……来看（判断的视角 / 立场依据）。

例：
- 彼の表情から見て、本当のことを言っているようだ。（从他的表情看，他像是说了真话。）
- 経験からすると、これはうまくいくはずだ。（从经验来看，这应该会顺利。）
- 親からいうと、子供の安全が一番だ。（从父母角度来说，孩子的安全最重要。）

⚠️ 三者大致互换；から見て 重客观证据、からすると 重综合推断、からいうと 重立场。

### 〜ことから / 〜ことだから

**接续**：
- 簡体 + ことから（イ形/动直接、ナ形+な、N+である）
- N + の + ことだから

**含义**：
- **〜ことから**：根据……（命名 / 判断的依据）。
- **〜ことだから**：因为是……（这种人 / 这种情况），所以……（说话人对对方性格的判断）。

例：
- 桜が多いことから、この町は「桜町」と呼ばれている。（因为樱花多，这镇被叫"樱町"。）
- 顔が赤いことから、お酒を飲んだことが分かる。（从脸红就能知道喝过酒。）
- 真面目な彼のことだから、約束は必ず守るだろう。（他那么认真，约定一定会守。）

⚠️ ことから 客观依据；ことだから 个人性格 / 习惯依据。

### 〜だけあって / 〜だけに

**接续**：動詞・イ形・ナ形（な / である）・N + だけあって / だけに
**含义**：
- **〜だけあって**：不愧是……（积极评价为主）。
- **〜だけに**：正因为是……所以更……（双向，可积极可消极；含"程度更深"）。

例：
- 彼は10年も日本にいただけあって、日本語がペラペラだ。（他在日本待了 10 年，难怪日语流利。）
- プロだけあって、仕事が早い。（不愧是专家，做事真快。）
- 期待していただけに、結果はがっかりだった。（正因为期待大，结果更失望。）

⚠️ だけあって 多正面；だけに 双向 + 强调"反作用"。

### 〜せい / 〜せいで / 〜のせいか

**接续**：簡体（ナ形 +な、N +の）+ せい / せいで / せいか
**含义**：
- **〜せいで**：（坏的）原因 / 责任在于……。
- **〜のせいか**：可能是因为……（不确定的归因）。

例：
- 雨のせいで、試合が中止になった。（都怪下雨，比赛取消了。）
- 寝不足のせいか、頭が痛い。（可能是没睡好，头疼。）
- 失敗を人のせいにするな。（别把失败怪到别人头上。）

⚠️ 对比〜おかげで（积极原因）。

### 〜だけのことはある

**接续**：動詞・イ形・ナ形（な）・N（である）+ だけのことはある
**含义**：值得……、不愧是……（积极评价、表"果然名副其实"）。

例：
- さすが、有名なシェフだけのことはある。料理が絶品だ。（不愧是名厨，菜真是绝品。）
- 毎日練習しただけのことはあって、優勝した。（每天练习果然没白练，得了冠军。）
- 高いだけのことはある、性能が素晴らしい。（贵得有道理，性能极佳。）

### 〜ものだから / 〜もので

**接续**：簡体（ナ形 +な、N +な）+ ものだから / もので
**含义**：因为……（找借口 / 解释 / 辩解的语气）。

例：
- 道が混んでいたものだから、遅刻してしまった。（路上太堵了所以迟到了。）
- 子供がうるさいもので、ゆっくり話せません。（孩子吵闹所以没法慢慢说。）
- 急いでいたものですから、ご挨拶できませんでした。（赶时间所以没能打招呼。）

⚠️ 多用于第一人称解释 / 辩解；后项一般不接意志、命令。

## 10.3 让步与逆接

### 〜くせに

**接续**：動詞・イ形 簡体 / ナ形 + な / N + の + くせに
**含义**：明明……却……（含责备、嘲讽、不满，主语 = 前后同一人）。

例：
- 知っているくせに、教えてくれない。（明明知道却不告诉我。）
- 子供のくせに、生意気だ。（明明是个小孩，真狂。）
- 下手なくせに、よくやるね。（明明不擅长，还真敢做。）

⚠️ 含**负面情感**；前后主语必须一致。比〜のに更主观、更口语。

### 〜にしては

**接续**：動詞簡体 / N + にしては
**含义**：作为……来说（出乎意料地）……。

例：
- 初心者にしては、上手だ。（作为初学者来说，挺厉害的。）
- 子供にしては、難しい言葉を使う。（作为孩子来说，用词挺难的。）
- 値段が安いにしては、品質がいい。（按价格来说质量算好的。）

⚠️ 前项需明确"标准"；后项是与该标准比的反差。

### 〜にしたら / 〜にすれば / 〜にしてみれば

**接续**：N + にしたら / にすれば / にしてみれば
**含义**：从……的立场来看（多用于推测他人感受）。

例：
- 親にしたら、子供の結婚は大事だ。（从父母的立场看，子女的婚事是大事。）
- 彼にしてみれば、当然の反応だろう。（站在他的立场，这反应是当然的吧。）
- 外国人にすれば、日本語の漢字は難しい。（在外国人看来，日语汉字很难。）

⚠️ 主语 ≠ 说话人；不可用于自己。

### 〜にしても

**接续**：動詞簡体 / イ形 / N + にしても
**含义**：① 即使……也……（让步）；② 不管是 A 还是 B（〜にしても〜にしても）。

例：
- 行くにしても、行かないにしても、連絡してください。（不管去不去都请联系。）
- 忙しいにしても、連絡くらいできるだろう。（再忙也总能联系一下吧。）
- 嘘にしても、ひどすぎる。（就算是谎话也太过分了。）

### 〜にしろ / 〜にせよ

**接续**：動詞簡体 / イ形 / ナ形 / N + にしろ / にせよ
**含义**：无论……还是……（书面化、正式语气；同〜にしても）。

例：
- 行くにしろ行かないにしろ、早く決めて。（去也好不去也好，快决定。）
- 何をするにせよ、計画が大事だ。（不管做什么，计划都重要。）
- 冗談にせよ、言ってはいけないことがある。（即使是玩笑，也有不能说的话。）

### 〜ながら（も）（让步用法）

**接续**：動詞ます形 / イ形 / ナ形（语干） / N + ながら（も）
**含义**：虽然……却……（书面、文雅）。

例：
- 知っていながら、教えてくれなかった。（明明知道却不告诉我。）
- 残念ながら、ご期待には添えません。（很遗憾，无法满足您的期待。）
- 狭いながらも、楽しい我が家。（虽然狭窄但很快乐的家。）

⚠️ 与"一边……一边……"的〜ながら同形，靠语境区分。

### 〜たところで

**接续**：動詞た形 + ところで
**含义**：即使……也（白费 / 没用）。后项多为否定 / 消极结果。

例：
- 今さら謝ったところで、許してもらえないだろう。（事到如今道歉也不会被原谅吧。）
- いくら急いだところで、間に合わない。（再怎么赶也来不及了。）
- 努力したところで、結果は変わらない。（再努力结果也不会变。）

⚠️ 后项**必为否定 / 无意义结果**；不接积极结果。

### 〜ものの

**接续**：動詞・イ形 簡体 / ナ形 + な / である / N + である + ものの
**含义**：虽然……但是……（前项肯定，后项不如预期）。

例：
- 大学を卒業したものの、就職先が見つからない。（虽然大学毕业了，却找不到工作。）
- 引き受けたものの、自信がない。（虽然接下来了，却没信心。）
- 体は丈夫であるものの、すぐ疲れる。（身体虽然结实，却容易累。）

⚠️ 偏书面；常表"前项成立、后项不顺"。

### 〜とはいえ

**接续**：簡体（ナ形・N 可直接接）+ とはいえ
**含义**：虽说……但实际上……。

例：
- 春とはいえ、まだ寒い。（虽说是春天，但还冷。）
- 子供とはいえ、礼儀は大切だ。（虽说是孩子，礼仪也重要。）
- 引退したとはいえ、彼の影響力は大きい。（虽说退休了，影响力还很大。）

⚠️ 与〜といっても 近义；とはいえ 更书面、转折更强。

### 〜くらいなら / 〜ぐらいなら

**接续**：動詞辞書形 + くらいなら
**含义**：与其……不如……（前项是不愿接受的极端）。后项多搭配〜ほうがましだ / 〜たほうがいい。

例：
- 彼に頼むくらいなら、自分でやる。（与其求他，不如自己做。）
- 捨てるくらいなら、人にあげたほうがいい。（与其扔，不如送人。）
- 謝るぐらいなら、最初からやるな。（要道歉就别一开始去做。）

### 〜にもかかわらず

**接续**：動詞・イ形 簡体 / ナ形（である）/ N + にもかかわらず
**含义**：尽管……却……（书面、正式）。

例：
- 雨にもかかわらず、試合は行われた。（尽管下雨，比赛仍然进行。）
- 努力したにもかかわらず、不合格だった。（尽管努力了，还是没合格。）
- 体調が悪いにもかかわらず、出勤した。（尽管身体不舒服，还是上班了。）

⚠️ 比〜のに 更正式、转折更强。

## 10.4 条件与假设

### 〜ないかぎり

**接续**：動詞ない形 + かぎり
**含义**：只要不……（就不……）。后项必为否定。

例：
- 謝らないかぎり、許さない。（不道歉就不原谅。）
- 努力しないかぎり、成功はない。（不努力就没有成功。）
- 証拠がないかぎり、信じられない。（没有证据就无法相信。）

⚠️ 后项必为否定 / 消极结果。

### 〜かぎり（は）/ 〜かぎりでは

**接续**：動詞辞書形 / ている形 / イ形 / ナ形 + な / である / N + である + かぎり
**含义**：① 只要……（条件持续）；② 在……范围内（〜かぎりでは）。

例：
- 体が元気なかぎり、働き続けたい。（只要身体好就想继续工作。）
- 私の知るかぎりでは、彼は誠実な人だ。（据我所知，他是个诚实的人。）
- 親である限り、子供を守る義務がある。（只要是父母，就有保护孩子的义务。）

### 〜さえ〜ば

**接续**：
- N + さえ + 動詞ば形
- 動詞ます形 + さえ + すれば
- イ形 + く + さえあれば

**含义**：只要……就……（最低限度的条件）。

例：
- お金さえあれば、何でもできる。（只要有钱什么都能做。）
- 練習しさえすれば、上手になる。（只要练习就会熟练。）
- 体さえ丈夫なら、何でもできる。（只要身体结实就什么都能做。）

⚠️ 强调"只要这一项满足就够"。

### 〜ば〜ほど（N2 升级用法）

**接续**：
- 動詞ば形 + 動詞辞書形 + ほど
- イ形ければ + イ形 + ほど
- ナ形なら（ば）+ ナ形 + な + ほど
- N であれば + N + であるほど

**含义**：越……越……。N2 强调与抽象 / 程度副词搭配。

例：
- 考えれば考えるほど、分からなくなる。（越想越糊涂。）
- 練習すればするほど、上達する。（越练越熟。）
- 静かであればあるほど、集中できる。（越安静越能集中。）

### 〜たら最後 / 〜たが最後

**接续**：動詞た形 + ら最後 / が最後
**含义**：一旦……（就必然出现严重后果）。

例：
- 彼に話したら最後、すぐ広まる。（一告诉他立刻就传开。）
- 一度始めたが最後、やめられない。（一开始就停不下来。）
- 彼女が怒ったら最後、誰も止められない。（她一旦生气，谁也拦不住。）

⚠️ 后项必为**负面、不可逆**的结果；语气强烈。

### 〜なくして（は）

**接续**：N + なくして（は）
**含义**：如果没有……就（不能）……（书面）。

例：
- 努力なくして成功はない。（没有努力就没有成功。）
- 彼の助けなくしては、完成できなかった。（没有他的帮助就完不成。）
- 愛なくして人生は語れない。（没有爱就谈不了人生。）

⚠️ 后项必为否定。书面化、文章 / 演讲常见。

## 10.5 限定与极端

### 〜きり / 〜きりだ / 〜たきり

**接续**：
- 動詞た形 + きり（〜たきり）
- N + きり
- 動詞辞書形 + きり

**含义**：
- **〜たきり**：……之后就再也（没……）。
- **〜きり**：仅仅……。

例：
- 彼は出かけたきり、戻ってこない。（他出去后就再也没回来。）
- 二人きりで話したい。（想两个人单独谈。）
- 一度会ったきりです。（只见过一次。）

⚠️ 〜たきり 后项**必为否定**或表"再也没……"。

### 〜こそ

**接续**：N / 動詞て形 + こそ
**含义**：正是……才……（强调）。

例：
- 今度こそ、頑張ろう。（这次一定要加油。）
- 失敗してこそ、成長できる。（正因为失败才能成长。）
- こちらこそ、ありがとう。（我才要谢谢你。）

### 〜さえ

**接续**：N（+ 助词）+ さえ / 動詞て形 + さえ
**含义**：① 连……都……（极端例示）；② 〜さえ〜ば 见 10.4。

例：
- 子供さえ知っている。（连小孩都知道。）
- 漢字どころか、ひらがなさえ書けない。（别说汉字，连平假名都不会写。）
- 親にさえ言えない秘密。（连父母都说不出口的秘密。）

⚠️ 与〜まで 类似；さえ 强调"最该……的也……"。

### 〜のみ / 〜のみならず

**接续**：
- N / 動詞辞書形 + のみ
- 動詞・イ形 簡体 / ナ形・N（である）+ のみならず

**含义**：
- **〜のみ**：仅仅……（书面〜だけ）。
- **〜のみならず**：不仅……而且……（书面〜だけでなく）。

例：
- 会員のみ入場可能。（仅限会员入场。）
- 彼は英語のみならず、フランス語も話せる。（他不仅会英语，还会法语。）
- 学生のみならず、社会人にも人気だ。（不仅学生、社会人也喜欢。）

### 〜はおろか

**接续**：N + はおろか
**含义**：别说……连……都……（强调连基本的都做不到）。后项常配〜さえ / も / すら。

例：
- 漢字はおろか、ひらがなさえ読めない。（别说汉字，连平假名都不会读。）
- 海外旅行はおろか、国内も行ったことがない。（别说出国，国内都没去过。）
- 走るはおろか、歩くのもつらい。（别说跑，连走都难。）

⚠️ 偏 N1 但 N2 综合教材常列；后项必为否定 / 消极。

### 〜ばかりか

**接续**：動詞・イ形 簡体 / ナ形・N（+ である）+ ばかりか
**含义**：不仅……而且……（带"超出预期"的语感）。

例：
- 彼は頭がいいばかりか、性格もいい。（他不仅聪明，性格也好。）
- 雨ばかりか、雷まで鳴り始めた。（不仅下雨，还打起雷来。）
- 失敗したばかりか、笑われた。（不仅失败了还被笑话了。）

⚠️ 与〜のみならず 近义；ばかりか 更口语 / 带评价。

### 〜ばかりに

**接续**：動詞た形 / イ形 / ナ形 + な / である / N + である + ばかりに
**含义**：就因为……（结果不好；含懊悔）。

例：
- 一言多く言ったばかりに、喧嘩になった。（就因为多说一句，吵起来了。）
- お金がないばかりに、進学を諦めた。（就因为没钱放弃了升学。）
- 油断したばかりに、試合に負けた。（就因为大意，比赛输了。）

⚠️ 后项必为**负面结果**；含说话人后悔。

### 〜抜きで / 〜抜きにして / 〜抜きには〜ない

**接续**：N + 抜きで / 抜きにして / 抜きには〜ない
**含义**：
- **〜抜きで / にして**：省略 / 不算……。
- **〜抜きには〜ない**：没有……就不能……。

例：
- 冗談抜きで、本気で言っている。（不开玩笑，是认真的。）
- 朝食抜きで出かけた。（没吃早饭就出门了。）
- 彼の協力抜きには、成功できなかった。（没有他的协助就成不了功。）

### 〜はもとより / 〜はもちろん

**接续**：N + はもとより / はもちろん
**含义**：……自不用说，连……也……。

例：
- 漢字はもとより、ひらがなもまだ書けない。（汉字就不用说，平假名也还不会写。）
- 平日はもちろん、週末も忙しい。（平日就不用说，周末也忙。）
- 国内はもとより、海外でも人気だ。（国内自不必说，国外也很受欢迎。）

⚠️ もとより 书面；もちろん 口语。

### 〜だらけ

**接续**：N + だらけ
**含义**：满是……（多为不好的 / 不洁的东西）。

例：
- 部屋がほこりだらけだ。（房间里全是灰。）
- 彼の答案は間違いだらけだった。（他的答卷错误连篇。）
- 服が泥だらけになった。（衣服沾满了泥。）

⚠️ 接负面、可数 / 不可数物（间違い・泥・血・ほこり 等）。

### 〜まみれ

**接续**：N + まみれ
**含义**：浑身沾满……（液体 / 粉状物，物理附着）。

例：
- 汗まみれになって働いた。（满身大汗地工作。）
- 血まみれの服。（沾满血的衣服。）
- 子供は泥まみれで帰ってきた。（孩子满身是泥地回来了。）

⚠️ まみれ vs だらけ：まみれ 限**液体 / 粉状物附着**；だらけ 范围广（含抽象，如 借金だらけ）。

### 〜ぐるみ

**接续**：N + ぐるみ
**含义**：连……带……整个一起；连皮带……。

例：
- 家族ぐるみで付き合っている。（全家整个一起来往。）
- 町ぐるみで観光客を迎える。（全镇一起接待游客。）
- 会社ぐるみの不正。（公司整个的舞弊。）

⚠️ 接组织 / 团体名词较多。

## 10.6 推断与判断

### 〜得る / 〜得ない

**接续**：動詞ます形 + 得る（うる / える）/ 得ない（えない）
**含义**：可能……（书面，"客观可能性"）。

例：
- そんなことはあり得ない。（那种事不可能。）
- 誰にでも起こり得る事故だ。（任何人都可能遇到的事故。）
- 考え得る最善の方法。（能想到的最好办法。）

⚠️ 肯定形读 うる / える 皆可；否定一定读 えない。

### 〜かねる / 〜かねない

**接续**：動詞ます形 + かねる / かねない
**含义**：
- **〜かねる**：（心情 / 立场上）难以……（婉拒）。
- **〜かねない**：可能会（不好的事）。

例：
- そのご要望にはお応えしかねます。（恕难答应您的要求。）
- そんな大金は決めかねる。（这么大笔钱难以决定。）
- 寝不足では事故を起こしかねない。（睡眠不足可能引发事故。）

⚠️ かねない 必接**坏事**；不能说「合格しかねない」。

### 〜に決まっている

**接续**：動詞・イ形 簡体 / ナ形 / N + に決まっている
**含义**：肯定是……、绝对……（说话人主观确信）。

例：
- そんな話、嘘に決まっている。（那种话肯定是假的。）
- 彼が来るに決まっている。（他肯定会来。）
- こんな問題、簡単に決まっている。（这种问题肯定简单。）

⚠️ 比〜に違いない 更口语、更主观。

### 〜に違いない / 〜に相違ない

**接续**：動詞・イ形 簡体 / ナ形 / N + に違いない / に相違ない
**含义**：一定……、肯定……（基于根据的确信推断）。

例：
- 彼は今、家にいるに違いない。（他现在肯定在家。）
- 犯人は内部の者に違いない。（犯人肯定是内部人员。）
- これは本物に相違ない。（这肯定是真品。）

⚠️ に相違ない 书面、正式；に違いない 通用。

### 〜に過ぎない

**接续**：動詞簡体 / イ形 / ナ形 / N + に過ぎない
**含义**：只不过是……（带轻视 / 谦虚）。

例：
- それは言い訳に過ぎない。（那不过是借口。）
- 私はただの素人に過ぎません。（我不过是外行人。）
- 彼は表面を見ているに過ぎない。（他不过看到了表面。）

### 〜とは限らない

**接续**：簡体 + とは限らない（ナ形・N 可直接接）
**含义**：未必……、不一定……。

例：
- 高いものがいいとは限らない。（贵的东西未必好。）
- 日本人だからといって、寿司が好きとは限らない。（虽说是日本人，未必喜欢寿司。）
- 必ずしも成功するとは限らない。（未必一定成功。）

⚠️ 常配「必ずしも」「〜からといって」。

### 〜まい / 〜まいか

**接续**：
- 一类动词辞書形 + まい（行くまい）
- 二类动词 + まい / 〜よう + まい（食べまい / 食べようまい）
- 三类：するまい / すまい / しまい、来るまい / 来まい / 来（こ）まい
- N + ではあるまい

**含义**：① 不会……吧（否定推量）；② 决不……（否定意志，书面）。

例：
- 彼はもう来るまい。（他大概不会再来了。）
- 二度と失敗するまい。（绝不再失败。）
- それは嘘ではあるまいか。（那不会是假话吧？）

⚠️ 书面、文章性强；现代口语少。

### 〜ようがない / 〜ようもない

**接续**：動詞ます形 + ようがない / ようもない
**含义**：无法……、没办法……。

例：
- 連絡先が分からないので、知らせようがない。（不知道联系方式所以没法通知。）
- どうしようもない。（毫无办法。）
- 直しようがないほど壊れた。（坏得没法修。）

### 〜わけだ / 〜わけがない / 〜わけではない / 〜わけにはいかない

**接续**：動詞・イ形 簡体 / ナ形 + な / N + な / である + わけ〜
**含义**：
- **〜わけだ**：原来如此、当然就……（合乎逻辑的结论）。
- **〜わけがない**：不可能……。
- **〜わけではない**：并不是……（部分否定）。
- **〜わけにはいかない**：（道理上 / 社会上）不能……。

例：
- 彼は10年も日本にいる。日本語が上手なわけだ。（他在日本待了 10 年，难怪日语好。）
- そんなはずがない、彼が裏切るわけがない。（不可能，他不可能背叛。）
- 嫌いなわけではない。（并不是讨厌。）
- 仕事中だから、お酒を飲むわけにはいかない。（工作中，不能喝酒。）

⚠️ 四种わけ 区分必考；わけがない（可能性=0）vs わけではない（部分否定）易混。

## 10.7 立场与对象

### 〜にあたって / 〜に際して

**接续**：動詞辞書形 / N + にあたって / に際して
**含义**：在……之际、值此……（正式 / 仪式场合的开场）。

例：
- 入社にあたって、決意を述べたい。（入社之际，想说说决心。）
- 出発に際して、注意事項を確認します。（出发之际，确认注意事项。）
- 新年を迎えるにあたって、目標を立てた。（迎接新年之际，立了目标。）

⚠️ 后项不接日常 / 琐碎；多接正式行为、致辞、表态。

### 〜において / 〜における

**接续**：N + において / における
**含义**：在……（场所 / 时间 / 领域），书面化〜で / 〜の。

例：
- 21世紀における技術の発展は著しい。（21 世纪的技术发展惊人。）
- 会議は東京において行われる。（会议在东京举行。）
- 教育においては、家庭の役割が大きい。（在教育方面，家庭作用很大。）

### 〜に応じて

**接续**：N + に応じて
**含义**：根据……、随……而……。

例：
- 収入に応じて、税金を払う。（根据收入纳税。）
- 状況に応じて、対応を変える。（视情况改变应对。）
- 年齢に応じた服装。（适合年龄的着装。）

### 〜に関して / 〜に関する

**接续**：N + に関して / に関する
**含义**：关于……（书面〜について）。

例：
- 環境問題に関して、議論された。（就环境问题进行了讨论。）
- 日本文化に関する本。（关于日本文化的书。）
- これに関しては、後で説明します。（关于这个，稍后说明。）

### 〜に対して / 〜に対する

**接续**：N + に対して / に対する
**含义**：① 对……（动作对象 / 态度）；② 与……相对（对比）。

例：
- 子供に対して、優しく接する。（对孩子要温柔。）
- 質問に対して、丁寧に答えた。（对问题做了详细回答。）
- 兄は静かなのに対して、弟は元気だ。（哥哥安静，弟弟则活泼。）

⚠️ 〜に対して（动作针对）vs 〜について（话题内容）：对人 = に対して，对话题 = について。

### 〜について / 〜についての

**接续**：N + について / についての
**含义**：关于……（话题 / 内容）。

例：
- この問題について、話し合おう。（关于这个问题谈一谈。）
- 日本の歴史についての本。（关于日本历史的书。）
- 詳しいことについては、後でお知らせします。（详细的情况稍后通知。）

### 〜にとって

**接续**：N + にとって
**含义**：对……来说、对……而言（立场视角）。

例：
- 私にとって、家族が一番大切だ。（对我来说，家人最重要。）
- 学生にとって、勉強は仕事だ。（对学生而言，学习就是工作。）
- 子供にとって遊びは必要だ。（对孩子来说，玩耍是必要的。）

⚠️ 主语后多为**评价 / 价值类形容词**；不接动作（动作用〜に対して）。

### 〜によって / 〜による（N2 升级）

**接续**：N + によって / による
**含义**：
- **手段 / 方法**：通过……。
- **原因**：由于……。
- **被动施动者**：被……。
- **差异 / 视情况**：因……而异。

例：
- インターネットによって、世界が変わった。（通过互联网世界变了。）
- 地震による被害。（地震造成的损失。）
- この本は夏目漱石によって書かれた。（这本书是夏目漱石写的。）
- 国によって習慣が違う。（习俗因国而异。）

⚠️ 〜によると（传闻）见下一条；不要混。

### 〜によると / 〜によれば

**接续**：N + によると / によれば
**含义**：根据……（信息来源 / 传闻依据）。后项常配〜そうだ / 〜とのことだ / 〜らしい。

例：
- 天気予報によると、明日は雨だそうだ。（据天气预报，明天下雨。）
- 友人によれば、彼は結婚するらしい。（据朋友说他要结婚了。）
- 報告によると、売上が増えたとのことだ。（据报告说销售额增加了。）

### 〜にわたって / 〜にわたる

**接续**：N（时间幅 / 范围）+ にわたって / にわたる
**含义**：横跨……、长达……（时间跨度 / 范围广）。

例：
- 10年にわたって研究を続けた。（持续了 10 年的研究。）
- 全国にわたる調査が行われた。（进行了覆盖全国的调查。）
- 3時間にわたる会議。（长达 3 小时的会议。）

⚠️ 名词多为**长时间 / 广范围**；短时间不用（× 3 分にわたって）。

### 〜をめぐって / 〜をめぐる

**接续**：N + をめぐって / をめぐる
**含义**：围绕……（争论 / 讨论的对象）。后项多为意见冲突。

例：
- 新法案をめぐって、議論が続いている。（围绕新法案争论不断。）
- 遺産をめぐる争い。（围绕遗产的争执。）
- その問題をめぐって、各国が対立した。（围绕该问题各国对立。）

### 〜を中心に / 〜を中心として

**接续**：N + を中心に / を中心として
**含义**：以……为中心。

例：
- 若者を中心に人気がある。（以年轻人为中心很受欢迎。）
- 東京を中心として、関東地方に広がる。（以东京为中心扩展到关东地区。）
- 環境問題を中心に議論された。（围绕环境问题展开讨论。）

### 〜を通して / 〜を通じて

**接续**：N + を通して / を通じて
**含义**：① 通过……（手段 / 媒介）；② 在……期间一直（时间跨度）。

例：
- 友人を通して彼を知った。（通过朋友认识了他。）
- ネットを通じて世界中と繋がる。（通过网络与世界相连。）
- 一年を通して暖かい。（全年都温暖。）

⚠️ 两者基本互换；を通して 偏口语，を通じて 偏书面。

## 10.8 起因与基础

### 〜をきっかけに / 〜を契機に

**接续**：N / 動詞辞書形・た形 + の + をきっかけに / を契機に
**含义**：以……为契机（开始新行为 / 变化）。

例：
- 留学をきっかけに、日本語を勉強し始めた。（以留学为契机开始学日语。）
- 病気を契機に、健康に気をつけるようになった。（以生病为契机开始注重健康。）
- 彼との出会いをきっかけに、人生が変わった。（以与他相识为契机，人生改变了。）

⚠️ を契機に 偏书面正式；をきっかけに 通用。

### 〜をもとに / 〜を基に

**接续**：N + をもとに（して）
**含义**：以……为依据 / 素材。

例：
- 実話をもとにした映画。（根据真实故事改编的电影。）
- アンケート結果をもとに、商品を改良した。（根据问卷结果改良了商品。）
- 経験をもとに、アドバイスする。（以经验为依据给建议。）

⚠️ 〜をもとに（素材 / 依据）vs 〜に基づいて（基准 / 准则）：素材改造用 をもとに，遵循规则用 に基づいて。

### 〜をはじめ / 〜をはじめとして

**接续**：N + をはじめ / をはじめとして
**含义**：以……为首（举代表 + 类推其他）。

例：
- 社長をはじめ、社員全員が出席した。（以社长为首全体员工出席。）
- 東京をはじめとして、各地で公演する。（以东京为首在各地公演。）
- 寿司をはじめ、日本料理が大好きだ。（最爱日本料理，尤其寿司。）

### 〜にもとづいて / 〜に基づいて

**接续**：N + にもとづいて / に基づく
**含义**：基于……、根据……。

例：
- 法律に基づいて、処分する。（依法处理。）
- 事実に基づいた報道。（基于事实的报道。）
- 経験に基づいて判断する。（根据经验判断。）

## 10.9 情态与心境

### 〜げ

**接续**：イ形（去 い）/ ナ形 / 動詞ます形 + げ
**含义**：……的样子（带情绪 / 表情；多形容他人外观感受）。

例：
- 寂しげな顔をしている。（脸上一副寂寞的样子。）
- 嬉しげに笑った。（高兴地笑了。）
- 物言いたげな目で見る。（用欲言又止的眼神看。）

⚠️ 用于**第三人称**的外观推断；自己不用。

### 〜っぽい

**接续**：N / 動詞ます形 / イ形（去 い）+ っぽい
**含义**：① 像……的感觉；② 容易……（倾向）。

例：
- 子供っぽい性格。（孩子气的性格。）
- 忘れっぽい人。（容易忘事的人。）
- このコーヒー、水っぽい。（这咖啡很水。）

⚠️ 口语色彩强；变形为一类形容词（〜っぽくない / 〜っぽかった）。

### 〜らしい（特性 vs 推断）

**接续**：N + らしい / 動詞・イ形 簡体 + らしい
**含义**：
- **特性**（N + らしい）：典型的……、有……风范。
- **推断**（簡体 + らしい）：好像……（客观传闻 / 间接证据）。

例：
- 男らしい男。（有男子气概的男人。）
- 春らしい天気になった。（变得有春天气息了。）
- 彼は風邪を引いたらしい。（他好像感冒了。）

⚠️ 特性 らしい 接 N 后接 N；推断 らしい 接简体后多结句。

### 〜てたまらない

**接续**：動詞て形 / イ形くて / ナ形で + たまらない
**含义**：……得受不了（强烈生理 / 情感）。

例：
- 暑くてたまらない。（热得受不了。）
- 故郷が懐かしくてたまらない。（怀念家乡得不行。）
- 眠くてたまらない。（困得不行。）

⚠️ 第一人称感情 / 感觉为主。

### 〜てならない

**接续**：動詞て形 / イ形くて / ナ形で + ならない
**含义**：不由得……、忍不住……（自然涌出的情感，多接思考动词）。

例：
- 不思議でならない。（觉得不可思议得很。）
- 残念でならない。（实在遗憾。）
- 故郷のことが思い出されてならない。（不由得想起故乡。）

⚠️ 多接**自发动词**（〜される、〜気がする、思える）和情感词。

### 〜てしようがない / 〜てしかたがない

**接续**：動詞て形 / イ形くて / ナ形で + しようがない / しかたがない
**含义**：……得没办法（程度极强；同 〜てたまらない）。

例：
- 暇でしようがない。（闲得不得了。）
- お腹がすいてしかたがない。（饿得受不了。）
- 気になってしようがない。（在意得不行。）

⚠️ 〜てたまらない vs 〜てならない vs 〜てしようがない：三者近义；ならない 偏自发情感 / 思考，たまらない / しようがない 偏生理 / 强烈感受。

### 〜ようでは

**接续**：動詞・イ形 簡体 / ナ形 + な / N + の + ようでは
**含义**：照……这样子（下去）的话（后项常为消极判断）。

例：
- そんな考えのようでは、成功しない。（这种想法的话不会成功。）
- こんなに失敗するようでは、任せられない。（这么常失败的话不能委以重任。）
- 毎日遅刻するようでは、首になるよ。（每天迟到的话会被解雇。）

⚠️ 后项几乎必为**否定 / 警告 / 不利预测**。

### 〜ようとする / 〜ようとしない

**接续**：動詞意志形（〜よう / 〜おう）+ とする / としない
**含义**：
- **〜ようとする**：试图……、正要……。
- **〜ようとしない**：不肯……、不愿……（主语态度上拒绝）。

例：
- 立ち上がろうとしたとき、めまいがした。（正要起身时头晕。）
- 何度言っても聞こうとしない。（说多少遍都不肯听。）
- 彼は本当のことを言おうとしなかった。（他不肯说真话。）

## 10.10 强调与列举

### 〜やら〜やら

**接续**：N + やら + N + やら / 動詞辞書形 + やら + 動詞辞書形 + やら
**含义**：又……又……（列举多种、含不胜其烦语感）。

例：
- 仕事やら勉強やらで忙しい。（又工作又学习忙得很。）
- 嬉しいやら恥ずかしいやらで、何も言えなかった。（又高兴又害羞，什么都说不出。）
- 雨が降るやら風が吹くやら、ひどい天気だった。（又下雨又刮风，天气糟糕。）

### 〜なり〜なり

**接续**：動詞辞書形 / N + なり + 動詞辞書形 / N + なり
**含义**：……也好……也好（任选其一、提建议）。

例：
- 電話するなりメールするなり、連絡してください。（打电话也好发邮件也好，请联系。）
- お茶なりコーヒーなり、お好きなものを。（茶也好咖啡也好，请挑喜欢的。）
- 親なり先生なりに相談しなさい。（找父母或老师商量。）

⚠️ 含"自己挑一个就行"的提议语气；不接已定事实。

### 〜やいなや

**接续**：動詞辞書形 + やいなや
**含义**：刚一……就……（书面 / 文学）。

例：
- ベルが鳴るやいなや、生徒たちは教室を飛び出した。（铃一响学生们就冲出教室。）
- 彼女は私の顔を見るやいなや、泣き出した。（她一看见我就哭了。）
- 着くやいなや、また出かけた。（一到就又出门了。）

⚠️ 偏 N1，N2 教材常列；书面性强。

### 〜とか（传闻 N2 用法）

**接续**：簡体 + とか
**含义**：（听说）……（含不确定 / 间接信息）。

例：
- 来月、転勤するとか。（听说下个月调动。）
- 彼、結婚するとか言っていた。（他说要结婚什么的。）
- 雨が降るとか聞いた。（听说会下雨。）

⚠️ 比〜そうだ 更随意 / 模糊。

### 〜ものか / 〜ものですか

**接续**：動詞・イ形 簡体 / ナ形 + な / N + な + ものか
**含义**：才不……呢（强烈反驳 / 否定）。

例：
- そんなこと、信じるものか。（那种事我才不信呢。）
- 二度と来るものか。（再也不来了。）
- 負けるものですか。（才不会输呢。）

⚠️ 强烈个人感情；女性常用 もんですか。

### 〜ことか

**接续**：動詞・イ形 簡体 / ナ形 + な / N + である + ことか
**含义**：多么……啊（感叹）。

例：
- どんなに嬉しかったことか。（多么高兴啊。）
- 何度注意したことか。（提醒过多少次了啊。）
- どれほど待ったことか。（等了多久啊。）

⚠️ 常配「どんなに / どれほど / 何度」。

### 〜どころか

**接续**：動詞・イ形 簡体 / ナ形 / N + どころか
**含义**：别说……连……都……（与预期相反，强烈对比）。

例：
- 漢字どころか、ひらがなも書けない。（别说汉字，平假名都写不了。）
- 優しいどころか、冷たい人だ。（别说温柔，是个冷淡的人。）
- 喜ぶどころか、怒り出した。（别说高兴，反而生气了。）

### 〜どころではない

**接续**：動詞辞書形 / N + どころではない
**含义**：哪里谈得上……、根本没心情 / 余力做……。

例：
- 仕事が忙しくて、旅行どころではない。（工作太忙，哪有心思旅行。）
- 風邪を引いて、勉強どころではなかった。（感冒了，根本没法学习。）
- お金がなくて、外食どころではない。（没钱，哪谈得上下馆子。）

## 10.11 评价与功能

### 〜にすぎない

（见 10.6 ── 此处同条目，归类于评价语气）

### 〜にほかならない

**接续**：N + にほかならない
**含义**：正是……、不外乎是……（强调断定）。

例：
- 成功は努力の結果にほかならない。（成功正是努力的结果。）
- これは愛にほかならない。（这正是爱。）
- 失敗の原因は油断にほかならない。（失败的原因正是大意。）

⚠️ 书面、断言色彩强；用于本质归因。

### 〜にあたる

**接续**：N + にあたる
**含义**：① 相当于……；② 处于……位置 / 关系。

例：
- 「ありがとう」は中国語の「谢谢」にあたる。（「ありがとう」相当于汉语的"谢谢"。）
- 彼は私の叔父にあたる。（他相当于我叔叔。）
- 1 ドルは約 150 円にあたる。（1 美元约相当于 150 日元。）

### 〜向け（むけ）

**接续**：N + 向け
**含义**：面向……（专门针对某对象设计）。

例：
- 子供向けの番組。（面向儿童的节目。）
- 海外向けの商品。（面向海外的商品。）
- 初心者向けに作られた教材。（为初学者制作的教材。）

### 〜向き（むき）

**接续**：N + 向き
**含义**：适合……（自然适合，非刻意设计）。

例：
- この本は初心者向きだ。（这本书适合初学者。）
- 南向きの部屋。（朝南的房间。）
- 子供向きの味付け。（适合孩子的调味。）

⚠️ 〜向け（刻意设计目标）vs 〜向き（恰好合适）。

### 〜ものだ / 〜ものではない

**接续**：動詞辞書形 / ない形 / イ形 / ナ形 + な + ものだ
**含义**：
- **〜ものだ**：① 一般来说……（常识 / 本质）；② 真是……啊（感叹）。
- **〜ものではない**：不应当……（道理上、社会上）。

例：
- 子供は元気なものだ。（小孩本来就活泼。）
- 月日が経つのは早いものだ。（光阴流逝真快啊。）
- そんなことを言うものではない。（不该说那种话。）

⚠️ 也有怀旧用法 〜たものだ（见 10.14）。

### 〜ことだ（建议）

**接续**：動詞辞書形 / ない形 + ことだ
**含义**：（你）应该……、最好……（建议、忠告）。

例：
- 健康のために、よく寝ることだ。（为了健康应该多睡觉。）
- 上達したければ、毎日練習することだ。（想进步就要每天练习。）
- 焦らないことだ。（不要着急为好。）

⚠️ 直接对对方说；不可用于自己。

## 10.12 终结与转换

### 〜ことになる / 〜ことにする（N2 升级用法）

**接续**：動詞辞書形 / ない形 + ことになる / ことにする
**含义**：
- **〜ことになる**：（被外部决定 / 自然发展为）变成……。
- **〜ことにする**：（自己主动）决定……。
- 〜ことになっている：规定 / 惯例上是……。
- 〜ことにしている：（自己）一直坚持……。

例：
- 来月から大阪に転勤することになった。（决定下个月调到大阪。）
- 留学することにした。（决定去留学。）
- 学校では制服を着ることになっている。（学校规定要穿校服。）

### 〜次第だ

**接续**：N + 次第だ
**含义**：取决于……、要看……。

例：
- 成功するかどうかは努力次第だ。（能否成功取决于努力。）
- 結果はあなた次第だ。（结果全看你。）
- 行くかどうかは天気次第だ。（去不去要看天气。）

⚠️ 与表"一……就……"的〜次第（见 10.1）形同义异，靠接续区分。

### 〜こととなる

**接续**：動詞辞書形 / ない形 + こととなる
**含义**：（书面正式版〜ことになる）（被）决定为……。

例：
- 来月から新制度が導入されることとなった。（决定从下个月起引入新制度。）
- 会議は延期されることとなった。（会议被决定延期。）
- この件は社長が判断することとなる。（此事将由社长来判断。）

⚠️ 多用于公告、文书、新闻。

## 10.13 重要书面体

### 〜ねばならない

**接续**：動詞ない形（去 ない）+ ねばならない（する → せねばならない）
**含义**：必须……（书面〜なければならない）。

例：
- 法を守らねばならない。（必须守法。）
- すぐ行動せねばならない。（必须立刻行动。）
- 真実を伝えねばならない。（必须传达真相。）

⚠️ 文章、社论、演讲常见；口语少。

### 〜ずに

**接续**：動詞ない形（去 ない）+ ずに（する → せずに）
**含义**：不……地（书面 〜ないで）。

例：
- 朝食を食べずに出かけた。（没吃早饭就出门。）
- 何も言わずに帰った。（一句话也没说就回去了。）
- 勉強せずに合格した。（没学习就考过了。）

### 〜ずにすむ / 〜ないですむ

**接续**：動詞ない形（去 ない）+ ずにすむ
**含义**：不用……就解决了 / 不必……（庆幸）。

例：
- 友達が貸してくれたので、買わずにすんだ。（朋友借给了，不用买了。）
- 雨が止んだので、傘を持っていかずにすんだ。（雨停了不用带伞。）
- 大した怪我ではなく、入院せずにすんだ。（伤不重不用住院。）

### 〜まま（N2 升级：〜ままに / 〜ままで）

**接续**：
- 動詞た形 / ない形 + まま / ままに
- イ形 + まま、ナ形 + な + まま、N + の + まま

**含义**：
- **〜まま**：保持原状。
- **〜ままに**：任由……、按照……的（被动地）。
- **〜ままで**：以……的状态。

例：
- 思うままに行動する。（随心所欲地行动。）
- 言われるままに従った。（任由别人说，照办了。）
- 昔のままで残っている。（保持着旧时的样子。）

⚠️ N3 已有「靴を履いたまま」基础义；N2 增加 〜ままに（任由）与书面 〜ままで（保持状态）。

## 10.14 句末与谓语表达

### 〜とは（惊讶 / 解释）

**接续**：簡体 + とは
**含义**：
- **惊讶**：竟然……、没想到……（句末或后接形容词 / 感叹）。
- **解释 / 定义**：所谓……（= 〜というのは）。

例：
- まさか彼が嘘をつくとは思わなかった。（没想到他会撒谎。）
- こんなに早く完成するとは、すごい。（这么快就完成，真厉害。）
- 「努力」とは、続けることだ。（所谓"努力"就是坚持。）

### 〜ということだ（传闻 / 总结）

**接续**：簡体 + ということだ
**含义**：
- **传闻**：据说……。
- **总结 / 解释**：也就是说……。

例：
- 田中さんは来月帰国するということだ。（据说田中下个月回国。）
- つまり、お金がないということだ。（也就是说没钱了。）
- 試験は中止になったということだ。（据说考试取消了。）

### 〜たものだ / 〜ものだった（怀旧）

**接续**：動詞た形 + ものだ
**含义**：以前常……（带怀旧 / 感慨）。

例：
- 子供のころ、よく川で泳いだものだ。（小时候常在河里游泳。）
- 学生時代はよく徹夜したものだ。（学生时代常熬夜。）
- 昔はここに大きな木があったものだ。（以前这里有棵大树。）

⚠️ 与 9.3「〜ものだ（常识 / 感叹）」靠**た形**区分；た形 → 怀旧专用。

| ようとしない / ようとする | 不肯 vs 试图 |
| かねる / かねない | 难以做 vs 可能（坏事）发生 |
| たところで / ても | 必无意义 vs 一般让步 |
| たら最後 / たとたん | 一旦……必负面 vs 瞬间意外 |
| ものだ（た形）/ ものだ（辞書形） | 怀旧 vs 常识 / 感叹 |

# 十一、最高级语法（N1）

> 本章约 115 个 N1 核心句型。N1 偏书面、文学、新闻、商务、演讲、合同条款等正式语境，口语日常较少出现。每条按"接续公式 + 含义 + 例句"组织。已在 N2 出现的 pattern 仅做 cross-ref 或升级补注，不重复。

## 11.1 时间・契机

### 〜が早いか

**接续**: 動詞辞書形 / 動詞た形 + が早いか
**含义**: 一……就立刻……（前后动作几乎同时发生，强调瞬间衔接）

例：
- ベルが鳴るが早いか、生徒たちは教室を飛び出した。（铃声一响，学生们就飞奔出教室。）
- 帰宅したが早いか、ソファに倒れ込んで寝てしまった。（一回到家就栽倒在沙发上睡着了。）
- 信号が青に変わるが早いか、車は一斉に発進した。（信号灯一变绿，车辆就齐齐起步。）

⚠️ 多用第三人称叙述、书面/新闻文体；后句不能是说话人的意志/命令/计划。
⚠️ 与 〜やいなや、〜なり、〜そばから 对比见下表。

### 〜や / 〜やいなや

**接续**: 動詞辞書形 + や / やいなや
**含义**: 一……立刻就……（书面语，文学色彩浓）

例：
- 大臣の発言が報じられるやいなや、株価は急落した。（大臣的发言一经报道，股价立刻暴跌。）
- ドアを開けるや、犬が飛び出してきた。（一开门，狗就冲了出来。）

⚠️ 「や」单用更文学化，多见于小说；后句同样不能是说话人意志。

### 〜なり

**接续**: 動詞辞書形 + なり
**含义**: 一……（紧接着就出乎意料地）……

例：
- 彼は部屋に入ってくるなり、何も言わずに泣き出した。（他一进房间，二话不说就哭了起来。）
- その知らせを聞くなり、彼女は顔色を変えた。（一听到那消息，她脸色就变了。）
- 立ち上がるなり、めまいがして倒れてしまった。（一站起来就头晕摔倒了。）

⚠️ 后句多为"出乎意料/异常"动作；与 〜が早いか 的中性叙述不同。
⚠️ 主语多为第三人称。

### 〜そばから

**接续**: 動詞辞書形 / 動詞た形 + そばから
**含义**: 刚……就（又）……（反复发生，含烦躁/无奈语气）

例：
- 教えるそばから忘れてしまうので、本当に困る。（刚教完就忘，真让人头疼。）
- 片付けるそばから子供が散らかす。（刚收拾好，孩子就又弄乱。）
- 稼ぐそばから使ってしまって、貯金できない。（刚赚到就花光，存不下钱。）

⚠️ 强调反复、循环、无效感；后句多为负面结果。

### 〜てからというもの

**接续**: 動詞て形 + からというもの
**含义**: 自从……以来（一直……，含较长跨度的状态变化）

例：
- 上京してからというもの、故郷に一度も帰っていない。（自从上京以来，一次也没回过老家。）
- 子供が生まれてからというもの、夜ぐっすり眠ったことがない。（自从孩子出生以来，就没安稳睡过一觉。）
- あの本を読んでからというもの、彼の人生観は一変した。（自从读了那本书以后，他的人生观完全变了。）

⚠️ 比 〜てから 强调"持续性变化"，更书面、感情色彩强。

### 〜にあって / 〜にあっては / 〜にあっても

**接续**: 名詞 + にあって（は / も）
**含义**:
1. 在……（特殊场合/时期）下（书面）
2. にあっては：在……这种立场下，理所当然……
3. にあっても：即使在……下也……

例：
- 戦時下にあって、人々は希望を失わなかった。（在战时之下，人们没有失去希望。）
- 危機的状況にあっては、迅速な判断が求められる。（在危机情况下，要求迅速判断。）
- 逆境にあっても、彼は笑顔を絶やさなかった。（即使在逆境中，他也始终保持微笑。）

⚠️ 多为重大场合/时期；不用于日常琐事。

### 〜にいたって / 〜にいたっては / 〜にいたっても

**接续**: 動詞辞書形 / 名詞 + に至って（は / も）
**含义**:
1. 到了……地步才……（事态发展到极端阶段）
2. 至于……（举出极端例子）
3. 即使到了……也……

例：
- 死者が出るに至って、ようやく当局が動き出した。（直到有人死亡，当局才终于行动。）
- 兄は東大、弟に至っては京大に合格した。（哥哥考上东大，至于弟弟，更是考上了京大。）
- 締切前日に至っても、彼はまだ手をつけていない。（即使到了截稿前一天，他还没动手。）

### 〜を皮切りに / 〜を皮切りとして / 〜を皮切りにして

**接续**: 名詞 + を皮切りに（として / にして）
**含义**: 以……为开端，（接二连三地……）

例：
- 東京公演を皮切りに、全国ツアーが始まる。（以东京公演为开端，全国巡演开始。）
- 彼の発言を皮切りに、議論は白熱した。（以他的发言为开端，讨论变得白热化。）
- 大阪支店の開設を皮切りとして、海外展開も視野に入れている。（以大阪分店的开设为开端，也将海外发展纳入视野。）

⚠️ 后句必须是"陆续展开/扩大"的积极发展；不用于负面事态。

### 〜ここに〜ない

**接续**: 名詞 + ここに〜ない（多与「今を逃しては」等组合）
**含义**: 罕见。强调"此刻/此时不容错过"，多见于演讲、报道。

例：
- ここに勝機を見出すしかない。（只能在此时此地寻得胜机。）

⚠️ 需 review：此模式作为独立 N1 项较冷僻，正式教材罕收，可视作语用表达。

## 11.2 列举与并列

### 〜つ〜つ

**接续**: 動詞ます形 + つ + 同一動詞受身/対動詞ます形 + つ
**含义**: 一会儿……一会儿……（两动作交替/相互；文语残留）

例：
- 二人は持ちつ持たれつの関係だ。（两人是互相支持的关系。）
- 行きつ戻りつ、彼は駅前を歩き回っていた。（来来回回，他在车站前走来走去。）
- 抜きつ抜かれつの接戦が続いた。（你超我赶的拉锯战持续着。）

⚠️ 用法固定，常见组合有限：持ちつ持たれつ／差しつ差されつ／行きつ戻りつ／抜きつ抜かれつ／浮きつ沈みつ 等。

### 〜やら〜やら

⚠️ N2 章已详述（杂多列举）。N1 升级：在书面体中可与抽象名词/感情名词搭配，表达"复杂情绪交织"，如「嬉しいやら恥ずかしいやら」。

### 〜だの〜だの

**接续**: 名詞/普通形 + だの + 名詞/普通形 + だの
**含义**: 又是……又是……（列举不满/抱怨对象，口语化带贬义）

例：
- 給料が安いだの休みが少ないだの、文句ばかり言っている。（又是工资低又是假少，净说怨言。）
- 寿司だの天ぷらだの、好きなものばかり頼んだ。（寿司啦天妇罗啦，净点喜欢的。）
- 太っただの老けただの、人の容姿に口を出すな。（又是胖了又是老了，别对别人的外表指手画脚。）

⚠️ 含贬义/不耐烦；正式场合避免。

### 〜といい〜といい

**接续**: 名詞 + といい + 名詞 + といい
**含义**: 无论从……还是从……来看，都……（举两面评价整体）

例：
- 色といいデザインといい、この服は申し分ない。（无论是颜色还是款式，这件衣服都无可挑剔。）
- 味といい盛り付けといい、さすが一流店だ。（味道和摆盘都不愧是一流餐厅。）
- 父といい母といい、頑固な性格だ。（父亲也好母亲也好，都是顽固性格。）

⚠️ 评价两个面/两个人都同方向；不能用于动词或形容词，仅接名词。

### 〜といわず〜といわず

**接续**: 名詞 + といわず + 名詞 + といわず
**含义**: 无论……还是……（全部都……，全面性强调）

例：
- 顔といわず手といわず、泥だらけだった。（脸也好手也好，全是泥。）
- 平日といわず週末といわず、彼は働き詰めだ。（不分平日周末，他都在拼命工作。）
- 国内といわず海外といわず、彼の作品は高く評価されている。（无论国内还是海外，他的作品都广受好评。）

⚠️ 比 〜といい〜といい 更强调"全方位/无例外"；后句多为同一状态。

### 〜であれ〜であれ

**接续**: 名詞 + であれ + 名詞 + であれ
**含义**: 无论是……还是……，都……（让步并列）

例：
- 男であれ女であれ、機会は平等であるべきだ。（无论男女，机会都应平等。）
- 賛成であれ反対であれ、自分の意見を述べてほしい。（无论赞成还是反对，希望阐述自己的意见。）
- 雨であれ雪であれ、試合は予定通り行う。（无论雨雪，比赛照常举行。）

⚠️ 书面/演讲；同义 〜にしろ〜にしろ 更口语。

### 〜なり〜なり

**接续**: 動詞辞書形 / 名詞 + なり + 動詞辞書形 / 名詞 + なり
**含义**: 或者……或者……（从中选其一，建议性列举）

例：
- 電話なりメールなりで連絡してください。（请用电话或邮件联系。）
- 親に相談するなり、先生に聞くなりしたほうがいい。（最好找父母商量或问老师。）
- 焼くなり煮るなり、好きに調理していい。（烤也好煮也好，随你怎么做。）

⚠️ 提示"任选其一"的建议；不用于已发生的事实列举。

### 〜にしろ〜にしろ / 〜にせよ〜にせよ

**接续**: 名詞 / 動詞普通形 + にしろ / にせよ
**含义**: 不管是……还是……，都……

例：
- 行くにしろ行かないにしろ、早めに連絡してほしい。（去也好不去也好，希望早点联系。）
- 賛成にせよ反対にせよ、理由を明示すべきだ。（不管赞成还是反对，都应明示理由。）
- 子供にしろ大人にしろ、ルールは守らねばならない。（不管小孩大人，都得守规则。）

⚠️ 〜にせよ 更书面；〜にしろ 略口语。

### 〜とも〜とも

**接续**: 名詞 / 普通形 + とも + 名詞 / 普通形 + とも
**含义**: 既不……也不……（无法明确判断、含糊不清）

例：
- 行くとも行かないとも、はっきり言わない。（说不清是去还是不去。）
- 賛成とも反対とも取れる返事だった。（是个既能解读为赞成也能解读为反对的回答。）
- 好きとも嫌いとも言えない、複雑な気持ちだ。（说不上喜欢也说不上讨厌，心情复杂。）

⚠️ 强调"两可不明"的状态。

## 11.3 让步与逆接

### 〜とはいえ

⚠️ N2 章已详述。N1 升级：可后接体言或形动用法「〜とはいえ、…である」，更书面。

### 〜ものの

⚠️ N2 章已详述。N1 升级：书面叙述中常以「〜とはいうものの」加强让步。

### 〜にもかかわらず

⚠️ N2 章已详述。N1 中常用于报道/正式书面：「〜にもかかわらず」单独作衔接副词。

### 〜ながらも

**接续**: 動詞ます形 / イ形 / ナ形 / 名詞 + ながら(も)
**含义**: 虽然……但是……（带文学/书面色彩的让步）

例：
- 狭いながらも、自分の家を持てて満足している。（虽然狭小，但能拥有自己的家就满足了。）
- 子供ながらも、彼は鋭い洞察力を持っている。（虽然是孩子，他却有敏锐的洞察力。）
- 不本意ながらも、その提案を受け入れた。（虽然不情愿，还是接受了那个提案。）

⚠️ 比 〜けれども 文学；常与"小/少/不愿"等限定语共现。

### 〜ところを

**接续**: 動詞普通形 / イ形 / ナ形な / 名詞の + ところを
**含义**:
1. 正在……之时（多用于客套）
2. 本应……却……（让步）

例：
- お忙しいところを恐縮ですが、少々お時間をいただけますか。（百忙之中打扰，能否占用您一点时间？）
- 本来なら直接お伺いするところを、メールで失礼します。（本应亲自前去，谨以邮件请安。）
- 寝ているところを起こされて、機嫌が悪い。（正睡着被叫醒，心情很糟。）

⚠️ 客套用法极常见于商务邮件。

### 〜ものを

**接续**: 動詞・イ形・ナ形・名詞 の普通形（連体形）+ ものを
**含义**: 明明……可是……（含遗憾、责备、不满）

例：
- 一言相談してくれればよかったものを、勝手に決めるなんて。（明明商量一句就好了，竟然擅自决定。）
- 早く病院に行けばすぐ治ったものを。（早点去医院就能治好的呀。）
- 黙っていればいいものを、余計なことを言って怒らせてしまった。（明明闭嘴就好了，偏要多嘴惹他生气。）

⚠️ 句末常省略下文，留余韵；含强烈遗憾/责备语气。

### 〜ばこそ

**接续**: 動詞ば形 / イ形ければ / ナ形であれば / 名詞であれば + こそ
**含义**: 正因为……才……（强调原因，书面/演讲）

例：
- 努力すればこそ、成功が得られる。（正因为努力才能获得成功。）
- 君を信頼していればこそ、こんなことを頼むのだ。（正因为信任你，才拜托这种事。）
- 厳しく指導するのも、君の将来を思えばこそだ。（严格指导，也是因为想到你的未来。）

⚠️ 书面色彩浓，演讲/作文常用；与 〜からこそ 相比更文学。

### 〜たって / 〜だって

**接续**: 動詞た形 / イ形かった / ナ形・名詞 + たって / だって
**含义**: 即使……也……（〜ても 的口语形，但 N1 中也用于书面对照）

例：
- いくら謝ったって、もう許してもらえないだろう。（再怎么道歉也不会被原谅了吧。）
- 子供だって、それぐらいわかる。（即使是孩子也懂这点。）
- 雨が降ったって、試合は中止しない。（即使下雨，比赛也不取消。）

⚠️ 口语形，但与"いくら～たって"等组合在书面也常见。

### 〜とて

**接续**: 名詞 / 動詞普通形 + とて
**含义**: 即使……也……（书面/古风的 〜ても / 〜だって）

例：
- 親とて、子供のすべてを理解できるわけではない。（即便是父母，也不可能理解孩子的全部。）
- 専門家とて、間違うことはある。（即便是专家也会犯错。）
- どんなに頼まれたとて、引き受けるつもりはない。（无论怎么被请求，也不打算接受。）

⚠️ 书面/演讲；常与「とて〜ない」搭配。

### 〜とはいっても

**接续**: 名詞 / 普通形 + とはいっても
**含义**: 虽说……但……（让步，比 〜とはいえ 略口语）

例：
- 春とはいっても、まだ寒い日が続いている。（虽说是春天，寒冷的日子还在继续。）
- 合格したとはいっても、油断はできない。（虽说通过了，也不能掉以轻心。）
- 友達とはいっても、そこまで親しいわけではない。（虽说是朋友，也没那么亲密。）

### 〜と思いきや

**接续**: 普通形 + と思いきや
**含义**: 本以为……没想到却……（反差/出乎意料）

例：
- 簡単な問題と思いきや、意外に難しかった。（本以为是简单问题，意外地难。）
- 彼は反対するかと思いきや、すんなり賛成した。（本以为他会反对，没想到爽快赞成。）
- 雨かと思いきや、すぐ晴れてきた。（以为要下雨，结果马上晴了。）

⚠️ 「思いき"や"」为古语助词残留；多用于书面叙述。

### 〜にひきかえ

⚠️ N2 章已详述。N1 升级：可置于段首作"对比转换标记"，文学性更强。

## 11.4 限定・极端・例外

### 〜だに

**接续**: 動詞辞書形 / 名詞 + だに
**含义**: 连……都（不/也）……（极端例举，书面/文学）

例：
- 想像するだに恐ろしい光景だった。（光是想象就令人毛骨悚然的景象。）
- そんなことは夢にだに思わなかった。（这种事连做梦都没想过。）
- 微動だにしない彼の姿に、皆驚いた。（他纹丝不动的样子，让大家震惊。）

⚠️ 接续有限，常见搭配：想像するだに／聞くだに／夢にだに思わない／微動だにしない／一顧だにしない 等。

### 〜たりとも

**接续**: 数詞 + たりとも〜ない
**含义**: 哪怕一……也不……（强调否定的极端）

例：
- 一日たりとも油断はできない。（哪怕一天也不能松懈。）
- 一円たりとも無駄にしてはならない。（一日元也不能浪费。）
- 一瞬たりとも目を離さないでください。（哪怕一瞬都别移开视线。）

⚠️ 数词后必须接「一+助数詞」类极小单位；后接否定。

### 〜なりに / 〜なりの

**接续**: 名詞 / 普通形 + なりに / なりの + 名詞
**含义**: 与……相称地（虽然有限，但尽自己所能；表"以适合各自身份/水平的方式"）

例：
- 子供は子供なりに、いろいろ考えている。（孩子也以孩子的方式在思考各种事。）
- 私なりに精一杯やったつもりです。（我已尽我所能去做了。）
- 新人には新人なりの悩みがある。（新人有新人的烦恼。）

⚠️ 含"虽不够完美但尽力"的认同语气；不要用于否定/贬低对方。

### 〜ならでは

**接续**: 名詞 + ならでは（の + 名詞）
**含义**: 只有……才有的；……特有的（正面评价）

例：
- これは京都ならではの味だ。（这是只有京都才有的味道。）
- ベテランならではの落ち着いた対応だった。（不愧是老手才有的沉稳应对。）
- この祭りは地元ならではの活気に満ちている。（这个节日充满了当地特有的活力。）

⚠️ 必为褒义；不能说「素人ならではの失敗」（不通；该用 〜ならではの〜 限定积极独特性）。

### 〜にとどまらず

**接续**: 動詞辞書形 / 名詞 + にとどまらず
**含义**: 不仅仅停留在……，还……（范围扩张）

例：
- 影響は国内にとどまらず、海外にまで及んだ。（影响不仅限于国内，还波及海外。）
- 彼の活動は音楽にとどまらず、映画にも及んでいる。（他的活动不仅限于音乐，还涉足电影。）
- 問題は経済にとどまらず、社会全体に広がっている。（问题不止于经济，正扩散到整个社会。）

⚠️ 书面/报道用语。

### 〜にもまして

**接续**: 名詞 + にもまして
**含义**: 比……更（在 N 之上又添加程度）

例：
- 以前にもまして、彼は熱心に勉強している。（比以前更加用功学习。）
- 何にもまして健康が大切だ。（健康比任何东西都重要。）
- 今年は去年にもまして暑い。（今年比去年还热。）

⚠️ 与"〜より"相比强调"已经很高的上面再加一层"；多用于书面。

### 〜のみならず

⚠️ N2 章已详述。N1 升级：常与「〜のみならず…さえ / すら」呼应，强化"范围扩散"，书面新闻常见。

### 〜はおろか

⚠️ N2 章已详述。N1 升级：后续常配「〜さえ / すら / も〜ない」，更书面、批判语气强。

### 〜まで(も)ない

**接续**: 動詞辞書形 + まで(も)ない
**含义**: 没必要……（程度未到需要……的地步）

例：
- 言うまでもなく、健康が一番大切だ。（不用说，健康最重要。）
- 病院に行くまでもない、軽い怪我だ。（不用去医院，轻伤。）
- わざわざ確認するまでもないことだ。（不必特意确认的事。）

⚠️ 与 〜ことはない（不必）相似，但 〜までもない 更强调"未到那程度"的不必要。

### 〜まじ / 〜まじき

**接续**: 動詞辞書形 + まじき + 名詞 / まじ
**含义**: 不应该……的（古风/书面，强烈道德谴责）

例：
- 教師にあるまじき言動だ。（这是不该出现在教师身上的言行。）
- 子を持つ親にあるまじき行為である。（这是身为父母不该有的行为。）
- 政治家として許すまじき発言だ。（作为政治家不容饶恕的发言。）

⚠️ 限固定搭配「〜にあるまじき〜」「〜まじき行為/言動/態度」；古文助动词「まじ」连体形。书面/新闻。

### 〜ばかりが〜ではない

**接续**: 名詞 + ばかりが + 動詞普通形 + ではない
**含义**: 并非只有……才……（否定单一限定）

例：
- お金ばかりが幸せをもたらすのではない。（并非只有金钱带来幸福。）
- 結果ばかりが大切なのではない、過程も重要だ。（不光结果重要，过程也重要。）
- 彼ばかりが悪いのではない。（也不光是他错。）

## 11.5 推断与判断

### 〜にかたくない

**接续**: 名詞 / 動詞辞書形 + にかたくない
**含义**: 不难……（不难想象/不难理解）

例：
- 彼女の心情は察するにかたくない。（不难体察她的心情。）
- 当時の苦労は想像にかたくない。（当时的辛苦不难想象。）
- 彼の怒りは理解するにかたくない。（不难理解他的愤怒。）

⚠️ 接续动词有限：察する／想像する／理解する／推察する 等心理动词。

### 〜にたえる / 〜にたえない

**接续**: 動詞辞書形 / 名詞 + にたえる / にたえない
**含义**:
1. にたえる：值得……（鑑賞・批判 etc）
2. にたえない：①不堪……（看下去/听下去）②不胜……（感慨/感谢）

例：
- 彼の作品は鑑賞にたえる。（他的作品值得欣赏。）
- 見るにたえない惨状だった。（不忍卒睹的惨状。）
- ご厚情、感謝にたえません。（您的厚意，不胜感激。）

⚠️ 第二种"不胜……"仅与感谢・感激・遗憾等情感名词搭配，多见演讲/书信。

### 〜には当たらない / 〜に当たる

**接续**: 動詞辞書形 / 名詞 + には当たらない
**含义**: 不必……（用不着惊讶/责备/感谢等）

例：
- そのくらいで驚くには当たらない。（这点事不值得惊讶。）
- 彼を責めるには当たらない、誰でも間違える。（不必责备他，谁都会出错。）
- 感心するには当たらない、当然の結果だ。（不必佩服，是当然的结果。）

⚠️ 接续动词限"心理反应类"：驚く／褒める／感心する／責める／非難する 等。

### 〜にたる / 〜に足る

**接续**: 動詞辞書形 / 名詞 + に足る
**含义**: 值得……的（具备充分的资格/价值）

例：
- 彼は信頼するに足る人物だ。（他是值得信赖的人。）
- 満足するに足る成果が得られた。（取得了令人满意的成果。）
- 尊敬に足る指導者である。（是值得尊敬的领导。）

⚠️ 书面/演讲；接续动词多为"信頼する／尊敬する／満足する／驚く" 等。

### 〜きらいがある

**接续**: 動詞辞書形 / 動詞ない形 / 名詞の + きらいがある
**含义**: 有……的倾向（多为负面）

例：
- 彼は物事を悲観的に考えるきらいがある。（他有把事情往悲观方面想的倾向。）
- 最近の若者は本を読まないきらいがある。（最近的年轻人有不读书的倾向。）
- この方法は時間がかかりすぎるきらいがある。（这种方法有过于费时的倾向。）

⚠️ 仅用于负面倾向；不要用于正面性格。

### 〜にして

**接续**: 名詞 + にして
**含义**:
1. 既是 N 又是 N
2. 因为是 N 才能……（强调主体的特殊性）
3. 在 N 这一时点/阶段

例：
- 彼は学者にして政治家でもある。（他既是学者又是政治家。）
- 名人にしてもこの問題は難しい。（即便是名人，这道题也难。）
- 70 歳にして再び挑戦を始めた。（70 岁开始重新挑战。）

⚠️ 与「〜ともなれば」「〜ならでは」语义有重叠，需依语境判断。

### 〜の至り / 〜の極み

**接续**: 名詞 + の至り / の極み
**含义**: ……之至；极尽……（感情、状态达到极点；书面/正式）

例：
- 受賞は光栄の至りです。（获奖荣幸之至。）
- お会いできて感激の至りです。（能见到您感激不尽。）
- 贅沢の極みを尽くした料理だった。（极尽奢华的料理。）

⚠️ 「の至り」多用于感情（光栄／感激／感謝）；「の極み」可用于状态（贅沢／疲労／幸福）。书面祝辞、致辞常见。

### 〜やもしれない / 〜やもしれぬ

**接续**: 動詞普通形 / 名詞 + やもしれない
**含义**: 也许……（「かもしれない」的古风/文学形）

例：
- 雨が降るやもしれぬ。（也许会下雨。）
- 真実は彼にしかわからぬやもしれない。（也许只有他才知道真相。）
- 大事に至るやもしれない。（也许会演变成大事。）

⚠️ 古风、文学；小说/严肃文章中出现，日常对话不用。

### 〜ともなく / 〜ともなしに

**接续**: 動詞辞書形 / 疑問詞 + ともなく / ともなしに
**含义**: 不经意地……；无意中……

例：
- 見るともなしにテレビを見ていた。（漫不经心地看着电视。）
- 聞くともなく彼らの会話が耳に入ってきた。（无意中听到了他们的对话。）
- どこからともなく、いい香りが漂ってきた。（不知从哪儿飘来一阵香气。）

⚠️ 与疑问词组合「どこからともなく／いつからともなく」表"不知何处/何时"。

### 〜ふしがある

**接续**: 動詞普通形 / 名詞の + ふしがある
**含义**: 有……的迹象/苗头；有某种可疑之处

例：
- 彼の話には不審なふしがある。（他的话里有可疑之处。）
- 嘘をついているふしがある。（有撒谎的迹象。）
- 彼女は何かを隠しているふしがある。（她似乎隐瞒着什么。）

⚠️ 书面/侦探/新闻；多含"察觉到细微迹象"语感。

## 11.6 强调与必然

### 〜ずにはおかない / 〜ないではおかない

**接续**: 動詞ない形（去 ない）+ ずにはおかない／ないではおかない
**含义**:
1. 必然会引起……（自然结果）
2. 一定要……（强烈意志）

例：
- 彼の演説は人々を感動させずにはおかない。（他的演讲必然让人们感动。）
- この事件の真相を明らかにせずにはおかない。（一定要查明此案真相。）
- 不正は必ず暴かないではおかない。（一定要揭露不正。）

⚠️ 「する→せずにはおかない」（特殊）。书面/演讲；意志用法主语多为强烈意志者。

### 〜ずにはすまない / 〜ないではすまない

**接续**: 動詞ない形（去 ない）+ ずにはすまない／ないではすまない
**含义**: 不……不行（社会/道义上必须）

例：
- 約束した以上、果たさずにはすまない。（既然约定了，就不能不履行。）
- これだけ迷惑をかけたら、謝らないではすまない。（添了这么多麻烦，不道歉不行。）
- 全額弁償せずにはすまないだろう。（恐怕不得不全额赔偿。）

⚠️ 与 〜ないではおかない 区别：前者"社会/道义压力"，后者"自然必然/强烈意志"。

### 〜てやまない

**接续**: 動詞て形 + やまない
**含义**: ……不已；衷心地……（感情持续不断）

例：
- ご成功を願ってやみません。（衷心祝愿您成功。）
- 故郷を愛してやまない。（深爱着故乡。）
- 平和を祈ってやまない。（衷心祈祷和平。）

⚠️ 仅与感情类动词搭配：愛する／願う／祈る／期待する／尊敬する 等；演讲/书信。

### 〜てはばからない

**接续**: 動詞て形 + はばからない
**含义**: 毫无顾忌地……；公然……

例：
- 彼は政府を批判してはばからない。（他毫无顾忌地批判政府。）
- 自分が一番だと公言してはばからない。（公然宣称自己最厉害。）
- 不満を口にしてはばからない態度だった。（毫不避讳地表达不满的态度。）

⚠️ 多用于"公开宣称/公然批评"；含中性偏批判色彩。

### 〜にかかっている

**接续**: 名詞 / 疑問詞節 + にかかっている
**含义**: 取决于……

例：
- 成功するかどうかは、君の努力にかかっている。（成败取决于你的努力。）
- 会社の将来は社長の決断にかかっている。（公司未来取决于社长的决断。）
- すべては明日の試合にかかっている。（一切都取决于明天的比赛。）

### 〜あっての

**接续**: 名詞 + あっての + 名詞
**含义**: 有了……才有……（前者是后者的必要前提）

例：
- お客様あっての商売だ。（有顾客才有生意。）
- 健康あっての人生だ。（有健康才有人生。）
- 努力あっての成功である。（努力才能成功。）

⚠️ 常用于格言式表达；后续名词常省略「だ／である」结句。

### 〜が最後 / 〜たら最後

⚠️ N2 章已详述。N1 升级：书面/小说中常用 〜が最後（更文学），强调"一旦……就再也无法挽回"。

### 〜なくしては / 〜なくして

**接续**: 名詞 + なくして(は) / 動詞ない形 + ことなくして(は)
**含义**: 没有……就（不能/不可能）……

例：
- 努力なくして成功はあり得ない。（没有努力就不可能成功。）
- 君の支えなくしては、ここまで来られなかった。（没有你的支持，我走不到今天。）
- 信頼関係なくしてビジネスは成り立たない。（没有信任关系，生意做不起来。）

⚠️ 书面/演讲；后句必接否定/不可能。

### 〜とあって

**接续**: 名詞 / 普通形 + とあって
**含义**: 因为是……（特殊场合），所以……（描述性，多用于报道）

例：
- 連休とあって、観光地は大混雑だ。（因为是连休，观光地非常拥挤。）
- 有名歌手の来日とあって、空港は人で溢れた。（因为是著名歌手访日，机场人满为患。）
- 久しぶりの再会とあって、話が尽きない。（因为是久别重逢，话题不断。）

⚠️ 报道/新闻文体；不用于第一人称的主观理由。

### 〜とあれば

**接续**: 名詞 / 普通形 + とあれば
**含义**: 若是……的话（就……）（带特殊情况下的让步/决心）

例：
- 君の頼みとあれば、断れない。（若是你的请求，无法拒绝。）
- 必要とあれば、いつでも駆けつける。（若有必要，随时赶到。）
- 子供のためとあれば、何でもする。（若是为了孩子，什么都做。）

⚠️ 与 〜とあって 区别：〜とあれば 假设/条件，〜とあって 既成事实+因果。

### 〜ともあろう（者・人・もの）

**接续**: 名詞 + ともあろう + 者 / 人 / もの
**含义**: 身为……（这样有身份的人）竟然……（含批评/震惊）

例：
- 警察官ともあろう者が、犯罪に手を染めるとは。（堂堂警官竟然涉案。）
- 大臣ともあろう人物が、こんな失言をするとは。（堂堂大臣竟说出这种失言。）
- 専門家ともあろうものが、こんな基本を間違えるとは。（堂堂专家竟弄错这种基础。）

⚠️ 句末常用「〜とは／〜なんて」呼应；强烈反差/失望语气。

### 〜にして（強調・限定）

**接续**: 名詞 + にして
**含义**: 唯独……才……；仅在……时

例：
- 70 歳にしてマラソンに挑む。（70 岁才挑战马拉松。）
- 彼にしてできなかったのだから、誰にもできないだろう。（连他都做不到，谁都做不到吧。）
- 一夜にして街は変わり果てた。（一夜之间街道面目全非。）

⚠️ 与 11.5 中 〜にして 重叠；多义模式，需依语境判断。

## 11.7 状态与样态

### 〜ずくめ

**接续**: 名詞 + ずくめ
**含义**: 清一色……；全是……

例：
- 黒ずくめの服装で現れた。（穿一身全黑出现。）
- 今年はいいことずくめの一年だった。（今年是好事连连的一年。）
- 規則ずくめの生活にうんざりしている。（厌倦了规矩繁多的生活。）

⚠️ 接续名词有限：黒／白／いいこと／規則／ご馳走 等固定搭配为主。

### 〜めく

**接续**: 名詞 + めく（动词化）
**含义**: 带有……的样子；显得有……气息

例：
- 春めいた陽気になった。（变得有春意盎然的气候。）
- 謎めいた笑みを浮かべた。（露出谜一般的微笑。）
- 皮肉めいた言い方をしないでくれ。（别用带讽刺意味的说法。）

⚠️ 接续有限：春／秋／皮肉／謎／怒り／冗談 等。

### 〜まみれ

⚠️ N2 章已详述（液体/粉末附着）。N1 升级：可比喻抽象事物，如「借金まみれ／嘘まみれ」。

### 〜ぐるみ

⚠️ N2 章已详述（家族・地域全体卷入）。N1 升级：可与抽象团体名词搭配，强调"集体性"。

### 〜だらけ

⚠️ N2 章已详述。N1 升级：可与抽象名词如「矛盾／間違い／欠点」搭配。

### 〜ながらに / 〜ながらの

**接续**: 名詞 / 動詞ます形 + ながらに / ながらの + 名詞
**含义**: 保持……的状态；……着的

例：
- 涙ながらに語った。（含泪诉说。）
- 生まれながらの才能だ。（与生俱来的才能。）
- 昔ながらの町並みが残っている。（保留着古朴的街景。）

⚠️ 接续有限：生まれ／涙／いつも／昔／居 等；不与一般动词自由组合。

## 11.8 立场・对象（升级）

### 〜にしてからが

**接续**: 名詞 + にしてからが
**含义**: 就连……都……（举出极端例子，连这都那样何况其他）

例：
- 専門家にしてからが、この問題は解けない。（就连专家都解不开这问题。）
- 親にしてからが子供の気持ちを理解していない。（连父母都不理解孩子的心情。）
- 自分にしてからが間違いだらけなのに、人を批判できない。（连自己都满是错误，还批评别人。）

⚠️ 与 〜でさえ 类似但更书面、强烈；后句多为否定/批判。

### 〜をもって

**接续**: 名詞 + をもって
**含义**:
1. 用……（手段/方式）
2. 以……（时点）（书面公告）

例：
- 本日をもって閉店いたします。（本店今日起停业。）
- 誠意をもって対応する。（以诚意应对。）
- 書面をもって通知する。（书面通知。）

⚠️ 正式公告/合同/通知用语。

### 〜を限りに

**接续**: 名詞 + を限りに
**含义**: 以……为最后（结束）

例：
- 今日を限りに禁煙する。（从今日起戒烟。）
- 今年を限りに引退します。（今年退役。）
- 今月を限りに閉店いたします。（本月起停业。）

⚠️ 「声を限りに」是惯用语，意为"声嘶力竭"，例外用法。

### 〜を余儀なくされる / 〜を余儀なくさせる

**接续**: 名詞 + を余儀なくされる / を余儀なくさせる
**含义**:
1. を余儀なくされる：被迫不得不……（被动）
2. を余儀なくさせる：迫使……（使役）

例：
- 台風のため、試合は中止を余儀なくされた。（因台风，比赛被迫中止。）
- 経営難が事業の縮小を余儀なくさせた。（经营困难迫使业务缩减。）
- 病気で入院を余儀なくされた。（因病不得不住院。）

⚠️ 书面/新闻；含"客观情势不可抗"语感。

### 〜をおいて

**接续**: 名詞 + をおいて〜ない
**含义**: 除了……没有……（强烈限定）

例：
- この仕事を任せられるのは彼をおいてほかにいない。（能托付这工作的，除他无人。）
- 彼女をおいて、適任者はいない。（除她之外没有合适人选。）
- 今をおいてチャンスはない。（除此之外没有机会。）

⚠️ 后必接否定；强调"唯一"。

### 〜をよそに

**接续**: 名詞 + をよそに
**含义**: 不顾……；无视……

例：
- 親の心配をよそに、彼は遊び歩いている。（不顾父母的担心，他整天玩。）
- 周囲の反対をよそに、結婚を決めた。（不顾周围反对，决定结婚。）
- 国民の不安をよそに、政府は対応を遅らせた。（无视国民的不安，政府推迟应对。）

⚠️ 多含批判语气；后句为"不顾该关心的事"的行为。

### 〜をものともせず(に)

**接续**: 名詞 + をものともせず(に)
**含义**: 不把……当回事；不畏……（正面评价）

例：
- 強風をものともせず、登山を続けた。（不畏强风继续登山。）
- 困難をものともせずに、彼は前進した。（不畏困难，他向前迈进。）
- 周囲の批判をものともせず、自分の信念を貫いた。（不顾周围的批判，贯彻自己的信念。）

⚠️ 与 〜をよそに 区别：〜をものともせず 褒义（不畏障碍），〜をよそに 贬义（无视应关心之事）。

### 〜ともなれば / 〜ともなると

**接续**: 名詞 / 動詞辞書形 + ともなれば / ともなると
**含义**: 一旦到了……（这种身份/时期），就……

例：
- 社長ともなれば、責任も重大だ。（一旦当上社长，责任也重大。）
- 50 歳ともなると、体力の衰えを感じる。（一到 50 岁，就感到体力衰退。）
- 連休ともなれば、高速道路は渋滞する。（一到连休，高速就堵车。）

⚠️ 强调"到了某阶段就当然/必然如此"。

## 11.9 判断与表述

### 〜にあらず

**接续**: 名詞 + にあらず
**含义**: 并非……（古风/书面，「ではない」的文语形）

例：
- 人にあらざる行為だ。（不是人该有的行为。）
- それは事実にあらず。（那并非事实。）
- 我々の本意にあらず。（并非我们的本意。）

⚠️ 文语；连体形「にあらざる」、终止形「にあらず」；多见小说、宣言、宗教文。

### 〜ものとして / 〜ものとする

**接续**: 動詞普通形 + ものとして / ものとする
**含义**:
1. 当作……（视为）
2. 规定为……（合同条款）

例：
- 彼は来ないものとして話を進めよう。（就当他不来推进吧。）
- 本契約は両者署名をもって成立するものとする。（本合同自双方签字起成立。）
- 異議がないものとみなす。（视为无异议。）

⚠️ 「〜ものとする」多见合同/法律条款。

### 〜までだ / 〜までのことだ

**接续**: 動詞辞書形 / 動詞た形 + までだ / までのことだ
**含义**:
1. 大不了……（不行的话就……）
2. 不过是……而已（解释行为）

例：
- 断られたら、また頼むまでだ。（被拒绝就再请求一次而已。）
- ただ事実を述べたまでのことだ。（不过陈述事实而已。）
- だめなら諦めるまでだ。（不行就放弃罢了。）

⚠️ 句末决断/解释语气强；与「までもない」（不必）不同。

### 〜とばかりに

**接续**: 普通形 / 命令形 + とばかりに
**含义**: 简直就像在说"……"似的（夸张的态度）

例：
- 帰れとばかりに、彼はドアを指さした。（他指着门，仿佛在说"滚出去"。）
- ここぞとばかりに、彼は反論を始めた。（他抓住时机般开始反驳。）
- 今が好機とばかりに、彼は勝負に出た。（他像在说"机不可失"似的出招。）

⚠️ 「ここぞとばかりに」是惯用，意为"瞅准时机"。

### 〜と相まって / 〜とあいまって

**接续**: 名詞 + と相まって
**含义**: 与……相结合（产生某种效果）

例：
- 美しい景色が音楽と相まって、感動を呼んだ。（美景与音乐相互交织，引发感动。）
- 努力が才能とあいまって、彼は成功を収めた。（努力与才华相辅相成，他获得了成功。）
- 円安が原油高と相まって、物価が高騰している。（日元贬值与油价高企相叠加，物价飞涨。）

⚠️ 书面；多用于"两要素叠加产生效果"的客观描述。

### 〜にいたるまで

**接续**: 名詞 + にいたるまで
**含义**: 直到……（极小/极远的范围都……，强调全面）

例：
- 服装から髪型にいたるまで、規則で決められている。（从服装到发型都由规则决定。）
- 子供から老人にいたるまで、皆この歌を知っている。（从小孩到老人，大家都知道这首歌。）
- 細部にいたるまで、徹底的にチェックした。（连细节都彻底检查了。）

⚠️ 多与"〜から〜にいたるまで"搭配，强调"全方位无遗漏"。

### 〜にせよ

⚠️ N2 章已详述（让步）。N1 升级：常用于"〜にせよ〜にせよ"对偶并列（见 11.2）。

### 〜ことだし

**接续**: 普通形（連体形）+ ことだし
**含义**: 既然……，那就……（多个轻原因之一）

例：
- 天気もいいことだし、散歩に行こうか。（天气也不错，去散步吧。）
- もう遅いことだし、そろそろ帰ろう。（也晚了，差不多回去吧。）
- お腹も空いたことだし、何か食べよう。（也饿了，吃点什么吧。）

⚠️ 比 〜から 委婉；列举的是众多原因中的"较轻一项"，含建议/邀请语气。

## 11.10 文学/书面体表达

### 〜べからず / 〜べからざる

**接续**: 動詞辞書形 + べからず / べからざる + 名詞
**含义**: 不可……；不应……（古风、告示）

例：
- 芝生に入るべからず。（禁止进入草坪。）
- 許すべからざる行為だ。（不可饶恕的行为。）
- 欠くべからざる存在となった。（成了不可或缺的存在。）

⚠️ 文语；多见告示牌、严肃文章、惯用搭配「欠くべからざる」。「するべからず」可作「すべからず」。

### 〜まじき

⚠️ 见 11.4。重复 cross-ref。

### 〜ねば / 〜ねばならぬ

**接续**: 動詞ない形（去 ない）+ ねば / ねばならぬ
**含义**: 必须……（「なければ」的文语形）

例：
- 急がねばならぬ。（必须快点。）
- 真実を語らねばなるまい。（必须说出真相。）
- 約束は守らねばならない。（必须遵守约定。）

⚠️ 书面；「する→せねば」（特殊）。

### 〜のなんのって / 〜のなんの

**接续**: 普通形 + のなんのって
**含义**: 别提多么……了（强调感叹，口语）

例：
- 寒いのなんのって、手がかじかんで動かない。（别提多冷了，手都冻僵了。）
- 嬉しいのなんのって、飛び上がりたいぐらいだった。（别提多高兴了，简直想跳起来。）
- 怒ったのなんのって、口もきいてくれない。（气得不行，话都不肯说。）

⚠️ 口语强调；多与感情形容词搭配。

### 〜とは

**接续**: 普通形 + とは
**含义**: 竟然……（震惊、感叹）

⚠️ N2 章已详述。N1 升级：句末常省略后文（「〜とは。」），余韵更强。

### 〜とはいえども

**接续**: 名詞 / 普通形 + とはいえども
**含义**: 虽说……（「とはいえ」的文语强化形）

例：
- 子供とはいえども、約束は守るべきだ。（虽说是孩子，也该守约。）
- 古いとはいえども、まだ十分使える。（虽说旧，但还能用。）

⚠️ 书面/演讲；古风色彩。

### 〜ようと(も)〜まいと(も) / 〜ようが〜まいが

**接续**: 動詞意向形 + (と)(も) + 動詞辞書形 + まい + (と)(も) / (が)
**含义**: 不管……还是不……，都……

例：
- 彼が来ようと来まいと、会議は予定通り行う。（不管他来不来，会议照常进行。）
- 信じようが信じまいが、これが事実だ。（不管信不信，这就是事实。）
- 雨が降ろうが降るまいが、出発する。（不管下不下雨，都出发。）

⚠️ 〜まい 接续：一段动词去 る + まい / 五段动词辞書形 + まい / する→すまい・しまい / 来る→こまい・くるまい。

### 〜なりとも

**接续**: 名詞 + なりとも
**含义**: 即使是一点点……也好（书面/谦逊）

例：
- 少しなりともお役に立てれば幸いです。（哪怕能尽一点绵薄之力也万幸。）
- 一言なりとも申し上げたい。（哪怕只一句话也想说。）
- わずかなりともご支援をお願いします。（敬请些许支持。）

⚠️ 谦逊用语；书信、致辞常见。

## 11.11 命令禁止与建议

### 〜たまえ

**接续**: 動詞ます形 + たまえ
**含义**: 你……吧（男性、上对下的温和命令；古风）

例：
- まあ、座りたまえ。（坐下吧。）
- もう少し考えたまえ。（再想想吧。）
- 早く帰りたまえ。（早点回去吧。）

⚠️ 男性长辈/上司对下；现实日常已少用，多见小说、戏剧。

### 〜まで(のこと)だ

⚠️ 见 11.9。重复 cross-ref。

### 〜まい

**接续**: 動詞辞書形 / 一段ない形 / 不規則 + まい
**含义**:
1. 不会……吧（否定推量）
2. 决不……（否定意志）

例：
- もう二度と失敗するまい。（决不再失败。）
- 彼は知るまい。（他大概不知道。）
- あれほどひどい目にあった以上、二度と行くまい。（吃过那么大的苦，再也不去了。）

⚠️ 古风/书面意志否定，与「〜ないつもりだ」相比更文学。

### 〜なさい

**接续**: 動詞ます形 + なさい
**含义**: 请你……（命令；多用于父母对孩子、教师对学生）

例：
- 早く寝なさい。（快睡觉。）
- ちゃんと食べなさい。（好好吃饭。）
- 静かにしなさい。（安静。）

⚠️ N1 中作为完整列入，但本质命令形；语调强弱依对象。

### 〜なさるな / 〜たもうな

**接续**: 動詞ます形 + なさるな（古风敬体禁止）/ 動詞辞書形 + たもうな（更古）
**含义**: 请不要……（古风文学禁止）

例：
- ご心配なさるな。（请勿挂念。）
- 嘆きたもうな。（莫要悲叹。）

⚠️ 古文/文学；现代极少使用，戏剧、台词、和歌等。

### 〜にしたところで / 〜としたところで

⚠️ 限"人"或"团体"；表立场体察。

## 11.13 修辞与文体

### 〜ても始まらない

**接续**: 動詞て形 + も始まらない
**含义**: ……也无济于事

例：
- 今さら後悔しても始まらない。（事到如今后悔也没用。）
- 文句を言っても始まらない。（抱怨也无济于事。）
- 嘆いても始まらないから、前を向こう。（叹息也没用，向前看吧。）

### 〜たところでどうにもならない / 〜たところで〜ない

**接续**: 動詞た形 + ところで + 否定 / 不可能表现
**含义**: 即使……也（无济于事）

例：
- 今さら謝ったところで、彼は許してくれないだろう。（事到如今道歉也得不到原谅吧。）
- どんなに頑張ったところで、結果は同じだ。（再怎么努力结果也一样。）
- 急いだところで間に合うまい。（就算赶也来不及吧。）

⚠️ 与 〜ても 类似但更书面、含"放弃感"。

### 〜と思しい / 〜と思しき

**接续**: 名詞 / 普通形 + と思しい / と思しき + 名詞
**含义**: 看似……的（书面、推测）

例：
- 犯人と思しき人物が現場から逃走した。（看似犯人的人物从现场逃走。）
- 学生と思しき若者が集まっていた。（看似学生的年轻人聚集着。）
- 高齢者と思しき女性が倒れていた。（看似高龄者的女性倒下了。）

⚠️ 新闻/小说；「思しき」连体形最常用。

### 〜ぶる

**接续**: 名詞 / イ形(去い) / ナ形 + ぶる
**含义**: 装……（含贬义）

例：
- 上品ぶった態度が鼻につく。（装高雅的态度令人反感。）
- 学者ぶって難しい言葉ばかり使う。（装学者似的净用难词。）
- 知らないくせに知ったかぶるな。（不知道还装懂。）

⚠️ 贬义；不用于自指中性描述。「もったいぶる」「えらぶる」等惯用。

### 〜ばや

**接续**: 動詞未然形 + ばや
**含义**: 想要……（古文愿望，文语）

例：
- 故郷へ帰らばや。（多想回故乡。）
- 一目見ばや。（多想看一眼。）

⚠️ 古文；现代不用，仅见和歌、古典作品。需 review：作为 N1 项极冷僻，仅作识别。

### 〜とて〜ない

⚠️ 见 11.3。重复 cross-ref。

### 〜ども

**接续**: 動詞已然形 + ども
**含义**: 即使……也……（「けれども」的文语形）

例：
- 努力すれども、報われない。（努力却得不到回报。）
- 言えども聞き入れず。（虽说也不听。）

⚠️ 文语；惯用「老いたりといえども／〜と言えども」。

### 〜ようものなら

**接续**: 動詞意向形 + ものなら
**含义**: 万一……的话（就糟糕了）

例：
- 一言でも文句を言おうものなら、すぐ怒鳴られる。（要是说一句怨言就立刻被骂。）
- 遅刻しようものなら、即解雇だ。（万一迟到立即解雇。）
- ミスをしようものなら、二度とチャンスはない。（一旦犯错就再无机会。）

⚠️ 后句必为严重/极端负面后果；强调"假设的严重性"。

### 〜つもりはない / 〜つもりもない

**接续**: 動詞辞書形 + つもりはない / もない
**含义**: 没有……的打算（强烈否定意志）

例：
- 君を責めるつもりはない。（无意责备你。）
- 嘘をつくつもりはなかった。（本无意撒谎。）
- 諦めるつもりは毛頭ない。（毫无放弃之意。）

⚠️ 与 N3 的「〜つもりだ」对照；N1 升级强调"主观意志的强烈否定"。

## 11.14 命题成立与必然

### 〜よりほか(に)ない

⚠️ N2 章已详述。N1 升级：演讲/书面中常以「〜よりほかなかろう」加强语气。

### 〜ほかなんでもない / 〜ほかでもなく

**接续**: 名詞 + (に)ほかでもなく / 名詞 + ほかなんでもない
**含义**: 不是别的，正是……

例：
- 今日お話ししたいのは、ほかでもなく将来の進路についてです。（今天想谈的不是别的，正是将来出路。）
- 彼が選ばれたのは、ほかならぬその実力ゆえだ。（他被选中，正是因为其实力。）
- 君を呼んだのは、ほかでもない。（叫你来不为别的。）

⚠️ 强调"特意/正是这件事"。

### 〜にほかならない

⚠️ N2 章已详述。N1 升级：句末判断口吻「〜にほかならぬ」更文学。

### 〜ことは〜が

**接续**: 同一動詞・形容詞 + ことは + 同一動詞・形容詞 + が
**含义**: ……是……，但是……（部分肯定+让步）

例：
- 行くことは行くが、長くはいられない。（去是去，但不能待久。）
- おいしいことはおいしいが、少し高い。（好吃是好吃，就是有点贵。）
- 読んだことは読んだが、内容は覚えていない。（读是读了，但内容不记得。）

⚠️ 同词重复，后接让步；口语书面均可。

### 〜きり〜ない

**接续**: 動詞た形 + きり〜ない
**含义**: 自从……就再没……

例：
- 彼とは去年会ったきり、会っていない。（与他自从去年见过后就没再见。）
- 朝出かけたきり、まだ帰ってこない。（早上出去后还没回来。）
- 一度貸したきり、返してもらっていない。（借出去那次以后就没拿回。）

⚠️ 表"动作之后状态持续无变化"，含意外/担忧语气。

### 〜なんてもんじゃない

**接续**: 名詞 / 普通形 + なんてもんじゃない
**含义**: 岂止是……，简直……（强烈超越）

例：
- 暑いなんてもんじゃない、まるで炎天地獄だ。（岂止是热，简直是炼狱。）
- 忙しいなんてもんじゃない、寝る時間もない。（岂止是忙，连睡觉时间都没有。）
- 驚いたなんてもんじゃない、腰が抜けた。（岂止吃惊，简直瘫了。）

⚠️ 口语强调；后接夸张比喻。

### 〜ったらありゃしない / 〜といったらない

**接续**: イ形 / ナ形 / 名詞 + ったらありゃしない（ありはしない）/ といったらない
**含义**: ……到不行；……得没法说（强烈感叹/抱怨）

例：
- うるさいったらありゃしない。（吵得不行。）
- 面倒くさいといったらない。（麻烦得没法说。）
- 嬉しいといったらない、夢のようだ。（高兴得没法说，跟做梦似的。）

⚠️ 「ったらありゃしない」多负面；「といったらない」可正可负。口语为主。

### 〜なんと〜なことか / 〜なんと〜だろう

**接续**: なんと + イ形 / ナ形 / 普通形 + ことか / だろう
**含义**: 多么……啊（强烈感叹）

例：
- なんと美しい景色なことか。（多么美的景色啊。）
- なんと素晴らしい演奏だろう。（多么棒的演奏啊。）
- なんと愚かなことをしたものか。（做了多么愚蠢的事啊。）

⚠️ 演讲/文学；书面感叹的典型形式。

| pattern | 主要含义 | 后句限制 | 语体 |
|---|---|---|---|

|---|---|---|
| 〜ずにはおかない | 自然必然/强烈意志 | 书面 |

|---|---|---|
| 〜をよそに | 无视应关心之事 | 贬 |

---

> N1 语法核心要点：
> 1. **接续严格**：动词形态（辞書形／ます形／た形／意向形／已然形）选错即错。
> 2. **语体识别**：「べからず／にあらず／ねば／たまえ」属文语，现代仅用于特定场合。
> 3. **限定搭配**：「だに」「まじき」「ならでは」等接续名词/动词有限，靠惯用搭配记忆。
> 4. **褒贬倾向**：「きらいがある」「ぶる」「だの〜だの」必贬，「ならでは」「をものともせず」必褒。
> 5. **主语限制**：「が早いか／やいなや／なり」后句不能是说话人意志/命令。
> 6. **古文残留**：「まじ／べし／ず／ぬ／ども／ばや」需识别其现代连体・终止形。
