### 5.2 存在句 ✅

**节定位**：把 2.4 / 2.5 学过的几条规则系统化收尾——あります／います 自他对立 + 場所に vs 場所で + A/B 语序 + 方位词。

**核心要点**：
- あります（无意识物 / 植物 / 事件）vs います（人 / 动物 / 鬼神）
- A 型：場所**に** + X **が** + あります／います（报告"某地有 X"）
- B 型：X **は** + 場所**に** + あります／います（说明"X 在某地"）
- 场所助词：物存在用 **に** / 事件发生用 **で**
- 普通名词必须 **+ の + 方位词**（机 → 机の上 / 家 → 家の近く）
- 方位词：上 下 中 外 前 後ろ 横 隣 近く（横 vs 隣：隣需同类紧贴）

### 5.3 愿望表达 ✅

**节定位**：N5 高频愿望句型——ほしい / 〜たい / 〜たがる，全部继承一类形容词时态规则（3.4 直接套用）。

**核心要点**：
- **ほしい**（一类形容词）：N **が** ほしい / 第 1·2 人称专用 / 第 3 → ほしがる + ている / **を**
- **〜たい**（一类形容词）：V ます形去ます + たい / 第 1·2 人称专用 / 第 3 → V たがる + ている / **を**
- **を ⇄ が 互换**：仅他动词的 を 才能换 が（「に」「で」标的角色不能换）
- **「がる」接尾**的本质：把形容词"动词化"，从"内心"变成"外部观察"
- **〜たがる**单独用：表"一般倾向 / 性格特征" / 配 ている：表"此刻的样子"

### 📚 用户主动追问 ⭐⭐（5.3 节核心本质问）

**追问：为什么第 3 人称「〜たがる」要配 ている，第 1·2 人称的「〜たい」不需要？**

- ⭐⭐ 直击「形容词 vs 动词」状态语义差异
- 系统讲解**3 层逻辑**：
  1. **词性差异**：形容词自含状态（ほしい / たい），动词描述动作（ほしがる / たがる）
  2. **动词原形 ≠ 此刻**：表惯常 / 未来 / 真理，不表"正在"——要表持续状态必须 + ている
  3. **「がる」文化层**：日语认为"断定他人内心"是失礼，所以第 3 人称感情必用「がる」+ ている 表"外部观察"
- 补充：「〜たがる」单独用 = "一般倾向"（子供は何でも食べたがる）

#### 5.3 Q1（ほしい + が + 第 1 人称）：完全掌握 ⭕⭕⭐

- 「我想要一台新电脑」→ **わたしは新しいパソコンが欲しいです**。 ✅
- 🌟 主动用**汉字「欲しい」**（N4 起点字汇深度，N5 学习者通常只用假名）
- 跨节巩固：「が」专属词类（2.2 ほしい 加入清单）+ 「新しい」一类直接接名词（3.1）

#### 5.3 Q2（〜たい + 一类时态 + 接续词でも）：完全掌握 ⭕（修正版）

- **第一版**：わたしは昨日寿司が**食べたかった**でも、今日**食べたくありません**。
  - ❌ **混合简礼**：第 1 句简体 + 第 2 句礼貌体
  - ❌ **「でも」位置错**：接续词「でも」应放新句句首，前面要断句（与 2.16 节助词「でも」位置完全相反）
- **第二版**：わたしは昨日寿司が**食べたかったです**。**でも**、**今日は**食べたくないです。 ✅
  - ✅ 体统一全礼貌
  - ✅ 「でも」断句后开新句
  - ⭐ 接受建议加「今日**は**」对比用法（2.1 は 对比即时应用）
- 🔑 跨节认知：「でも」**助词（2.16）vs 接续词（5.3）位置完全相反**——助词在名词后，接续词在句首

#### 5.3 Q3（〜たがる + ている + を）：完全掌握 ⭕（修正版）

- **第二版**：父は新しい車**を**買いたがっています。 ✅

  - 私は車**が**買いたい（也可を）/ 父は車**を**買いたがる（只能を）

### 5.5 能力（〜ことができる）✅

**节定位**：与 5.4 姊妹句型，接续点不同（た形→原形），构造"能做〜"能力表达；和 4.11 直接可能形形成 3 种说法对照。

**核心要点**：
- 句型：**V 原形 + こと + が + できる** = 能做〜
- 「できる」= 2 类动词 + が专属词（2.2 清单又一员）
- **三种说法 A/B/C 对照**：
  - A 名词版：日本語**が**できます（已知技能名词，最简洁）
  - B 完整句型：日本語**を**話す**ことができます**（书面 / 商务最正式）
  - C 直接可能形：日本語**が**話せます（4.11 口语首选）
- **⚠️ 助词陷阱**：B 保留 **を**（动词原形仍是他动词）/ C 改 **が**（可能形助词转换）

- 📖 副词「少し」语序建议：紧贴动词更地道（→ 日本語**を少し**話す）

### 5.6 义务（必须）✅

**节定位**：N5 **最复杂的句型节**——4 种基本说法 + 2 种口语缩约，全部依赖"双重否定 = 强肯定"逻辑。

**核心要点**：
- 双重否定逻辑："**不做的话 + 不行**" = "必须做"
- **4 种合法组合**：
  - なくては + いけない（イケナイ组：个人主观）
  - なくては + ならない（ナラナイ组：社会规则）
  - なければ + いけない（イケナイ组）
  - なければ + ならない（ナラナイ组）
- **判别看后半**：ならない=社会 / いけない=个人（差异细微，初学不必过分纠结）
- **接续**：V ない形 → 去「い」+ くては / ければ
  - なくては：「ない」的中顿形「なくて」+ 主题 / 对比は
  - なければ：「ない」的假设形「なければ」（N4 假设形提前借用）
- **口语缩约**：
  - なくては → な**くちゃ**（保留く）
  - なければ → な**きゃ**（保留き）
  - 缩约形结句常**省略**

- 🆕 **新认知**：简体↔礼貌"整词替换"，不是叠加变形
  - ならない（简体）↔ なりません（礼貌）「なる」的语体切换
  - いけない（简体）↔ いけません（礼貌）「いける」的语体切换
- 🌟 **组别判别看后半**：选「ならない」=ナラナイ组（社会规则）✅

**🎉 5.7 不必 ✅ 通关——与 5.6 形成完整镜像，5.6/5.7/5.8 4 角句型矩阵 2/4 节击破**

### 5.8 许可与禁止 ✅ 🏆 首次单节零修正

**节定位**：4 角句型矩阵的**最后一角**——5.6/5.7/5.8 三节构成完整"必须·不必·许可·禁止"系统。本节包含 6 个相关句型。

**核心要点（4 角矩阵）**：
| 节 | 句型 | 助词 | 结句 | 意思 |
|---|---|---|---|---|
| 5.6 | な**くては** + いけない | は | いけない | 必须做 |
| 5.7 | な**くても** + いい | も | いい | 不必做 |
| 5.8 | **ても** + いい | も | いい | 可以做 |
| 5.8 | **ては** + いけない | は | いけない | 不许做 |

**对称美**：
- 横向：は ↔ も（一字之差，意思反转）
- 纵向：な（ない形）有 ↔ 无 = 否定假设 vs 肯定假设

**6 个衍生句型**：
1. 〜てもいい（许可，V て形 + も + いい）
2. 〜てはいけない（禁止，V て形 + は + いけない）
3. 〜てください（请做，V て形 + ください）
4. 〜ないでください（请勿，V ない形 + **で** + ください，⚠️中间是で）
5. 〜をください（请给我，物 + を + ください）
6. でございます（です 的尊敬版，商务超敬语）

**3 套句式的视角分流**（同一场景"不能拍照"）：
- 能力视角：〜ことができません（5.5）
- 规则视角：〜てはいけません（5.8 公告）
- 当面请求：〜ないでください（5.8 个人）

#### 5.8 Q1（许可疑问 + て形 + 着落点に）：完全掌握 ⭕⭕⭐

- 「这里可以坐吗？」→ **ここに座ってもいいですか**。 ✅
- ⭐ **精准选「に」**（着落点 2.4 用法 4）而非「で」（动作场所 2.5 用法 1）
- 「座る」助词辨析：N5 精细辨析点

#### 5.8 Q2（禁止句型 + 与 5.5 镜像）：完全掌握 ⭕⭕⭐⭐

- 「这家美术馆里不许拍照」→ **この美術館では写真を撮ってはいけません**。 ✅
- ⭐⭐ **视角分流认知打通**：
  - 5.5 Q2 你写的：撮ること**ができません**（能力视角）
  - 5.8 Q2 现在写：撮**ってはいけません**（规则视角）
  - 同一场景 "不能" 在日语二分
- ⭐ 「では」复合助词跨节稳定（5.5 → 5.8）

#### 5.8 Q3（〜ないでください + 多陷阱零失误）：完全掌握 ⭕⭕⭐⭐

- 「请不要在教室里吸烟」→ **教室ではタバコを吸わないでください**。 ✅
- ⭐⭐ **「吸わ」う段→わ**：4.5 节"1 类·う 例外"规则跨 30+ 节首次实战，零失误
- ⭐ **「ない**で**ください」**：5.8 最大陷阱"中间是 で 不是 て" 第一遍就对
- ⭐ **「では」**第 3 次稳定调用（5.5 → 5.8 Q2 → 5.8 Q3）
- 🌟 6 个跨节考点稳定运用：4.5 例外 / 4.9 て形 / 2.4 用法 4 / 2.10 修饰 / 5.5 では / 5.6-5.7 镜像

**🎉🎉🎉 5.8 许可与禁止 ✅ 完美通关——N5 4 角句型矩阵完整闭环 + 首次单节零修正！**

### 5.9 推量（〜でしょう）✅

**节定位**：N5 短小但语用复杂的句型——接续简单，但**两种意义靠声调区分**。

**核心要点**：
- **两种意义**：
  - **推測**：自我猜测，升调 ↗ ("大概…吧"）—— 天气预报最常用
  - **確認**：求同感，降调 ↘ ("…吧？"）—— 与对方共有认知
- **接续规则**：
  - 动词 / 一类形容词 → **简体** + でしょう
  - 名词 / 二类形容词 → **直接（词干）** + でしょう（⚠️ **不加だ / 不加です**）
- **底层逻辑**：「でしょう」自身就是「です」的推测形 → 不能再叠加です / だ
- **〜でしょうか**：更礼貌的疑问（"会不会…呢？"）
- **简体**：だろう（多见男性 / 自言自语 / 文学）

#### 5.9 Q1（推測 + 天气预报句式）：完全掌握 ⭕（修正 1 次）

- **第一版**：明日雪が降るでしょう**か**。 ❌ 多加か（变成疑问而非推测陈述）
- **第二版**：明日雪が降るでしょう。 ✅
- 🔑 「でしょう ↗ 推测陈述」vs「でしょうか ↗ 委婉疑问」的语用差异
- ⭐ 「雪**が**」自然现象主语用 が（2.2 / 2.20 跨节稳定）

#### 5.9 Q2（確認 + 二类形容词不加だ陷阱）：完全掌握 ⭕⭕⭐⭐

- 「这家店挺安静的吧？」→ **この店は静かでしょう**。 ✅
- ⭐⭐ **5.9 最大陷阱零踩**：静か（二类词干）**直接接 でしょう**，绝大多数 N5 学习者第一次都会写「静かだでしょう」（受 3.4 二类时态"だ"影响），你第一次就对
- 「は」主题 + 確認语气

#### 5.9 Q3（3 类動詞なかった形 + 推測）：完全掌握 ⭕⭕⭐

- 「他昨天大概没来吧」→ **彼は昨日来なかったでしょう**。 ✅
- ⭐ **「来る」3 类不规则三态记忆精准**：来ない=**こ** / 来た=**き** / 来なかった=**こ**
- 4.5 / 4.7 节扎实掌握的跨章节体现

**🎉 5.9 推量 ✅ 通关——推測/確認双意义 + 4 词类接续 + 不规则动词跨节稳定**

### 5.10 比较 ✅

**节定位**：整合 2.9 学过的「より」+ 新增的「のなかで/で」+「一番」最高级，构成 N5 比较系统。

**核心要点**：
- **より**（2.9 复习）：**A は B より + 形容词** = A 比 B〜
  - 助词位置：B 在前 + より 后置（与中文"比"前置反转）
- **范围助词判别**（新）：
  - **范围词**（日本 / クラス / 公園 / 世界）→ 直接接 **で**
  - **非范围词**（果物 / 人 / 友達）→ **+ の中で**
  - 与 5.2「場所詞 vs 普通名词 + 方位」同源逻辑
- **一番**（新）：N5 最高级唯一副词
  - 句型：A は [范围] で 一番 + 形容词
  - 书面更高级版：最も（もっとも）N4 学
- **より vs 一番**：2 物比较（は + より）vs 多者最高（が + 一番）
- **N5 问答模板**：「A と B と、どちらが〜ですか」/「A の方が〜です」（N5 后期句型）

#### 5.10 Q1（より + 跨节复习）：完全掌握 ⭕⭐（自主纠错）

- **第一版**：中国語**はより英語**難しいです。 ❌ 语序错位（より 前置）
- **第二版（主动撤回修正）**：中国語**は英語より**難しいです。 ✅
- ⭐ **今日第 2 次自主纠错**（5.7 Q1 + 5.10 Q1）——元认知警觉性持续激活
- 🔑 「より」铁律：**B より**（后置），与中文"比 B"前置反转

#### 5.10 Q2（一番 + 范围词で）：完全掌握 ⭕⭕⭐

- 「富士山是日本最高的山」→ **富士山は日本で一番高い山です**。 ✅
- ⭐ 「日本**で**」范围词直接接 — 判别精准
- 「高い山」一类直接接名词（3.1 跨节）

#### 5.10 Q3（のなかで + が + 好き）：完全掌握 ⭕⭕⭐⭐

- 「在水果中我最喜欢草莓」→ **果物の中でいちごが一番好きです**。 ✅
- ⭐⭐ **范围词 vs 非范围词判别双题通关**（Q2 日本で / Q3 果物の中で）
- ⭐⭐ **「いちご**が**」双重理由集中**：凸显"最"对象（5.10）+ 好き 是が专属词（2.2）→ 跨节助词逻辑稳固
- ⭐ 「好きです」二类直接接 です（不加だ）— 与 5.9 二类不加だ 同源逻辑

**🎉 5.10 比较 ✅ 通关——整合 2.9 より + 新「のなかで/で/一番」+ 跨节助词系统稳定**

### 5.11 条件 / 让步 ✅ ⭐⭐⭐ 章节五最大难关攻克

**节定位**：N5 → N4 之间的最大分水岭——5 个不同条件 / 让步表达共存。

**5 种表达全景**：
| 形式 | 适用场景 | 后项可接命令/意志？| 时间顺序 |
|---|---|---|---|
| **〜ても** | 让步（即使）| ✅ | 假设 |
| **〜たら** | **最通用** | ✅ | 假设/先后 |
| **〜ば** | 自然规律/谚语 | △ | 假设 |
| **〜なら** | 基于对方话题/建议 | ✅ | **可反向** |
| **〜と** | 必然/机械/操作说明 | ❌ | 必然 |

**核心要点**：
- **〜ても**：动词て形 + も / 一类くても / 二类·名词でも
- **〜たら**：万能默认，后项最自由（4 种た形 + ら）
- **〜ば**：1 类 え段 + ば / 2 类 + れば / 一类 ければ / 否定 なければ
- **〜なら**：简体（4 时态全部）+ なら / 二类·名词直接 + なら
- **〜と**：辞書形 + と；后项绝对禁命令/意志
- ⚠️ **5.8 vs 5.11 区分**：「〜ても」后接 いい → 5.8 许可 / 后接任何谓语 → 5.11 让步
- ⭐⭐ **〜なら 时间反向**：行く**なら**辞書を買う（去之前）vs 行っ**たら**辞書を買う（到达之后）
- **黄金法则**：选不出来时**默认用 〜たら**
