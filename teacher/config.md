## 三、核心学习方法（已锁定）

### 1. 影子跟读（Shadowing）—— 最重要
- 使用绿皮书
- 流程：
  1. 预听 + 理解
  2. 看书跟读几遍
  3. 合上书，只听音频做影子跟读（尽量同步）
  4. 同一段至少跟读 10-15 遍以上
  5. 录音自查，对比原音
  6. 提取 5-8 个完整句子
  7. **关键**：做替换练习（熟练说 + 熟练替换）
  8. 无提示测试（关书关音频复述/使用）

目标：达到“熟练说出来 + 能灵活替换 + 离开音频后也能用”的程度。

### 2. Anki 句子卡
- 从 shadowing 材料中挖完整句子做卡
- 推荐格式：
  - Front：日语句子
  - Back：中文翻译 + 必要拆解 + 替换方向
- 优先挖自然、常用、包含目标文法的句子

### 3. 输出练习
- 每天必须开口说完整句子 + 做替换
- 录音

### 4. 语法巩固
- 使用《标准日语》第一册
- 结合 shadowing 句子一起练

## 四、每日计划（已调整为可持续版）—— Phase 1 版本

- 影子跟读：30-40 分钟

**被动输入**：可作为额外磨耳朵（走路、做事时放音频）

## 九、长期规划与语法词汇学习方式（新增）

### 1. 计划性质
当前 90-120 分钟 + 重影子跟读的计划**不是固定长期计划**，而是 **Phase 1（基础重建期）**，预计执行 4-6 周。

长期来看，这是一个**分阶段系统**：
- 每日固定学习 + 汇报的框架长期保持
- 材料、重点、强度和目标会随阶段逐步升级

### 2. 长期路线图（简化版）

| 阶段       | 时间（约）     | 主要目标                     | 材料重点                              | 每日核心时长趋势 |
|------------|----------------|------------------------------|---------------------------------------|------------------|
| **Phase 1** | 现在 - 6周    | N5 彻底扎实 + 听力基础       | 绿皮 Shadowing + 标准日语第一册      | 90-120 分钟     |
| **Phase 2** | 第7-14周      | 完成 N4 + 听力速度提升       | Shadowing 进阶 + 标准日语进阶        | 100-150 分钟    |
| **Phase 3** | 第4-7个月     | 达到 N3 水平 + 简单自然交流  | みんなの日本語中级 / 総まとめ N3-N2  | 2-2.5 小时      |
| **Phase 4** | 第8个月以后   | N2 稳定 + 冲击 N1            | 新完全マスター系列 + 真实材料        | 2.5-3.5 小时    |

**关于 12 月 N1**：
我们会在 Phase 2 后期开始**并行**接触部分 N2/N1 高频语法和听力技巧，但 Phase 1 必须老老实实打牢基础。

### 3. 词汇如何学习（长期策略）
- **核心方法**：从影子跟读材料中挖掘**完整句子**做 Anki 卡
- 坚决不做孤立单词卡
- 早期以 shadowing 书句子为主
- 中后期逐步从 NHK 新闻简易版、对话、阅读材料中挖句
- 通过大量替换练习和输出，让词汇真正“活”起来
- 目标不是背 10000 个词，而是掌握能在真实场景中脱口而出的高频表达

### 4. 语法如何学习（长期策略）
- **不是死记语法书**，而是“解释 + 立即使用”
- 当前阶段：用《标准日语》做系统解释
- 学完一个文型后，必须立刻：
  - 在 shadowing 材料里找例句
  - 做大量替换练习（换人称、时态、场景）
  - 用 Anki 记录典型句子
- 后期阶段：
  - 教材切换为《どんなときどう使う》、《日本語総まとめ》、《新完全マスター》
  - 重点转向细微差别、敬语、复杂句型、逻辑连接
  - 更多通过真实输入（听力/阅读）反推语法

**整体逻辑**：语法和词汇都依附于**影子跟读 + 输出**这个主干，而不是单独模块。
