### 五、句型与表达（已通 11/31 节）

#### 5.11 条件 / 让步 ✅ ⭐⭐⭐ 章节五最大难关攻克
- **5 种条件 / 让步对比表**：
  | 形式 | 适用场景 | 后项可接命令/意志？| 时间顺序 |
  |---|---|---|---|
  | **〜ても** | 让步（即使）| ✅ | 假设 |
  | **〜たら** | **最通用** | ✅ | 假设/先后 |
  | **〜ば** | 自然规律/谚语 | △ | 假设 |
  | **〜なら** | 基于对方话题/建议 | ✅ | **可反向** |
  | **〜と** | 必然/机械/操作说明 | ❌ | 必然 |
- **接续要点**：
  - 〜ても：て形 + も / 一类くても / 二类·名词でも
  - 〜たら：4 种た形 + ら
  - 〜ば：1 类え段 + ば / 2 类 + れば / 一类 ければ / 否定 なければ
  - **〜なら：简体（4 时态全部）+ なら**（含过去形 食べたなら）
  - 〜と：辞書形 + と
- ⭐⭐ **〜なら 时间反向**（独有）：行く**なら**辞書を買う（去之前）vs 行っ**たら**辞書を買う（到达后）
- ⚠️ **5.8 vs 5.11 区分**：「ても」后接 いい→许可 / 后接任何谓语→让步
- **黄金法则**：选不出来时**默认 〜たら**
- 🆕 **笔误模式 ⑩**：片假名浊音元音混淆（バ vs ボ）
  - バ（ba）= バス / バナナ / バー / バタン（拟声词"砰"）
  - ボ（bo）= ボタン / ボール / ボーナス
- 🌟 **Q1 一句调 6 跨节**：5.11/5.8/4.9/4.6/4.14/2.4
- 🌟 **〜と 判别精准**：从 4 种里选出"机械必然"用 と
- ⭐ **用户深度追问** "为什么例子是食べたなら" → 触发文档自审，补全 〜なら 接续 4 时态表

#### 5.10 比较 ✅ — 整合 2.9 + 新句型
- **核心 3 元素**：
  - **より**（2.9 复习）：A**は**B**より** + 形容词（B 后置，与中文"比 B"反转）
  - **范围助词**（新）：范围词（日本/クラス/世界）+ **で** ／ 非范围词（果物/人/友達）+ **の中で**
  - **一番**（新）：N5 最高级唯一副词（书面：最も N4 后学）
- **句型**：A は [范围] で 一番 + 形容词（A 用 は / が 都可）
- **5.2 → 5.10 同源逻辑**：場所詞 vs 普通名词 + の上 ／ 范围词 vs 非范围词 + の中
- **N5 问答模板**：A と B と、どちらが〜ですか／A の方が〜です（の方が N5 后期高频）
- 🌟 Q1 **自主纠错（今日第 2 次）**：はより英語 → 英語より —— 元认知警觉持续激活
- 🌟 Q3 「いちご**が**」双重理由集中：凸显"最"对象（5.10）+ 好き 是が专属词（2.2）

#### 5.9 推量（〜でしょう）✅ — 推測/確認双意义 + 陷阱零踩
- **两种意义靠声调区分**：
  - **推測** ↗（自我猜测，"大概…吧"）— 天气预报标准句式
  - **確認** ↘（求同感，"…吧？"）— 与对方共有认知
- **接续表**：
  - 动词 / 一类形容词：**简体** + でしょう
  - 名词 / 二类形容词：**直接（词干）** + でしょう ⚠️**不加だ / 不加です**
- **底层逻辑**：「でしょう」=「です」的推测形 → 不能再叠加です/だ
- **更礼貌**：〜でしょうか（委婉疑问 "会不会…呢？"）
- **简体**：だろう（多见男性 / 自言自语 / 文学）
- 🌟 **Q2 二类不加だ陷阱第一次就对** — 5.9 最大陷阱零踩（绝大多数 N5 学生会受 3.4 二类"だ"影响误写"静かだでしょう"）
- 🌟 **「来る」3 类不规则三态记忆精准**：来ない=こ / 来た=き / 来なかった=こ（4.5/4.7 跨章节稳定）

#### 5.7 不必（〜なくてもいい）✅ — 与 5.6 镜像配对
- **句型**：V ない形 + な**くても** + いい = 不必做
- **镜像核心（一字之差，意思反转）**：
  - 5.6 必须：な**くては** + いけない（**は** = 主题/对比）
  - 5.7 不必：な**くても** + いい（**も** = 即使）
- **拆解**：なくて（中顿）+ も（即使）+ いい（好的）
- **过去庆幸**：〜なくてもよかった（"幸好不用"）
- **3.5 巩固**：いい → よい / よかった（变形必先变 よい，3.5 铁律）
- 🆕 **笔误模式 ⑨**：5.6/5.7 句型「な」漏写——句型本质是 "V**ない形（带な）** + くても / くては"，没"な"就不是双重否定
- 🌟 **Q1 自主纠错**：「ここは → ここで」无需提示主动修正——5.5 笔误教训自动激活检测
- 🌟 **Q3 一句集大成**：5.6 + 5.7 + 双は对比 + でも + 早く + 2 类·1 类 ない形

#### 5.6 义务（必须）✅ — N5 最复杂句型节打通
- **核心逻辑**：双重否定 = 强肯定（"不做 + 不行" = "必须做"）
- **4 种合法组合**：なくては / なければ × いけない / ならない
- **组别判别看后半**：ならない=社会规则 / いけない=个人主观（差异细微，初学不过分纠结）
- **接续**：V ない形 → 去「い」+ くては（中顿は）/ ければ（假设形 N4 提前借用）
- **口语缩约**：なくては → な**くちゃ**（保留く）/ なければ → な**きゃ**（保留き）/ 结句常省略
- **简体 ↔ 礼貌整词替换**：ならない↔なりません（不是叠加）
- 🆕 **笔误模式 ⑧**：「早い vs 速い」N5 高频混淆
  - 早い=时间维度（赶紧 / 早点 / 提前） / 速い=速度维度
  - 中文"快"在日语二分；判别："关心**几点**完成 vs **多快**完成"
- ⭐ Q1 → Q2 → Q3 三题完成"ナラナイ社会 → イケナイ个人 → 缩约自言自语"语感梯度
- 🌟 用户深度本质追问 + 反向揪出 study-record 总结笔误（"速く起きる"→"早く起きる"）→ 元认知亮点

#### 5.5 能力（〜ことができる）✅ — A/B/C 三型系统完整打通
- **核心三型对照**：
  - A 名词版：日本語**が**できます（已知技能名词，最简洁）
  - B 完整句型：日本語**を**話す**ことができます**（书面 / 商务最正式）
  - C 直接可能形：日本語**が**話せます（4.11 口语首选）
- **⚠️ 助词陷阱**：B 保留 **を** / C 改 **が**——5.5 最大考点
- **「できる」**= 2 类动词 + が专属词（2.2 清单又一员）
- **5.4 ↔ 5.5 速记**："经验" = ある / "能力" = できる
- 🆕 **「では」复合助词首次实战**：で（场所）+ は（主题），为 N4 句型铺路
- 🌟 Q3 双版本对比"我能说一点英语"——B 用 を / C 用 が 助词分流精准
- 🌟 「話す → 話せる」可能形变形跨 4.11 (23 节后) 稳定调用

#### 5.4 经验（〜たことがある）✅ — 5 大跨节助词稳定调用
- **句型**：V **た形** + こと + **が** + ある = 曾经〜过
- **「こと」**形式名词初登场（与 2.10 用法 4「の」并列）：こと=抽象事实 / の=具体感知
- **经验时间感**：必须有"距离" → ❌ 昨日 ／ ✅ 子供の頃 / 一度
- **次数副词**：一度 / 二度 / 三度 / 何度も / **一度も + 否定**（与 2.11 全否定连动）
- 🆕 **笔误模式 ⑦ 升级**：存在·所有句型中关键助词漏写
  - 5.2 「机の上 [ に ] あります」漏 **に**
  - 5.4 「こと [ が ] あります」漏 **が**
  - 💡 预防口诀：**ある／いる 前必有一助词**
- 🌟 Q3 一句调用 5 个跨节助词点全对：は（2.1）/ 不加 に（2.4 用法 1）/ を（2.3）/ が（2.2）/ か（2.13）

#### 5.3 愿望表达 ✅ — N5 高频愿望系统打通
- **核心三型**：
  - **ほしい**（一类形容词）：N **が** ほしい / 第 1·2 人称
  - **〜たい**（一类形容词）：V ます形去 ます + たい / 第 1·2 人称 / **を ⇄ が 可换**（仅他动词）
  - **ほしがる / 〜たがる**（动词化）：第 3 人称专用 / 配 **ている** 表"此刻" / 助词回归 **を**
- 🔑 **"がる"接尾本质**：形容词→动词，内心状态→外部动作；助词从"が专属"特权回归 を 宾格
- 🔑 **形容词 vs 动词的状态语义**：形容词自含状态（无需ている）/ 动词原形=惯常·未来·真理（要表"正在"必加 ている）
- 🔑 **「でも」位置律**（2.16 + 5.3 交叉）：
  - 助词「でも」(2.16) → **名词后**（コーヒー**でも**飲む）
  - 接续词「でも」(5.3) → **新句首**（〜です。**でも**、〜）
- 🌟 Q1 主动用**汉字「欲しい」**（N4 字汇深度初现）
- 🌟 Q3 4 段变形链零差错：買う→買い→買いたい→買いたがる→買いたがっています

#### 5.2 存在句 ✅ — 助词链 4 连胜
- **核心 5 大对立**：あります（无意识 / 事件）vs います（人 / 动物 / 鬼神）／ A 型「場所に X が」vs B 型「X は 場所に」／ 物存在に vs 事件発生で ／ 普通名词 + の + 方位词 ／ 助词「に」绝不省

- **常考事件词**（必配 で）：試合 / 試験 / 会議 / パーティー / 結婚式 / コンサート / 火事 / 地震 / 事故

- 🌟 Q3 收官 4 助词链「の・の・に・が」全对，A/B 型自主判断准确

### 三、形容词系统（已通 3/8 节）

#### 3.1 分类与判别 ✅
| 类别 | 词尾 | 修饰名词 | 例 |
|---|---|---|---|
| **一类（い）** | 词尾「い」 | **直接接** | 高い山、新しい本 |
| **二类（な）** | 词典中「だ」结尾 / 词干独立 | **+ な** | 静かな部屋、便利な道具 |

#### 3.2 长得像一类的二类陷阱词 ✅
- ⚠️ 4 个陷阱：**きれい / きらい / 有名（ゆうめい）/ 得意（とくい）**
- 写法以「い」结尾，**但属二类** → 修饰名词加 **な**
- ⚠️ 「きれい vs きらい」只差元音，意思相反（漂亮 vs 讨厌）
- 🔑 **汉字-假名规律**：「綺麗/嫌い/有名/得意」末尾汉字读音已含"い"，**汉字写法时不再加假名"い"**
- **📌 2026-05-18 文档修正**：原列表含「元気（げんき）」，但其词尾是「き」不是「い」，不符合陷阱特征——已删除
- 用户反思：错"綺麗いな"是"不会写きれい"的拼写问题，非规则误解（精确元认知）

#### 3.8 形容词名词化（一类 + さ）✅ — 章节三收官节
- 核心：把形容词变成**名词**（表"属性 / 程度 / 量度"）
  - **一类**：词尾 い → **さ**（高い → 高さ）
  - **二类**：词干 + **さ**（静か → 静かさ）
- 完整一类名词化表：長さ / 高さ / 大きさ / 重さ / 厚さ / 深さ / 寒さ / 暑さ
- 作用：把"属性"变成可测量、可比较、可讨论的对象
  - 富士山の**高さ**は 3776 メートルです（"高度"被讨论）
- 关联：N5 7000 词汇里大量「〜さ」词都是这套机制（暑さ/寒さ/大きさ 等）
- ⚠️ N4 后续：「〜み」是更感性、主观的名词化（暖か**み** / 重**み**）

#### 3.7 第二连用形（并列 / 中顿）✅ — 中顿系统完整闭环
- 核心句型："**既 A 又 B**"
  - **一类**：い → **くて**（安**くて**美味しい）
  - **二类**：+ **で**（静か**で**便利）
  - **名词**：+ **で**（学生**で**会社員）
- ⚠️ **时态只挂最后一项**！前项永远用中顿形不变
  - ✅ ハンサム**で**、親切**でした**（过去时只挂在 親切 上）
  - ❌ ハンサム**でした**、親切**でした**（双重时态，错）
- 🔑 **跨节合并**：2.5 用法6（で 中顿）+ 3.7（くて / で 中顿）= 统一"形容词中顿系统"
- 「くて」= "く" + "て"（て 是"连接器"，副词「く」+ 连接「て」）

#### 3.6 第一连用形（副词化，修饰动词）✅
- 核心：形容词修饰**动词**时变副词形
  - **一类**：い → **く**（早い → 早く起きる）
  - **二类**：+ **に**（静か → 静かに座る）
- ⚠️ **2026-05-21 修正**：原 study-record 误记为「速い → 速く起きる」，与文档不符。文档原文是「早く起きる」（早起 = "早い 时间维度"）。"早 vs 速" 是 N5 高频混淆点：早=时间 / 速=速度
- 🔑 **形容词 4 种角色完整对照**：
  | 修饰对象 | 一类 | 二类 |
  |---|---|---|
  | 名词 | 直接接 | + な |
  | 动词 | い → く | + に |
- ⚠️ 副词形必须**紧贴动词**
- ⚠️ 「多くの」特例：多**く**集まった（修饰动词）／ 多**くの**メンバー（修饰名词，特殊形式）
- 「いい」副词形 = **よく**（高频词：よく行く=经常去 / よく食べる=好好吃）

#### 3.5 特殊词 いい / よい ✅ — 不规则铁律
- 「いい」是**唯一不规则的一类形容词** —— **变形必先切换为「よい」**
- 完整变化表：

| 形态 | 写法 |
|---|---|
| 现肯 | **いい** / **よい**（都对，いい更口语） |
| 现否 | **よく**ない（绝不写 いくない） |
| 过肯 | **よかった**（绝不写 いかった） |
| 过否 | **よく**なかった |
| 中顿 | **よく**て |
| 变化 | **よく**なる（变好） |

- **复合词继承**：かっこいい / 気持ちいい / 頭がいい —— 全部走"よく〜"
- ⚠️ **N5 常见错误**：「ありません」自身只是现否，**过去化必须加でした**
  - 现否：よく**ありません**
  - 过否：よく**ありませんでした**

#### 3.4 4 时态全表 ✅ — 本章最大难关已通

**一类形容词时态表**（以「大きい」为例）：
| 时态 | 丁寧体 | 简体 |
|---|---|---|
| 现肯 | 大きい**です** | 大きい |
| 现否 | 大き**くない**です / 大き**くありません** | 大き**くない** |
| 过肯 | 大き**かった**です | 大き**かった** |
| 过否 | 大き**くなかった**です / 大き**くありませんでした** | 大き**くなかった** |

- ⚠️ 一类核心：**词尾「い」变形**——い→くない / かった / くなかった
- ⚠️ **过否 = 先现否再过去**："ない" 自身是一类形容词，ない→なかった

**二类形容词 / 名词时态表**（完全共用规则）：
| 时态 | 丁寧体 | 简体 |
|---|---|---|
| 现肯 | 静か**です** | 静か**だ** |
| 现否 | 静か**じゃありません** / **ではありません** | 静か**じゃない** / **ではない** |
| 过肯 | 静か**でした** | 静か**だった** |
| 过否 | 静か**じゃありませんでした** / **ではありませんでした** | 静か**じゃなかった** / **ではなかった** |

- ⚠️ **二类与名词共用规则**——大幅简化
- ⚠️ **「では」vs「じゃ」**：では=正式书面 / じゃ=口语缩约（意思相同）

**修饰名词**（已在 3.1 学过）：
- 一类：直接接（高**い**山）
- 二类：+ な（静か**な**部屋）
- 名词：+ の（日本語**の**本）

**⚠️ 「同じ」是二类例外**：
- ✅ 同じ**クラス**（修饰普通名词**不加 な**）
- ❌ 同じ**な**クラス
- 但谓语位置正常：これと**同じです** / 后接 のに 加な：**同じなのに**

**⚠️ 新笔误模式（重要！）**：助词 wa **永远写「は」，绝不写「わ」**
- ✅ **では**ありません ／ こんに**は** ／ こんばん**は**
- ❌ **でわ**ありません ／ こんにち**わ** ／ こんばん**わ**

#### 3.3 真正一类（看似不像）✅
- ⚠️ **かっこいい** = 帅／酷 —— **一类形容词**，虽然看似复合词
- 修饰名词直接接：**かっこいい俳優**（不加 な）
- 活用时不规则（因继承"いい"）：かっこ**よく**ない / かっこ**よかった**（3.4 详述）

### 一、指示词体系（こそあど）
- 状态：已通过 Q1（连体词 この/その/あの/どの + 必须后接名词）
- 遗留：完整体系（近/中/远/疑 × この/これ/ここ/こちら）建议未来回头复测

### 二、助词系统

#### 2.1 は（主题 / 对比）✅
- 读音 **wa**（助词时）
- 主题标记 + 「が前・は后」强调规则
- 两个「は」并列 = 对比用法（中间常配「が」表转折）

#### 2.2 が（主语 / 对象 / 转折）✅
- **用法 1**：主语 / 新信息（が 前面=新登场/听者也未知的事物）
- **用法 2**：喜好/能力/情感对象用 が（不用 を）
  - ⚠️ 上手不能自夸 → 改用 **得意**
  - ⚠️ 苦手（主观抵触）vs 下手（客观技术差）
  - 词类：好き/嫌い、上手/下手/苦手、欲しい/わかる/できる/怖い
- **用法 3**：疑问词主语铁律必用 が（誰/どれ/どの/どこ/何）
- **用法 4**：句末 が = 转折"但是" / 缓和"那个…"

#### 2.3 を（宾语 / 移动范围 / 离开点）✅
- **用法 1**：他动词宾语（ご飯を食べる）
- **用法 2**：移动/通过的范围 + 自动词（公園を散歩、空を飛ぶ）
  - 对比 で：で=动作舞台（停留型）／を=移动路径（穿越型）
- **用法 3**：离开/出发的起点（卒業、出発、家を出る、バスを降りる）
  - ⚠️ **真正判别 を vs から**：看**动词语义焦点**——"离开"型用 を；"到来/前往"型用 から（不只看"主观主动"）

#### 2.20 は vs が 专节 ✅ — 章节二压轴通关
- **核心规则 1（信息新旧）**：は = 已知话题（重心在后）／ が = 新信息·强调主语（重心在前）
  - 童话标准开头：おじいさん**が**住んでいました → そのおじいさん**は**…
- **核心规则 2（疑问词主语铁律）**：何/誰/どれ/どの/どこ + **が**（连回答也用 が）
- **核心规则 3（双主语句型）** ⭐ N5 经典：**大主语 は + 小主语 が + 谓语**
  - 象**は**鼻**が**長い / 北京**は**冬**が**寒い / 中国**は**夏**が**暑い
  - 私**は**猫**が**好き / 私**は**日本語**が**できる
  - 认知升级：2.2 学的"对象用 が"实际就是双主语的"小主语"
- **4 大对比场景**：现象描写 が / 客观判断 は / 信息焦点 / 否定倾向 は
- ⚠️ **用户笔误模式（第 4 次浊点）**：「散歩し**で**」→「散歩し**て**」
  - て/で 是长期挑战点：动词连接 = て（永远清音）／ 助词跟名词 = で

#### 2.19 终助词 ね / よ / な / の ✅ — 5 个终助词 3 题 5 答全胜
| 助词 | 用法 | 我 vs 对方 | 例 |
|---|---|---|---|
| **ね** | 求认同 | 双方都已知 | 可愛い**ね** |
| **よ** | 告知 | 我已知 / 对方未知 | 面白いです**よ** |
| **な（咏叹）** | 自我感叹 | 形容词+だ+な | 綺麗だ**な** |
| **な（禁止）** ⚠️ | 强硬制止 | 动词原形+な | 行く**な**！ |
| **の** | 软口气疑问 | 亲近关系 | どうした**の**？ |

- ⚠️ **な 同字两用**：接续点不同
  - 形容词+だ / 名词+だ / 动词简体 + **な** = 咏叹
  - 动词原形 + **な** = 禁止
- ⚠️ **禁止形 〜な** 商务场合**永不用**——改用「〜ないでください」（N4）
- "简体 + の" 标准搭档；"礼貌形 + の" 不自然
- 场景速记 か vs の：商务正式用 か / 家人朋友小孩用 の

#### 2.18 だけ vs しか〜ない（限定）✅ — 1 题双答双理由
- **だけ**：客观陈述"只"（可接肯定可接否定）—— わたし**だけ**です
- **しか〜ない**：主观"才"，强调不足 / 遗憾 —— **必须配否定**
  - ⚠️ 铁律：「しか」前后必须都齐，缺一不可
- 🔑 同一意思的两个版本：
  - 100円**だけ**ある（中性："只有 100 日元"）
  - 100円**しか**ない（主观："才 100 日元，不够"）
- 助词搭配：
  - だけ + が/を → 可省略（**第 4 次出现 "が/を 让位" 规则**，前 3 次：も / か / でも）
  - だけ + に/で/と/へ → 保留
  - しか + が/を → 吸收 ／ + に/で → にしか/でしか

#### 2.17 ずつ（平均分配）✅ — 2 用法 + 1 对比 3 题全胜
- **用法 1**：均等分配——**数字/数量 + ずつ**（一人**に**一個**ずつ**）
  - 搭档助词「に」= 2.4 用法 6 "分母线"
- **用法 2**：少量递增——少し**ずつ**わかる（每次同样的小量累积）
- ⚠️ **ずつ vs だんだん 关键区分**：
  - **ずつ**（助词）=横向等量（每份/每次量相同）
  - **だんだん**（副词）=纵向时间渐变（强调过程感）
  - 固定搭配：少しずつ ✓ ／ だんだんずつ ✗
- 🌟 用户主动使用「〜てください」请求形（N4 伏笔自主吸收）

#### 2.16 でも（举例提议）✅ — "が/を让位"规则三连验证
- 句型：**名词 + でも + 动词**（建议/邀请） = "…之类的吧 / 要不…"
- ⚠️ **不是接续词"但是"**！判别：位置在名词后 = 本节助词；位置在句首 = 接续词
- 语感：日式委婉——保留对方选择自由 / 降低被拒尴尬 / 不强求
- ⚠️ 助词搭配（**与 2.11 も / 2.13 か 共享底层规则**）：
  - **が/を 让位**：お茶**でも**飲む（不写 お茶**を**でも）
  - **に/へ 保留**：公園**にでも**行く
- 🌟 万能模板：**[名词] でも + [动词]ませんか** = 礼貌邀请
  - コーヒーでも飲みませんか（要不喝杯咖啡）— tosho-trading 职场实用
  - 公園にでも行きませんか（要不去公园）

#### 2.15 とか（口语列举）✅ — 列举三兄弟完整闭环
- 句型：**A とか B とか… [+ する / します]** = A 啦 B 啦【这些】
- ⭐ 独有优势：**とか 可以接动词原形**（や 只能接名词）
- 🏆 **列举三兄弟完整对照**：
  | 助词 | 列举性质 | 接续 | 语体 |
  |---|---|---|---|
  | **と** | 完全列举（就这些） | 名词 | 通用 |
  | **や〜など** | 不完全（举例） | 名词 | 偏书面 |
  | **とか** | 不完全（举例） | 名词 / 动词 | 口语 |
- 模板：**[A とか B とか] + します**（"做这些之类"，最地道的口语收尾）
- ⚠️ **笔误模式反向**：过去漏浊点（て→で），本节多浊点（とか→どか）。to / do 要分清

#### 2.14 へ（方向）✅ — 与 2.4 末尾对比辨析形成长期巩固
- 用法：表示**方向感**的目的地标记（≠ に 的"具体到达点"）
- 「こちら／あちら／どちら」+ **へ** = 服务业招牌用语（こちら**へ**どうぞ）
- ⚠️ 助词读音 **e**（不读 he）
- 复合形「への」已在 2.10 用法 6 学过（母**への**手紙）
- 跨节验证：2.4 Q8 学的"具体点 vs 方向感"间隔 12 节再考依然精准

#### 2.13 か（3 用法）✅ — 3 题全胜无重测
- **用法 1**：句末疑问（〜です**か**。）—— 日语零调换语序，只加 か
  - ⚠️ 传统书面用「。」；现代口语/SNS/省略 か 时可用「？」
- **用法 2**：选择并列（A **か** B）—— 中文"或 / 还是"
  - 永久模板：**[A か B] + 助词 + 动词**（助词只加一次，加在选项组合后）
- **用法 3**：疑问词 + か = **不定指代**（何**か** / 誰**か** / どこ**か** / いつ**か**）
  - ⚠️ "が/を 让位"规则在此同样适用（何**か**食べた，省略 を）
  - 判别：**句末有 か**=疑问句 ／ **句末没 か**=陈述句中的"某 X"
- 🔑 与 2.11 も 用法 2 形成完整对照：
  - 疑问词 + も + 否定 = 全否定（何**も**ない）
  - 疑问词 + か（+ 肯定）= 不定指代（何**か**ある）

#### 2.12 や（不完全列举）✅ — 与 2.6 と 完整对照
- 句型：**A や B（や C）など** = A 啊 B 啊【还有别的】
- ⚠️ **核心对比**（2.6 と 用法 2 伏笔兑现）：
  - **や〜など**（不完全）：举例，言外之意"还有别的"
  - **と**（完全）：穷尽，言外之意"就这些"
- 「など」非必加但**强烈建议**——更自然地道
- 使用频度：日常口语用 や 多；正式清单用 と
- 未来修正：「食べる」+ 喝的东西略奇怪——N4 「〜たり〜たり」可解决

#### 2.11 も（3 用法）✅ — 3 题全胜无重测
- **用法 1**：类同 "也"（兄**も**学生です）
  - ⚠️ **「も」替换 は/が**，不能写"はも / がも"
- **用法 2**：疑问词 + も + 否定 = **全面否定**（何**も**食べない／どこ**にも**行かない）
  - ⚠️ **助词搭配双轨规则**："**が/を 让位（省略），其他保留**"
  - 省略派：が→何**も**、を→何**も** ／ 保留派：にも、でも、からも、とも、へも
- **用法 3**：数量 + も = 主观觉得"多"/惊讶（20万円**も**する）
  - 常配「〜んですか」加强惊讶 / 关心语气
- 🔑 N5 高频实战词：兄/姉/父/母（自家谦让）vs お〜さん（别家敬称）
- 🔑 「〜円する」= 卖价 / 「〜んです」= 强调·解释（N4 再细讲）

#### 2.10 の（6 用法）✅ — 助词系统过半的里程碑节
- **用法 1**：所属（私の本）—— 中文"的"
- **用法 2**：性质 / 内容（日本語の本、木の机）—— "关于/做的"
- **用法 3**：同位 ⚠️ 中文没有！（社員の田中、友達の王さん）—— 前后指同一个对象
- **用法 4**：形式名词（大きい**の**）⚠️ 「の」自身不能省
- **用法 5**：动词名词化（〜する**の**は…）—— 把动词短语打包成名词
  - ⚠️ 区分：学習（名词→前面 の）／ 学習する（动词→前面 を）
  - 永久模板：**[宾语 を 动词简体] の は 〜です**
- **用法 6**：助词复合（での／への／との／からの／までの）
  - ⚠️ 「に + の」不存在，必须用「への」
- 🔑 修饰顺序："左到右修饰累加，最右那个才是真主角"
- 🔑 职场自介金句：**[公司名] の [部门/职位] の [姓名] です**

#### 2.9 より（比较基准）✅
- 句型：**A は B より + 形容词** = A 比 B 〜
- より 前面 = 被超越的一方；は 前面 = 胜者
- 中日语序差异：中文"比 B 〜" / 日语"B より 〜"（"比"被推到后面当后缀）
- 🌟 用户自发使用句末「ね」（寻求共鸣），语感开始外溢
- 📌 N4 后续：ね（共鸣）/ よ（告知）/ よね（求确认）

#### 2.8 まで / までに（终点对比）✅
- **まで** = 持续到某时（持续性动作）：10時**まで**勉強した／12時**まで**テレビを見た
- **までに** = 某时点之前完成（瞬间动作）：10時**までに**帰る／9時**までに**駅に着く
- 🔑 判别核心：**看动词的"持续 vs 瞬间"本质属性**，不死记搭配
- ⚠️ 同一时间点，动词不同 → 助词不同（5時**まで**会社にいる ／ 5時**までに**会社に着く）

#### 2.7 から（2 用法 + 接续要点）✅
- **用法 1**：时间 / 地点起点（7時から、海外から），常配「まで」组"从 A 到 B"
- **用法 2**：主观原因（から）vs 客观原因（ので，N4 再细讲）
- ⚠️ **接续规则**：
  - 一类形容词 / 动词简体 + から（直接）：寒いから / 食べるから
  - 二类形容词 / 名词 + **だ** + から：好きだから / 学生だから ← N5 高频翻车点
  - 礼貌形 + から：寒いですから（最稳妥）
- **复习衔接**：から 已在前节出现过两层
  - 2.3 用法 3：来源（アメリカから来た）
  - 2.5 用法 4：化学变化原料（ぶどうからワイン）

#### 2.6 と（3 用法）✅
- **用法 1**：共同动作的伙伴（友達と食べる）—— **双向** vs 2.4 に 用法 3（单向偶遇）
- **用法 2**：完全列举（パンと飴 = "就这两个"）—— 未来对比「や」举例列举
- **用法 3**：引用（と言う / と思う / と申す）
  - ⚠️ **「と思う」前必须接简体形**，禁用礼貌形
  - 简体可以是现在/过去/肯定/否定全部，不限"非过去"
- 🔑 用户神级类比："と 像程序员的管道符 |"，把前面的引用块送给后面动词

#### 2.5 で（6 用法）✅ — 7 题全胜
- **用法 1**：动作发生场所（食堂でご飯を食べる）
- **用法 2**：工具 / 手段 / 方法（日本語で書く、バスで行く）
- **用法 3**：范围限定 + 最高级（日本で一番高い） / 时间范围（5 分で着く）
- **用法 4**：材料——**で**=肉眼可见（木で箱）／ **から**=化学反应、原料消失（ぶどうから ワイン）
- **用法 5**：事件举办（教室で試験がある）—— vs に=物品存在（教室に机がある）
- **用法 6**：中顿 / 并列（名词、二类形容词）—— 私は中国人で、学生です
- 🔑 で 的核心意象 = **"通过/借助 X"**（X 是动作的舞台 / 媒介 / 范围）
- 标点小笔记：日语句末用「。」，句中停顿用「、」

#### 2.4 に（多用法）✅ — 本章最复杂的一节，已全通（已扩展至 8 用法）
**📌 2026-05-18 文档补充**：原文档 2.4 节只列 7 用法，由用户在 2.12 や 节后追问触发遗漏检测，**新增第 8 用法**：
- **8 用法**：
  - ① 时间点·精确（数字加/相对不加；星期=指定日加；朝/夜/季节不加）
  - ② 目的地（学校に行く）
  - ③ 动作对象·单向（母に手紙を書く）
  - ④ 着落点·附着位置（椅子に座る、バスに乗る、冷蔵庫に入れる）
  - ⑤ 存在场所（公園に犬がいる；⚠️ いる=有生命 vs ある=无生命）
  - ⑥ 频率（一週間に3回 — に 是分母线）
  - ⑦ 移动目的：V-ます形词干 + に + 行く/来る（食べに行く、散歩に行く）
  - ⑧ **用途 / 名目 / 资格**（朝ごはん**に**パンを食べる；お土産**に**お菓子を買う）⭐ 新增
    - 判别：中文翻"**作为〜**"的「に」 = 用法 8
    - 区分用法 7：に 前面是动词ます形→7；是名词→8
- **3 对比辨析**：
  - に（存在）vs で（动作）：学校にいる ／ 学校で勉強する
  - に（单向偶遇）vs と（双向约定）：偶然彼に会う ／ 友達と会う
  - に（具体点）vs へ（方向感）：学校に行く ／ あちらへ行く
- ⚠️ 助词「へ」读 **e** 不读 he

- **第三章 形容词**：い形容词 + です，词尾「い」不能丢（美味しい です ✓）

## ⚠️ 用户常见笔误规律（重点提醒）

1. **浊点（゛）漏写**：て/で、か/が、た/だ 等——一漏意思就变
2. **形容词「い」尾**：N5 早期容易丢
3. **术语精度**：「宾语」是 を 专用术语，不要混用于「に」标记的"动作对象"
4. **主动 ≠ を**：判离开起点要看**动词本身**是否含"离开"语义，不是只看"主观意志"
